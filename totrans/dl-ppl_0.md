*人工智能*（AI）是将人类思维嵌入计算机的领域：换句话说，创建一个模仿生物大脑功能的仿生大脑。人类能够智能地做到的事情现在都需要转移到机器上。第一代人工智能专注于人类可以形式化描述的问题。使用人工智能，智能行为的步骤以机器遵循的指令形式进行描述。机器遵循人类，没有变化。这些特征是人工智能第一时代的特征。

人类只能完全描述像棋类这样的简单问题，而无法描述更复杂的问题。在棋类游戏中，问题可以通过将棋盘表示为 8×8 的矩阵，描述每个棋子和它的移动方式，以及描述目标来简单解释。机器将被限制在人类正式描述的任务上。通过编程这些指令，机器可以智能地玩棋。现在的机器智能是人工的。机器本身并不智能，但人类已经将他们的智能以几行静态代码的形式转移到机器上。“静态”意味着行为在所有情况下都是相同的。在这种情况下，机器与人类绑定，不能独立工作。这就像主仆关系。人类是主人，机器是仆人，它们只是遵循人类的命令，没有更多。

为了使机器能够识别物体，我们可以以机器能够理解的方式向其提供专家的先前知识。这样的基于知识的系统构成了人工智能的第二代。这类系统中的一个挑战是如何处理不确定性和未知因素。人类即使在不同的复杂环境中也能识别物体，并能智能地处理不确定性和未知因素，但机器却不能。

## 目标

深度学习是机器学习的一个分支，其中你以概念层次结构的形式来建模世界。这种学习模式类似于人类大脑的学习方式，它允许计算机建模复杂的概念，这些概念在其他传统的建模方法中往往被忽视。因此，在现代计算范式下，深度学习在建模复杂现实世界问题中起着至关重要的作用，尤其是在利用今天可用的海量非结构化数据方面。

由于深度学习模型涉及到的复杂性，很多时候使用它的人将其视为一个黑盒。然而，为了从这一机器学习分支中获得最大利益，人们需要通过观察与之相关的科学和数学来揭示隐藏的奥秘。在这本书中，作者非常注重从数学和科学的角度解释与深度学习相关的概念和技术。此外，第一章完全致力于构建理解深度学习概念所需的数学基础。选择 TensorFlow 作为深度学习包的原因是其研究目的的灵活性以及易用性。选择 TensorFlow 的另一个原因是它能够利用其服务功能轻松地在实时生产环境中加载模型。

总结来说，*深度学习管道*应提供实用专业知识，以便您可以从头开始学习深度学习管道，这样您就可以部署有意义的深度学习解决方案。本书将使您能够快速掌握 TensorFlow 的使用，并优化不同的深度学习架构。本书强调了与任何行业相关的所有深度学习的实用方面。您将能够使用展示的原型来构建新的深度学习应用程序。本书中展示的代码以 iPython 笔记本和脚本的形式提供，允许您尝试示例并以有趣的方式扩展它们。您将具备在此领域进行研究并回馈社区所需的数学基础和科学知识。

本书中的所有代码都是使用 Python 实现的。由于原生 Python 在处理图像方面比较复杂，因此使用了多个库来帮助实现跨章节的效率。

## 本书面向对象

本书面向数据科学家和寻求解决复杂商业问题的机器学习专业人士，面向通过 TensorFlow 开发深度学习解决方案的软件开发人员，以及有持续学习愿望的研究生和开源爱好者。

## 本书组织结构

### 部分

+   *第一部分：介绍* —在这一部分，我们通过提供所有必要的先决条件来准备读者，以便他们开始从机器学习到深度学习的旅程。

+   *第二部分：数据* —作为管道的第一步，读者需要了解有关数据的所有信息，从数据收集到理解数据中的信息，再到数据处理和准备。

+   *第三部分：TensorFlow* —在这一部分，我们开始介绍有趣的内容。首先，我们阐述了深度学习的基本和重要概念；然后我们深入探讨神经网络的核心以及神经网络的类型，描述每种类型；并展示了深度学习方程的重要概念。同时，我们也不能忘记展示每种类型的真实生活示例。

+   *第四部分：应用所学知识* —这一部分旨在确保读者通过使用 TensorFlow 进行实践并构建管道。

### 章节列表

+   *第* *1* *章：轻松入门* —本章提供了全景图，向读者展示本书描述的领域；对该领域的介绍；以及描述机器学习工作原理的数学方程式和符号。

+   *第* *2* *章：设置您的环境* —本章介绍了本书中需要的编程工具和包以及一些有助于理解的理论；还包括对 Python 编程语言的简要介绍。

+   *第* *3* *章：深度学习管道的愉快之旅* —在第三章中，我们介绍了整本书的管道；深度学习方法和子领域；深度学习管道的步骤；以及 TensorFlow 相对于其他深度学习框架的独特之处所添加的额外功能。

+   *第* *4* *章：构建您的第一个玩具 TensorFlow 应用* —为了确保我们不会在书的中间让读者感到困惑，我们展示了使用 TensorFlow 的一个小示例，该示例将快速展示深度学习管道的每个步骤；并确保观众了解管道的每个步骤，它的重要性以及如何使用它。

+   *第* *5* *章：定义数据* —正如其名所示，本章是关于定义数据的。读者应该知道他们正在处理的数据类型，这非常重要，这样他们才能选择正确的数据准备方法。

+   *第* *6* *章：数据整理和预处理* —在理解数据之后，读者现在应该选择准备数据的方法和途径，因此本章有助于确保读者在这一步骤中选择正确的方法。

+   *第* *7* *章：数据重采样* —在清理和准备数据集之后，现在读者应该知道如何以正确的方式采样这个数据集。从你的数据中选择错误的数据样本可能会影响你模型的输出，因此在本章中，我们展示了所有需要的技巧和方法，以正确地采样你的数据集。

+   *第* *8* *章：特征选择和特征工程* —在本章中，我们描述了管道数据步骤中的一个非常重要的话题：特征选择和工程。读者应该知道如何选择和选择对输出特征贡献最大的重要输入特征。特征工程是使用数据领域的知识来创建使机器学习算法工作的特征的过程。特征选择和工程是机器和深度学习应用的基础，读者应该知道何时以及如何使用它们。

+   *第九章* *9* *：深度学习基础* —在本章中，我们描述了深度学习基础中一个非常重要的主题，即深度学习所基于的基本函数。然后我们尝试从这些函数构建层，并将这些层组合起来以获得更复杂的模型，这将帮助我们解决更复杂的问题。所有这些都将通过 TensorFlow 示例进行描述。

+   *第十章* *10* *：改进深度神经网络* —在本章中，我们描述了一个重要主题：在构建深度学习模型之后，改进工作就开始了。本章涉及优化、调整和选择超参数技术，以及权重归一化，以及这些如何使学习过程更加容易和快速。之后，读者应该知道如何评估、优化和调整模型参数以达到最佳解决方案和令人满意的准确度。

+   *第十一章* *11* *：卷积神经网络* —深度学习的重要类别之一是卷积神经网络。在本章中，我们详细介绍了 CNN 的所有内容，从一维掩码到高级内容，如权重共享以及等变性（equivariance）与不变性（invariance）之间的区别。我们通过使用著名的 CIFAR-10 数据集进行案例研究来展示。

+   *第十二章* *12* *：序列模型* —深度学习的另一类是序列模型。在本章中，我们描述了序列数据的问题以及循环神经网络（RNN）的兴起，以及 GRU 和 LSTM 的问题和演变，当然也包括一个案例研究。

+   *第十三章* *13* *：计算机视觉中的精选主题* —在第三部分完成卷积神经网络（CNN）之后，增加一些额外的知识是有益的，这使读者在工作时更容易，例如使用预构建的架构和迁移学习。

+   *第十四章* *14* *：自然语言处理中的精选主题* —本章填补了读者在处理文本时所需的知识空白，为读者提供了自然语言处理的所有高级方法和技巧。

+   *第十五章* *15* *：应用* —在这里，我们展示了几个案例研究，以确保读者能够全面了解如何构建管道，并使用现实生活中的例子。
