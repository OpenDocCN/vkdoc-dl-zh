# 引言

人工智能是计算机科学中一个快速发展的领域。自计算机发明以来，我们观察到了一些引人入胜的现象。对人类而言困难的任务（如繁重的计算、搜索、记忆大量数据等）对计算机来说轻而易举，而人类天生擅长且能快速完成的任务（如识别部分遮挡的物体、智能、推理、创造力、发明、理解语音、科学研究等）对计算机来说却十分困难。似乎我们每个人脑海中都有一台超级计算机。

人工智能作为一门学科诞生于 20 世纪 50 年代。然而，当时已知的神经网络架构使用具有线性激活函数的感知器，因此无法解决非线性问题。这一原因加上计算能力的不足，使得人工智能的早期起步并不成功。人工智能在 1974–1980 年间复兴，但最终再次失败。

在第二次失败后，新的非线性网络架构被开发出来，加之机器计算能力的巨大提升，最终促成了人工智能在 20 世纪 90 年代取得非凡成功。逐渐地，人工智能能够解决许多工业级问题，如图像识别、语音识别、自然语言处理、模式识别、预测、分类、自动驾驶汽车、机器人自动化等。

人工智能近年来的巨大成功引发了各种无根据的猜测。你可能会读到一些关于不久的将来机器人将匹配并超越人类智能的讨论。我们需要记住，当前的人工智能是一套巧妙的数学和处理方法，让计算机能够从处理的数据中学习，并运用这些知识解决许多重要任务。许多属于人类的东西，如智能、情感、创造力、感觉、推理等，仍然超出了人工智能的认知范围。

然而，情况正在迅速变化。近年来，计算机在国际象棋方面变得如此出色，以至于它们可靠地击败了人类对手。这并不奇怪，因为它们的创造者`教会`了程序几个世纪以来人类积累的国际象棋经验。如今，世界计算机国际象棋锦标赛已经设立，机器之间相互竞争。一个名为`Stockfish 8`的最佳国际象棋程序在 2016 年赢得了世界计算机国际象棋锦标赛。

几年前，谷歌开发了一个名为`AlphaZero`的国际象棋程序，它在 2017 年世界计算机国际象棋锦标赛中击败了`Stockfish 8`程序。令人惊叹的是，没有人像之前开发其他国际象棋程序那样教`AlphaZero`国际象棋策略。相反，它利用最新的机器学习原理，通过与自己博弈来自学国际象棋。该程序花了四个小时学习国际象棋策略（同时与自己博弈）就击败了`Stockfish 8`。自我学习是人工智能新的里程碑式成就。

人工智能有许多分支。本书致力于其中之一：神经网络。神经网络使计算机能够从观测数据中学习，并基于这些知识进行预测。本书是关于神经网络的训练，以及利用训练进行函数逼近、预测和图像分类。
