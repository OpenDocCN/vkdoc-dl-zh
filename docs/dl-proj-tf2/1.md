## 安装 Python 3

Python 是一种通用解释型、命令式、面向对象的高级编程语言。它是最古老的编程语言之一。然而，随着机器学习的出现，Python 获得了新生。它已经成为机器学习和深度学习的流行工具。目前，Python 有两个不同的版本——Python 2 和 Python 3。

本书所有项目都使用 Python 3，所以最好确保安装了。
