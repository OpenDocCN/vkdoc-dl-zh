# 3. 使用 PyTorch 的 CNN 与 RNN

概率与随机变量是 PyTorch 这类图计算平台中计算不可或缺的一部分。理解概率及其相关概念至关重要。本章涵盖概率分布及其使用 PyTorch 的实现，以及如何解读测试结果。在概率论与统计学中，随机变量也被称为*随机变量*，其取值完全取决于随机现象。概率分布有多种类型，包括正态分布、二项分布、多项分布和伯努利分布。每种统计分布都有其自身的特性。

## 配方 3-1. 设置损失函数

### 问题

如何设置损失函数并对其进行优化？选择合适的损失函数能提高模型收敛的可能性。

### 解决方案

在本配方中，我们使用另一个张量作为更新变量，将张量引入样本模型并计算误差或损失。然后，我们计算损失函数的变化率，以衡量损失函数的选择对模型收敛的影响。

### 工作原理

在以下示例中，`t_c` 和 `t_u` 是两个张量。它们可以由任何 NumPy 数组构建。

现在我们来定义模型。`w` 参数是权重张量，它与 `t_u` 张量相乘。结果加上一个常数张量 `b`，所选的损失函数是自定义的；该函数在 PyTorch 中也可用。在以下示例中，`t_u` 是使用的张量，`t_p` 是预测张量，`t_c` 是预先计算好的张量，需要将预测张量与 `t_c` 进行比较以计算损失函数。

## 配方 3-14：限制模型过拟合

### 问题

当我们拟合大量神经元和层来预测目标类别或输出变量时，函数通常会过拟合训练数据集。由于模型过拟合，我们无法在测试集上做出良好的预测。测试准确率与训练准确率不一致。训练和测试准确率之间会存在偏差。

### 解决方案

为了限制模型过拟合，我们有意引入了`dropout`比率，即随机删除（比方说）网络中 10%或 20%的权重，并同时检查模型准确率。如果我们在删除 10%或 20%的权重后，仍能达到相同的模型准确率，那么我们的模型就是好的。

### 工作原理

让我们看下面的例子。当训练好的模型无法泛化到其他测试场景时，就会发生模型过拟合。当训练准确率与测试准确率出现显著差异时，即可识别出过拟合。为了避免模型过拟合，我们可以在模型中引入`dropout`比率。

## 方法 3-18. 在 RNN 中嵌入层

### 问题

### 解决方案

第一步是创建一个嵌入层，它是一个固定的字典和固定大小的查找表，然后引入`dropout`比率，之后再创建门控循环单元。

### 工作原理

让我们看下面的例子。当文本数据以序列形式输入时，信息会以顺序方式被处理；例如，当我们描述某事物时，我们会按顺序使用一组词语来传达含义。如果我们使用单个词作为向量来表示数据，那么生成的数据集会非常稀疏。但是，如果我们使用基于短语的方法或词语组合来表示特征向量，那么这些向量就会变成一个密集层。密集向量层被称为*词嵌入*，因为嵌入层传达的是上下文或含义作为结果。这绝对比词袋方法要好。
