# 2. 使用 PyTorch 的概率分布

概率和随机变量是 PyTorch 这类图计算平台中不可或缺的组成部分。理解概率及其相关概念至关重要。本章涵盖概率分布及其在 PyTorch 中的实现，以及如何解释测试结果。

在概率论和统计学中，随机变量也称为*随机变量*，其结果取决于纯粹的随机现象。概率分布有多种类型，包括正态分布、二项分布、多项分布和伯努利分布。每种统计分布都有其自身的特性。

`torch.distributions` 模块包含概率分布和采样函数。每种分布类型在计算图中都有其重要性。该分布模块包含二项分布、伯努利分布、贝塔分布、类别分布、指数分布、正态分布和泊松分布。

## 方法 2-7. 分布

### 问题

统计分布的知识对于使用 PyTorch 进行基于神经网络的操作中的权重归一化、权重初始化以及梯度计算至关重要。我们如何知道该使用哪些分布以及何时使用它们？

### 解决方案

每个统计分布都遵循一个预先建立的数学公式。我们将使用最常用的统计分布及其在问题场景中的参数。

### 工作原理

伯努利分布是*二项分布*的一种特殊情况，其中试验次数可以多于一次；但在伯努利分布中，实验或试验的次数保持为一次。它是一个随机变量的离散概率分布，当事件成功的概率存在时，该变量取值为 1；当事件失败的概率存在时，该变量取值为 0。一个典型的例子是抛硬币，其中 1 代表正面，0 代表反面。让我们来看一下程序。

![../images/474315_1_En_2_Chapter/474315_1_En_2_Figz_HTML.jpg](img/474315_1_En_2_Figz_HTML.jpg)

*贝塔分布* 是一族在 0 到 1 范围内定义的连续随机变量。该分布通常用于贝叶斯推断分析。

![../images/474315_1_En_2_Chapter/474315_1_En_2_Figaa_HTML.jpg](img/474315_1_En_2_Figaa_HTML.jpg)

当结果具有双重性且实验可重复时，二项分布适用。它属于离散概率分布族，其中成功的概率定义为 1，失败的概率定义为 0。二项分布用于对多次试验中成功事件的数量进行建模。

![../images/474315_1_En_2_Chapter/474315_1_En_2_Figab_HTML.jpg](img/474315_1_En_2_Figab_HTML.jpg)

在概率论和统计学中，分类分布可以定义为广义的伯努利分布，它是一种离散概率分布，用于解释任何可能取其中一个类别的随机变量的可能结果，每个类别的概率在张量中专门指定。

![../images/474315_1_En_2_Chapter/474315_1_En_2_Figac_HTML.jpg](img/474315_1_En_2_Figac_HTML.jpg)

*拉普拉斯分布* 是一种连续概率分布函数，也称为*双指数分布*。拉普拉斯分布用于语音识别系统中以理解先验概率。它在贝叶斯回归中对于确定先验概率也很有用。

![../images/474315_1_En_2_Chapter/474315_1_En_2_Figad_HTML.jpg](img/474315_1_En_2_Figad_HTML.jpg)

*正态分布* 由于中心极限定理的性质而非常有用。它由均值和标准差定义。如果我们知道分布的均值和标准差，就可以估计事件的概率。
