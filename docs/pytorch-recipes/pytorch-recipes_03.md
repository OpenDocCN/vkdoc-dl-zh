# 1. PyTorch、张量及张量运算简介

`PyTorch` 已发展成为一个用于编写动态模型的大型框架。正因如此，它在部署大规模深度学习框架的数据科学家和数据工程师中非常受欢迎。本书为专家在处理实际数据科学问题时的操作活动提供了结构化的指导。从我们日常使用的应用程序中可以明显看出，产品功能中嵌入了多层智能。这些功能旨在为用户提供更好的体验和更优质的服务。

世界正朝着人工智能的方向发展。人工智能包含两个主要组成部分：深度学习和机器学习。没有深度学习和机器学习，就无法想象人工智能的实现。

`PyTorch` 是用于在 GPU（图形处理单元）和 CPU（中央处理单元）上执行深度学习任务的最优化的高性能张量库。`PyTorch` 的主要目的是提升大规模计算环境中算法的性能。`PyTorch` 是一个基于 Python 和 Facebook 人工智能研究团队提供的 Torch 工具的库，用于执行科学计算。

在 GPU 上基于 `NumPy` 的操作不足以高效处理繁重的计算任务。静态深度学习库成为提升计算灵活性和速度的瓶颈。从实践者的角度来看，`PyTorch` 张量与基于 Python 的 `NumPy` 库中的 N 维数组非常相似。`PyTorch` 库提供了桥接选项，可以将 `NumPy` 数组转换为张量数组，反之亦然，从而使该库在不同的计算环境中具有灵活性。

其最常用的应用场景包括自然语言处理、图像处理、计算机视觉、社交媒体数据分析以及传感器数据处理。尽管 `PyTorch` 提供了大量的库和模块用于计算，但有三个模块尤为突出。

- `Autograd`。该模块提供了张量自动求导的功能。程序中的记录器类会记住操作，并通过名为 `backward` 的触发器检索这些操作以计算梯度。这在神经网络模型的实现中非常有用。

- `Optim`。该模块提供了可用于最小化特定模型误差函数的优化技术。目前，`PyTorch` 支持多种先进的优化方法，包括 Adam、随机梯度下降（SGD）等。

- `NN`。`NN` 代表*神经网络*模型。手动定义函数、层以及使用完整的张量运算进行进一步计算，很难记住和执行。我们需要能够自动化层、激活函数、损失函数和优化函数的函数，并提供用户定义的层，从而减少人工干预。`NN` 模块包含一组内置函数，可以自动化运行张量运算的手动过程。

应用人工智能的行业包括银行、金融服务、保险、医疗保健、制造业、零售、临床试验和药物测试。人工智能涉及对物体进行分类、识别物体以检测欺诈等。每个学习系统都需要三样东西：输入数据、处理和输出层。图 1-1 解释了这三个主题之间的关系。如果任何学习系统通过从新示例或数据中学习而随着时间的推移提高性能，则称为*机器学习系统*。当机器学习系统变得过于复杂而难以反映现实时，就需要深度学习系统。

在深度学习系统中，会部署不止一层的学习算法。在机器学习中，我们考虑监督学习、无监督学习、半监督学习和强化学习系统。监督机器学习算法是指数据带有类别标签或结果标记的算法。我们向机器展示带有相应标签或标记的输入数据。机器通过一个函数来识别这种关系。请注意，这个函数将输入与标签或标记连接起来。

在无监督学习中，我们只向机器展示输入数据，并要求机器基于关联、相似性或差异性等对输入进行分组。

在半监督学习中，我们向机器展示输入特征和标记数据或标签；然而，我们要求机器预测未标记的结果或标签。

在强化学习中，我们引入奖励和惩罚机制，其中每个正确的行为都会得到奖励，每个错误的行为都会受到惩罚。

在所有这些机器学习算法的例子中，我们假设数据集很小，因为获取大量标记数据是一个挑战，而且机器学习算法处理大规模矩阵计算需要花费大量时间。由于机器学习算法无法扩展到大规模数据集，因此我们需要深度学习算法。

部署深度学习模型面临着各种挑战，这些模型需要大量标记数据、更快的计算设备和智能算法。任何深度学习系统的成功都需要良好的标记数据和更好的计算设备，因为智能算法已经存在。

用于图计算的主要框架包括 `PyTorch`、`TensorFlow` 和 `MXNet`。`PyTorch` 和 `TensorFlow` 在神经计算方面相互竞争。`TensorFlow` 和 `PyTorch` 在性能方面同样出色；然而，真正的差异只有在对特定任务进行基准测试时才能知晓。在概念上存在某些差异。



在 TensorFlow 中，我们必须定义张量、初始化会话并为张量对象保留占位符；然而，在 PyTorch 中我们不需要执行这些操作。

在 TensorFlow 中，以情感分析为例。输入句子被标记为正面或负面标签。如果输入句子的长度不相等，那么我们设置最大句子长度并添加零以使其他句子的长度相等，以便循环神经网络能够运行；然而，这是 PyTorch 中的内置功能，因此我们不必定义句子的长度。

在 PyTorch 中，调试更加容易和简单，但在 TensorFlow 中这是一项困难的任务。

在数据可视化方面，模型部署在 TensorFlow 中绝对更好；然而，PyTorch 正在发展，我们期望最终能在未来看到相同的功能。

TensorFlow 无疑经历了许多变化才达到稳定状态。PyTorch 刚刚进入游戏，因此需要一些时间才能充分发挥该工具的潜力。

## 什么是 PyTorch？

PyTorch 是由 Facebook 的人工智能部门开发的机器学习和深度学习工具，用于处理大规模图像分析，包括对象检测、分割和分类。然而，它并不仅限于这些任务。它可以与其他框架一起使用来实现复杂的算法。它是使用 Python 和 C++ 语言编写的。为了在 GPU 环境中处理大规模计算，编程语言应相应地进行修改。PyTorch 提供了一个很好的框架来编写自动在 GPU 环境中运行的函数。
