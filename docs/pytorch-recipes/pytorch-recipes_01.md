# 引言

近来，开发人工智能产品和解决方案已成为常态；因此，对基于图论的计算框架的需求正在上升。只有当建模框架具有动态性、灵活性并能适应其他框架时，深度学习模型才能在现实应用中发挥作用。

`PyTorch` 是图计算工具/编程语言领域的新成员。为了解决先前框架的局限性，`PyTorch` 承诺在部署深度学习模型方面提供更好的用户体验，并能够结合卷积神经网络、循环神经网络、LSTM 和深度神经网络来创建高级模型。

`PyTorch` 由 Facebook 的人工智能研究部门创建，旨在使模型开发过程简单、直接且动态，这样开发人员就无需在编译和执行模型之前声明对象。它基于 Torch 框架，是 Python 的一个扩展。

本书面向数据科学家、自然语言处理工程师、人工智能解决方案开发者、现有图计算框架从业者以及图论研究人员。本书将引导您了解张量基础、计算、执行基于算术的操作、矩阵代数以及使用 `PyTorch` 框架进行基于统计分布的操作。

第 3 章和第 4 章详细介绍了神经网络基础。探讨了高级神经网络，如卷积神经网络、循环神经网络和 LSTM。读者将能够使用 `PyTorch` 函数实现这些模型。

第 5 章和第 6 章讨论了模型的微调、超参数调整以及生产环境中现有 `PyTorch` 模型的优化。读者将学习如何选择超参数来微调模型。

在第 7 章中，解释了自然语言处理。深度学习模型及其在自然语言处理和人工智能中的应用是业界最受欢迎的技能之一。读者将能够对 `PyTorch` 在深度学习模型中的实现进行基准测试，以执行和处理自然语言。他们将能够将 `PyTorch` 与其他基于图计算的深度学习编程工具进行比较。
