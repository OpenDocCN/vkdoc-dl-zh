# 长短期记忆网络

长短期记忆网络（LSTM）是一种深度学习序列神经网络，旨在解决信息保留的挑战。作为循环神经网络（RNN）的一种特殊形式，LSTM 有效解决了传统 RNN 遇到的梯度消失问题，这是机器学习算法中的一个重要难题。`Hochreiter 和 Schmidhuber`^(¹³) `开发了 LSTM 以克服传统 RNN 固有的局限性`，提出了一种提升机器学习模型性能的解决方案。LSTM 模型的实现可通过 Python 中的`Keras`库来完成。

举例来说，想象一下在观看视频或听一首歌时回忆起之前的场景，或者在读书时记起前几章的情节。类似地，RNN 通过保留先前信息并利用其处理当前输入来运作。然而，由于梯度消失问题，RNN`在记住长期依赖关系方面存在局限`。而 LSTM 则专门设计用于规避这些与长期依赖相关的挑战。

与传统神经网络不同，长短期记忆网络（LSTM）集成了反馈连接，使其能够处理整个数据序列，而非单个数据点。这一独特能力使其在理解和预测序列数据（包括时间序列、文本和语音）中的模式方面表现出色。

在人工智能和深度学习领域，LSTM 已成为一种强大工具，通过从序列数据中提取有价值的见解，为各个领域的突破做出了贡献。

从广义上讲，LSTM 的运作方式与 RNN 单元非常相似。LSTM 网络的内部机制包含三个关键组件，每个组件都有不同的功能：

- 遗忘门
- 输入门
- 输出门

第一部分决定来自上一时间戳的信息是否相关，应予以保留，还是被视为无关而丢弃。第二部分专注于从输入中学习新信息。最后，在第三部分中，来自当前时间戳的更新信息被传递到下一个时间戳，构成 LSTM 循环中的一个时间步。

这三个组件也被称为`门`。这些门控制着信息进出记忆单元或 LSTM 单元的流动。遗忘门、输入门和输出门调节着这些流动。一个 LSTM 单元由这些门以及一个记忆单元或 LSTM 单元组成，可以类比为`传统前馈神经网络`中的一层神经元，其中每个神经元都包含一个隐藏层和一个当前状态。

与基本 RNN 类似，LSTM 具有一个隐藏状态，记为上一时间戳的`H(t-1)`和当前时间戳的`H(t)`。此外，LSTM 还包含一个单元状态，分别用`C(t-1)`和`C(t)`表示上一时间戳和当前时间戳。隐藏状态被称为`短期记忆`，而单元状态则被称为`长期记忆`。

**示例：** *考虑两个以句号分隔的句子：“约翰是个好人，”接着是“另一方面，艾莉森很坏。”当我们从第一个句子过渡到第二个句子时，LSTM 网络利用遗忘门，可以恰当地忽略关于约翰的信息，并意识到焦点已转移到艾莉森身上。*

> **注意**
>
> **双向 LSTM**
>
> *超越 LSTM，双向 LSTM 是一种循环神经网络（RNN）架构，它以前向和后向两个方向处理输入数据。与仅考虑从过去到未来信息流的传统 LSTM 不同，双向 LSTM 融入了* `未来到过去的上下文`，增强了其捕捉双向依赖关系的能力。
>
> *该架构由两个 LSTM 层组成——一层用于前向处理，另一层用于后向处理——在需要全面理解输入序列的任务中尤其有效，例如* `情感分析`、`机器翻译`和`命名实体识别`。双向 LSTM 从两个方向整合信息，增强了其捕捉`长期依赖关系`的能力，从而在复杂的序列数据中实现更准确的预测。
