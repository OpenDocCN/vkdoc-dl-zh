# 马尔可夫、香农与语言模型

语言模型的起源可追溯至 20 世纪早期和中期**安德雷·马尔可夫**^(¹) **与克劳德·香农**^(²)的贡献。马尔可夫为我们如今所称的 n 元语法奠定了数学基础。有趣的是，n 元语法的概念最早出现在马尔可夫的早期著作中。安德雷·马尔可夫是深入研究语言模型的最早科学家之一，尽管当时"语言模型"这一术语尚未被创造出来。

考虑条件概率`p(w1|w0) = p(w1)`。不同的语言模型采用不同的方法来计算条件概率，记为`p(wi|w1, w2, ..., wi-1)`。获取并应用语言模型的过程被称为**语言建模**。n 元语法模型是一种基础模型类型，它假设任意位置的单词仅受其前面`n – 1`个位置的单词影响。本质上，该模型是一个**n – 1 阶马尔可夫链**。

1906 年，马尔可夫开始研究马尔可夫链，最初探索了一个仅有两个状态及它们之间相应转移概率的简单模型。他证明，通过根据转移概率在这些状态之间移动，转移到每个状态的频率最终会收敛到期望值。在随后的几年里，马尔可夫扩展了该模型，并证明这一结论在更普遍的情境下也成立。

为了说明其模型的实用性，马尔可夫于 1913 年将其应用于亚历山大·普希金的诗体小说《叶甫盖尼·奥涅金》。通过移除空格和标点符号，并将前 20000 个俄文字母分类为元音和辅音，马尔可夫得到了一个代表小说元音和辅音分布的序列。利用手动计数方法，马尔可夫计算了元音和辅音之间的转移概率，并用所得数据验证了最简单马尔可夫链的特性。

有趣的是，马尔可夫链最初的应用领域正是语言，马尔可夫的研究是对**语言模型最简单形式**的基础性探索。

时间来到 20 世纪中期，**克劳德·香农**建议使用马尔可夫过程来模拟自然语言。他利用 n 阶马尔可夫链，开发了一个能够描述字母序列（包括单词和句子）概率的统计模型。从数学角度来看，香农的方法涉及统计长度为 n 的字符序列（即 n 元语法）的出现频率。

1948 年，克劳德·香农以其开创性论文《通信的数学理论》革新了信息论领域。这篇论文标志着熵和交叉熵等关键概念的开端，以及对 n 元语法模型的探索。（香农在约翰·冯·诺依曼的建议下，从统计力学中借用了"熵"这一术语。）

在此背景下，熵表示概率分布固有的不确定性，而交叉熵则概括了一个概率分布相对于另一个概率分布的不确定性。值得注意的是，熵是交叉熵的下界。具体来说，当单词序列长度趋于无穷时，可以定义语言的熵，假设该熵是一个可以从语言数据中估计的常数值。

香农的贡献为评估语言模型提供了一个有价值的工具。如果一个语言模型在预测单词序列方面比另一个更准确，那么它将表现出更低的交叉熵。这一见解为评估语言建模提供了稳健的框架。需要认识到，语言模型不仅限于自然语言，还包括形式语言和半形式语言。

马尔可夫和香农的开创性工作为语言建模的多种方法铺平了道路，包括基于规则的系统、基于神经网络的系统，以及最近基于注意力机制的**基于 Transformer 的预训练大语言模型**（LLMs）。

## 乔姆斯基与语言模型

1956 年，诺姆·乔姆斯基^(³)提出了**乔姆斯基层级文法**，这是一个旨在表示**语言句法**的框架。乔姆斯基强调了有限状态文法（包括 n 元语法模型）在充分描述自然语言方面的局限性。

乔姆斯基的理论认为，语言由有限或无限的句子集合组成，其中每个句子是从有限词汇表中抽取的有限单词序列。根据乔姆斯基的说法，文法由一组能够生成语言中所有句子的产生式规则定义。不同的文法产生不同复杂程度的语言，形成一个**层级结构**。

能够生成有限状态机可接受的句子的文法称为有限状态文法或正则文法，而能够生成非确定性下推自动机可接受的句子的文法则称为上下文无关文法。有限状态文法被包含在上下文无关文法之中。

构成有限马尔可夫链或 n 元语法模型基础的文法与有限状态文法一致。然而，有限状态文法在生成具有语法关系的英语句子（例如 (i) 和 (ii) 所示的句子）方面存在局限性。

*   如果 S1，那么 S2。
*   要么 S3，要么 S4。
*   要么如果 S5，那么 S6，要么如果 S7，那么 S8。

虽然这些关系理论上可以无限组合以形成正确的英语表达（如示例 iii），但有限状态文法无法解释所有此类组合。乔姆斯基认为，用有限状态文法（包括 n 元语法模型）来描述语言存在重大限制。相反，他主张使用上下文无关文法来更有效地建模语言。这一影响导致在随后的几十年里，上下文无关文法在自然语言处理（NLP）中得到更广泛的应用。尽管乔姆斯基的理论在当代 NLP 中可能不再具有显著影响力，但它仍保留着重要的科学价值。

## 基于规则的语言模型

本节概述了第一个聊天机器人 ELIZA，它由约瑟夫·韦森鲍姆于 1966 年创建。ELIZA 基于规则的对话模拟方法标志着一个重要的里程碑，尽管它缺乏对语言细微差别的真正理解。这个时代关注基于规则的聊天机器人程序，并催生了几个类似的系统，包括 PARRY、ALICE 和 CLEVER。
