# 多模态学习如何运作

多模态神经网络通常融合多个单模态神经网络。例如，在一个视听模型中，有两个不同的单模态网络：一个专门处理视觉数据，另一个处理音频数据。每个单模态网络都通过一个称为编码的步骤独立处理其输入。在单模态编码之后，从每个模型中提取的信息需要被有效地融合在一起。

已经提出了各种融合技术，从简单的拼接到引入注意力机制。融合多模态数据的过程是决定成功与否的关键因素。在融合过程之后，一个最终的“决策”网络接收融合后的编码信息，并为最终任务进行训练。

本质上，多模态架构通常包含三个组成部分：

1.  单模态编码器，分别对不同的模态进行编码，通常每个输入模态对应一个编码器
2.  一个融合网络，负责在编码阶段合并从每个输入模态中提取的特征
3.  一个分类器，接收融合后的数据并生成预测

这些组件通常被称为编码模块、融合模块和分类模块。
