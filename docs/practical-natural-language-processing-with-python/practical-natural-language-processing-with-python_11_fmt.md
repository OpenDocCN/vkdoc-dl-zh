## 关于作者

**玛坦吉·斯里**是印度著名的数据科学领军人物。她在直观客户体验、室内定位和用户画像领域拥有 11 项已授权专利和 20 多项已公开专利。她在构建世界级数据科学解决方案和产品方面拥有超过 16 年的可靠业绩。她精通机器学习、文本挖掘和 NLP 技术及工具。她曾在花旗银行、汇丰银行、通用电气等大型企业以及`247.ai`、`PhonePe` 和 `Gojek` 等科技初创公司组建数据科学团队。她为初创公司、企业和风险投资公司提供数据科学战略和路线图方面的咨询。她是多所顶尖学府机器学习领域的活跃贡献者。
