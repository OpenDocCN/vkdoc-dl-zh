## 短信数据

短信是触达全球 35%人口的最佳方式。作为沟通渠道，短信拥有最高的打开率（打开短信的人数与收到短信的人数之比）：是电子邮件打开率的 5 倍（https://blog.rebrandly.com/12-sms-text-message-marketing-statistics）。美国人平均每天收到 33 条短信（www.textrequest.com/blog/how-many-texts-people-send-per-day/）。许多应用程序公司会访问客户的短信并挖掘数据以改善用户体验。例如，像`ReadItToMe`这样的应用可以在用户开车时朗读他们收到的任何短信。`Truecaller`会读取短信并将其分类为垃圾短信和非垃圾短信。`Walnut`则根据用户收到的短信提供其消费视图。仅通过分析交易类短信数据，就可以提取出大量用户信息：用户的收入、支出、支出类型、网购偏好等。如果我们只分析商业短信，数据源会更加结构化。见图 1-4。

## 内容数据



数字内容在我们的生活中无处不在。在线新闻文章、博客、视频、社交媒体和电子书是我们每天消费的主要内容类型。

根据[`cmo.adobe.com/articles/2019/2/5-consumer-trends-that-are-shaping-digital-content-consumption.html`](https://cmo.adobe.com/articles/2019/2/5-consumer-trends-that-are-shaping-digital-content-consumption.html)的数据，消费者平均每天花费 8.8 小时消费数字内容。以下是数据科学家在使用文本挖掘时需要解决的关键问题：

- 内容聚类（相似内容分组）
- 内容分类
- 实体识别
- 分析用户对内容的评价
- 内容推荐

其他关键数据，例如用户对内容本身的反馈，则更具结构性：点赞数、分享数、点击量、停留时长等。通过将用户偏好数据与内容数据相结合，我们可以深入了解用户的偏好信息，包括生活方式、人生阶段、爱好、兴趣和收入水平。

## IVR 语音数据

IVR 是交互式语音应答的缩写。IVR 是一种帮助用户在语音通话中导航各种菜单选项的系统。根据[www.marketsandmarkets.com/Market-Reports/interactive-voice-response-market-33851149.html](http://www.marketsandmarkets.com/Market-Reports/interactive-voice-response-market-33851149.html?gclid=Cj0KCQjwh8jrBRDQARIsAH7BsXfkISk9rnRcUPrjpBP6qC_a3-xBFCaNzEVz4FANz5y__ZXbN7yA9ZoaAkovEALw_wcB)的预测，到 2023 年，IVR 市场规模预计将达到 55 亿美元。基于自然语言的 IVR 是一种先进的 IVR 系统，它通过类似机器人的界面帮助客户找到问题的解决方案。图 1-5 展示了一个对话记录示例。

**IVR 机器人**：您好，欢迎致电 XYZ 银行。请问有什么可以帮您？您可以说“查询余额”或“我的卡刷不了”等。

**用户**：你好，我在一家零售店，我的卡用不了。

**IVR 机器人**：为了验证身份，请提供您的卡号。

**用户**：XXXXXXXX

**IVR 机器人**：感谢您的验证。由于长期未使用，您的卡已被停用。请问需要我为您激活这张卡吗？

**用户**：好的，请激活。

**IVR 机器人**：您的卡现已激活。您可以继续交易了。

**用户**：谢谢，太好了。

***图 1-5.** IVR 对话记录示例*

IVR 语音通常比较简短。用户知道自己在和机器人对话，因此倾向于使用短句。由于这些语音内容非常直接，除了用户当前面临的问题外，我们无法了解用户的其他属性或偏好。与您之前看到的其他示例相比，这是一种结构性强得多的文本。
