# 数据类型

计算机自然如此。这是一个从人类必须“学习”使用计算机，转变为计算机被训练去理解人类的时代。它是人工智能（`AI`）中处理语言的一个分支。该领域可追溯到 20 世纪 50 年代，当时在机器翻译领域开展了大量研究。艾伦·图灵曾预测，到 21 世纪初，计算机将能够完美地理解自然语言并用其做出回应，以至于你无法区分人类和计算机。在`NLP`领域，我们距离这一基准还很远。然而，有些人认为，这甚至可能不是衡量该领域成就的正确视角。无论如何，`NLP`对许多企业的成功至关重要。很难想象没有谷歌搜索、Alexa、YouTube 推荐等的生活。如今，`NLP`已无处不在。

为了更好地理解`AI`的这一分支，让我们从基础开始。任何数据科学领域的基础都是数据。因此，理解文本数据及其各种形式是执行自然语言处理的核心。让我们从商业用途的角度，从一些最熟悉的日常文本数据来源开始：

- 搜索
- 评论
- 社交媒体帖子/博客
- 聊天数据（企业对消费者和消费者对消费者）
- 短信数据
- 内容数据（新闻/视频/书籍）
- IVR 语音数据

## 搜索

从客户角度来看，搜索是最广泛使用的数据来源之一。所有搜索引擎的搜索，无论是通用搜索引擎还是网站或应用内部的搜索，其核心都使用索引、检索和相关性排序算法。搜索（也称为查询）通常由两三个单词的短句组成。搜索引擎的结果是近似匹配的，它们不一定需要完全精确。对于一个查询，总是会呈现多个选项作为结果。这种用户界面将寻找答案的责任转移回用户身上。回想一下，你有多少次因为对结果不满意而修改了查询？你不太可能去责怪搜索引擎的性能，而是专注于修改你的查询。

## 评论

评论可能是被分析得最广泛的数据。由于这些数据公开可用，或者通过网页爬取很容易提取，许多组织都在使用这些数据。评论本质上是自由流动的，且非常非结构化。评论挖掘是亚马逊、Flipkart、eBay 等电商公司的核心业务。像 IMDB 和 Tripadvisor 这样的评论网站，其核心也是评论数据。还有其他组织/供应商提供对这些公司收集的评论的洞察。图 1-1 展示了来自[www.amazon.in/dp/B0792KTHKK/ref=gw-hero-PC-](http://www.amazon.in/dp/B0792KTHKK/ref=gw-hero-PC-dot-news-sketch?pf_rd_p=865a7afb-79a5-499b-82de-731a580ea265&pf_rd_r=TGGMS83TD4VZW7KQQBF3) [dot-news-sketch?pf_rd_p=865a7afb-79a5-499b-82de-731a580ea265&pf_rd_](http://www.amazon.in/dp/B0792KTHKK/ref=gw-hero-PC-dot-news-sketch?pf_rd_p=865a7afb-79a5-499b-82de-731a580ea265&pf_rd_r=TGGMS83TD4VZW7KQQBF3) [r=TGGMS83TD4VZW7KQQBF3.](http://www.amazon.in/dp/B0792KTHKK/ref=gw-hero-PC-dot-news-sketch?pf_rd_p=865a7afb-79a5-499b-82de-731a580ea265&pf_rd_r=TGGMS83TD4VZW7KQQBF3) 的示例评论数据。

## 社交媒体帖子/博客

社交媒体帖子和博客与评论一样，被广泛研究、提取和分析。推文和其他微博内容简短，因此看似易于提取。然而，根据用例的不同，推文可能包含大量噪音。根据我的经验，平均每 100 条推文中只有 1 条包含关于某个感兴趣概念的有用信息。在使用 Twitter 数据分析品牌情感时尤其如此。在这篇关于情感分析的研究论文中，在针对特定主题收集推文后，发现只有 20%的英文推文和 10%的土耳其文推文是有用的：

## 聊天数据

### 个人聊天

个人聊天是 WhatsApp 聊天、Facebook 或任何其他即时通讯服务的经典日常语料。它们无疑是了解用户行为（尤其是在亲友圈中）最丰富的信息来源之一。这些聊天内容充满了大量需要剔除的噪音，就像你在推特数据中看到的那样。只有一小部分语料与提取有用的商业信息相关。这种商业有用信息的出现率并不高。也就是说，它的信噪比很低。这篇论文在[www.aaai.org/ocs/index.php/ICWSM/ICWSM18/](http://www.aaai.org/ocs/index.php/ICWSM/ICWSM18/paper/view/17865/17043)[paper/view/17865/17043](http://www.aaai.org/ocs/index.php/ICWSM/ICWSM18/paper/view/17865/17043)中研究了来自各种 WhatsApp 聊天的公开提交数据。
