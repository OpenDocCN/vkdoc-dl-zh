# 引言

我有幸接触并有机会解决复杂的自然语言处理问题，这些问题的解决惠及了全球各地的不同业务。本书源于我的学习心得，因此它是一本从实践者角度解决文本问题的著作。解决自然语言处理问题需要创造力和技术知识的结合。有时，最好的深度学习方法并不如简单的解决方案有效。我始终铭记奥卡姆剃刀原则：当存在多种选择时，最简单的方案往往最能解决问题。
在我看来，任何问题的答案都蕴含在数据中。这就是第一章讨论文本数据的原因。我涵盖了不同类型的文本数据以及可从这些数据中提取的信息。第 2、3、4 和 5 章聚焦于不同行业及其面临的常见文本挖掘问题。在介绍这些用例时，我还讲解了相应的文本挖掘概念以及相关代码和 Python 库。
第 2 章快速审视了客户服务行业，并深入探讨了对话语料库。它重点介绍了可从对话语料库中提取的各种信息类型，并附有详细代码。我还详细介绍了词袋模型、向量化、基于规则的方法和监督学习方法。
第 3 章聚焦于一个非常流行的文本挖掘用例：评论挖掘。我深入探讨了情感分析和意见挖掘的各个方面。我介绍了使用无监督和监督技术进行情感分析的方法。本章还引入了用于情感分析的神经网络。
第 4 章概述了银行和金融服务行业使用的技术。传统上，银行业使用结构化数据来驱动决策。近来，文本用例正在这一领域得到探索，我将详细讨论其中一个。本章中，我使用无监督技术和基于嵌入的神经网络深入探讨了命名实体识别器。
第 5 章专门讨论虚拟助手。我探索了使用最先进的神经网络架构构建机器人的技术。本章还介绍了自然语言生成的概念。
