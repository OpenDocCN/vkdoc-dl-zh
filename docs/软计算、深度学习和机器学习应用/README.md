
# 软计算、深度学习和机器学习应用

## 前言

所谓的第四范式在过去的二十年中得到了蓬勃发展，在这个范式中，科学家和工程师可以获得大量的观测数据。 大数据的特点是五个 V: 体量、多样性、价值、速度和真实性。大数据的概念与地质工程和地球科学的特点自然契合。 大规模、全面、多方向和多领域的岩土工程数据分析正在成为一种趋势。 另一方面，软计算（SC）、机器学习（ML）、深度学习（DL）和优化算法（OA）提供了从数据中学习并深入洞察岩土工程问题的能力。 研究人员使用不同的软计算、机器学习、深度学习和优化算法模型来解决与地质工程和地球科学相关的各种问题。

因此，有必要通过整合SC、ML、DL和OA技术来扩展其与大数据研究的研究。
本书侧重于地质工程和地球科学中SC、ML、DL和OA算法的最新技术和应用。 首先简要介绍了各种SC、ML、DL和OA方法，主要涉及易于解释的多元自适应回归样条（MARS）模型、监督学习、无监督学习、深度学习和优化算法。 然后通过VOSviewer演示总结了它们在地质工程和地球科学中的代表性应用。 作者还提供了他们从这些应用中学到的思考以及正在进行和未来的建议。 本书旨在为地质工程和地球科学等研究领域的研究人员和工程师提供全面的总结和基本指南，以便他们如何整合和应用SC、ML、DL和OA方法。

# 第1章 引言

在介绍更常用的人工智能（AI）、机器学习（ML）、深度学习（DL）和优化算法（OA）技术表达之前，首先应该提到软计算（SC）的定义，因为前三个术语与软计算更相关。

软计算是使用近似计算来提供复杂计算问题的不精确但可用的解决方案的方法。这种方法使得对于可能无法解决或仅仅是当前硬件解决起来耗时较长的问题能够得到解决。软计算有时被称为计算智能，与硬计算进行比较。它提供了一种除了计算机以外的问题解决方法。与传统的计算模型不同，软计算以人类思维为榜样，容忍部分真相、不确定性、不精确性和近似性。

软计算的容错性使研究人员能够处理传统计算无法处理的问题。

随着计算机性能的快速发展，人工智能逐渐成为第四次工业革命的重要驱动力。人工智能的概念诞生于1956年的达特茅斯会议，由几位计算科学家（如约翰·麦卡锡、马文·明斯基、克劳德·香农）提出，他们最初的想法是讨论利用机器模仿人类学习和其他智能方面的能力。自2012年以来，由于先进的监测技术或设备（Dikshit等，2020年；Gupta等，2020年；Meng等，2021年；Reichstein等，2019年）导致的数据量迅速增加，计算能力的提升以及新的机器学习算法——深度学习的出现，不断推动了人工智能的热门。人工智能已广泛涵盖许多领域，如专家系统、机器学习、进化计算、计算机视觉、自然语言处理等。尽管“人工智能、机器学习和深度学习”这三个常见术语彼此相关，但实际上它们并不相同，这可以从图1.1中看出。图中还显示了软计算。一般来说，机器学习可以被视为实现人工智能的方法，能够从数据中捕捉内部模式，然后提供合理的决策作为指导。受到人脑处理信息的方式的启发，

## 第二章 软计算

正如引言部分和图2.1所示，SC的定义与ML、DL和OA有一些重叠。它使用了模糊逻辑、机器学习、概率推理、进化计算、感知器、遗传算法、微分算法、支持向量机、元启发式、群体智能、蚁群优化、粒子优化、贝叶斯网络、人工神经网络、专家系统等研究领域的组成部分。作为一门数学和计算机研究领域，SC自上世纪90年代以来一直存在。灵感来自于人类大脑通过近似形成现实世界的解决方案的能力。与可能性相反，当没有足够的信息来解决问题时，使用这种方法。相反，SC用于问题没有足够明确以使用传统的数学和计算机技术。它在家庭、商业和工业等各种实际应用中都有很多应用。

## 2.1 什么是软计算

在1994年之前，Zadeh (1994) 首次定义“软计算”之前，当前处理的概念通常以孤立的方式被提及，每个概念都单独讨论，并指出了模糊方法的使用。尽管建立软计算领域的想法可以追溯到1990年 (Zadeh 2001)，但在Zadeh (1994) 中，Zadeh对软计算的定义如下：

> “基本上，软计算不是一个统一的概念和技术体系。相反，它是一种以某种方式符合其指导原则的不同方法的合作。在这个关键时刻，软计算的主要目标是利用对不确定性和模糊性的容忍来实现可处理性、鲁棒性和低成本的解决方案。软计算的主要组成部分是模糊逻辑、神经计算和概率推理，后者包括遗传算法。”

算法、信念网络、混沌系统和部分学习理论。 在模糊逻辑、神经计算和概率推理的合作中，模糊逻辑主要关注不确定性和近似推理；神经计算关注学习和曲线拟合；概率推理关注不确定性和信念传播。

因此，很明显，软计算并没有一个精确的定义，而是通过不同的概念和技术来扩展定义，试图克服在不确定、模糊和难以分类的真实问题中出现的困难。

因此，有各种后续尝试进一步完善这个定义，结果各不相同，在可能的替代定义中，也许最合适的是Li等人 (1998) 提出的定义：“每个计算过程都有意将不确定性引入到一个或多个层次的计算中，并允许这种不确定性在某个阶段改变（减少）问题的粒度，或者在某个阶段“软化”最优化目标，被定义为属于软计算领域。”

软计算被认为是我们所说的硬计算的对立面。因此，它可以被看作是一系列技术和方法，使得真实的实际情况可以像人类处理它们一样处理，即基于智能、常识、类比、方法等。在这个意义上，软计算是一个以近似推理和功能以及优化逼近方法为首的问题解决方法家族，包括搜索方法。因此，软计算是智能系统领域的理论基础，显然人工智能领域与智能系统领域的区别在于前者基于硬计算，而后者基于软计算。

## 2.2 软计算的组成部分

从第二层的另一个视角来看，软计算可以扩展为其他组成部分，这些部分通过扩展来对其进行定义，例如上一节中给出的定义。从一开始 (Bonissone 2002)，在这个第二层中被认为最重要的组成部分是概率推理、模糊逻辑和模糊集、神经网络和遗传算法 (GA)，由于它们的跨学科应用和结果，它们立即凸显出其他方法学（如之前提到的混沌理论、证据理论等）之上。GA的流行，以及它们在各种领域和应用中的证明效率，它们试图模仿自然生物（例如植物、动物、人类），这些生物显然是软性的（即灵活、适应性强、创造性、智能等），尤其是扩展和不同版本，将这第四个第二级成分转化为众所周知的进化算法（EA），因此成为软计算的第四个基本组成部分，如下图所示：

1. 从第一层开始，采用近似推理方法时，当我们只关注概率模型时，我们会遇到Dempster-Shafer理论和贝叶斯网络。然而，当我们考虑将概率方法与模糊逻辑以及其他一些多值逻辑相结合时，我们会遇到可以称之为混合概率模型的情况，基本上是模糊事件的概率论模型、模糊事件信念模型和模糊影响图。

2. 当我们看到与模糊逻辑直接相关的发展时，模糊系统特别是模糊控制器突出出现。然后，由模糊逻辑与神经网络和进化算法相结合产生的模糊逻辑基础混合系统也引人注目，其中最重要的代表是模糊神经系统、由神经网络调整的模糊控制器（与前面提到的模糊神经系统不同）以及使用进化算法创建和调整的模糊逻辑控制器。

3. 通过第一层进入由软计算（功能方法/优化方法）覆盖的另一个大区域时，首先出现的组件是神经网络及其不同的模型。与模糊逻辑方法和进化算法方法的相互作用相关的是混合神经系统，特别是网络参数的模糊控制以及神经网络中的形式生成和权重生成。

4. 软计算的第四个典型组成部分，也许是最新的但可能是最新的进化算法，与之相关的有四个重要领域：进化策略，进化编程，遗传算法和遗传编程。如果我们只关注这些最后的领域，我们可以认为在这种情况下，与软计算相关的方法和技术的融合形成了三条重要线路：模糊遗传系统，生物启发系统以及用于进化参数模糊控制的应用。

在进一步研究这最后一个组成部分时，需要考虑一些额外的因素。首先，独立于采用的开放思维方法来思考模糊遗传系统、生物启发系统和模糊控制应用于进化参数的内容，这个描述中还缺少其他重要的主题。其次，如果我们特指生物启发系统，很明显它们不仅仅是模糊逻辑、神经网络或进化算法的产物（考虑到这三个组成部分的所有变体），还涉及其他非常重要的方法论。

## 2.3 启发式和元启发式

正如Jiménez等人 (2003) 所述，自20世纪90年代模糊集合（即软计算）的兴起以来，基于模糊集合的方法已成为研究、开发和创新的永久组成部分，并且它们的应用已扩展到我们日常生活的各个领域：健康、银行、家庭，也是不同教育层次的研究对象。同样，毫无疑问，由于我们目前拥有的技术潜力，计算机可以处理各种新领域中的极其复杂的问题（无论是在理解还是在规模上）。

正如我们上面提到的，自从1990年代中期以来，遗传算法（或从一般观点来看，进化算法）已经被证明在这些领域中寻找良好解决方案非常有价值，并且由于它们的科学吸引力、应用的多样性以及在智能系统中的解决方案的高效性，它们已经被纳入软计算组件的第二层级。

然而，遗传算法只是另一类启发式算法或元启发式算法，就像禁忌搜索、模拟退火、爬山法、变邻域搜索、估计分布算法 (EDA)、散列搜索、GRASP、反应式搜索和许多其他算法一样。一般来说，所有这些启发式算法（元启发式算法）通常提供的解决方案并不是理想的，但它们往往能够满足决策者或用户的要求。当这些算法基于满足性优于优化时，它们完美地诠释了扎德的名言：“与传统的硬计算相比，软计算利用对不精确性、不确定性和部分真实性的容忍来实现易处理性、鲁棒性、低解决方案成本和更好的与现实的关联性”。

因此，在软计算组件中，应考虑启发式算法甚至元启发式算法，而不是仅仅使用EA（它只能代表搜索和优化方法的一部分）。

通常对元启发式算法和启发式算法之间的区别存在争议，虽然我们不打算在这里参与这场辩论，但我们对这两个概念进行了简要的思考。启发式一词来自希腊词“heuriskein”，其含义与发现某物的概念相关。

因此，元启发式算法比启发式算法更加广泛。在接下来的章节中，我们将重点介绍元启发式算法的概念，并指出在我们定义的术语中，某些元启发式算法在解决问题时的性能总是优于其他算法。

为了实现元启发式算法的最佳性能，希望它们具有一系列“良好的特性”，包括简单性、独立性、一致性、有效性、高效性、适应性、鲁棒性、互动性、多样性和自治性 (Melian et al., 2003)。从它们的定义和一系列期望的特性来看，很明显进化算法是元启发式算法中的一种，并且它们与其他二级软计算组件一起，为实现新的理论和实践方法、框架提供了便利，以更好地理解和处理现实世界中的广义模糊性。

## 2.4 软计算中的混合元启发式

在本节中，我们将考虑前面提到的三个主要的元启发式算法组。然后，我们将描述新出现的元启发式算法，简要介绍较不发达或不流行的算法，因为它们是较新的。

1. 进化元启发式。这些元启发式是迄今为止最流行的，它们定义了解集的搜索空间中发展演化的机制，以接近具有元素的理想解。这些元素将在后续的种群中存活。在软计算的背景下，以这些元启发式为参考的混合方法是基础。

2. 放松元启发式。当一个实际问题通过消除、削弱或修改其特征元素来简化时，它可以被放松。放松元启发式是在启发式设计中放松问题的策略，能够找到在没有使用这种方法的情况下很难解决的问题的解决方案。这些的例子包括四舍五入或调整，就像当一个不精确和语言表达的数量与一个确切的数值相关联时发生的那样。从这个角度来看，一个真正的选择是灵活化精确算法，引入模糊停止准则，最终导致基于规则的放松元启发式；接受系数的模糊性，为解决具有模糊参数的问题提供算法，并放松对约束条件的验证，允许某些违反其履行。

3. 搜索元启发式方法。一般来说，这些可能是最重要的元启发式方法，它们的基本操作是建立探索问题解空间的策略并迭代起始解。尽管乍一看它们可能与进化搜索相似，但它们并不是，因为进化搜索是基于搜索空间中个体群体的演化。这些元启发式方法通常通过各种隐喻进行描述，将它们分类为仿生、社会学、自然等，这使得它们非常受欢迎。

然而，如果使用各种元启发式方法进行搜索过程，这些方法之间总是存在合作的可能性 (Cruz 2005)，因此将迄今为止描述的所有内容推广到并行性的背景中，这显然超出了本文的范围，但这是一个有趣的思考问题，因为随着并行计算的普及，更强大的工作站和更快的通信网络，元启发式方法的并行实现已经成为一种自然而然的选择，并提供了一个提高解搜索速度的有趣的替代方案。相应地，已经提出和应用了各种策略，并且这些策略在解决大规模问题和寻找比其顺序对应物更好的解方面证明了其非常高效的能力，这要归功于搜索空间的划分，或者因为它们改进了搜索的强化和多样化。因此，并行性（因此也包括多个元启发式方法）不仅是减少单个元启发式方法执行时间的一种方式，还可以提高它们的效果和鲁棒性。

## 2.5 基于数据驱动的非参数可解释MARS模型

多元自适应回归样条（MARS）最早由Friedman（2007）提出，作为一种适应性回归程序，在高维分析中表现良好。它是一种基于分割和征服策略的非参数统计方法，其中训练数据集被划分为不同梯度（斜率）的分段线性段（样条）。一般情况下，样条通过加权求和和平滑连接在一起，以处理线性结构，在某些特殊情况下，这些分段曲线，也称为基函数（BFs），也可以参与乘法，以获得拟合非线性行为的高阶项。

分段线性基函数的形式如下：

```
(x - t)_+ = \begin{cases} x - t, & \text{if } x > t, \\ 0, & \text{otherwise}, \end{cases}

And

(t - x)_+ = \begin{cases} t - x, & \text{if } x < t, \\ 0, & \text{otherwise}, \end{cases}
(2.1)
```

其中下标“+”表示正部分。

如图2.2所示，每个基函数(BFs)$(x - 1)_+$和$(1 - x)_+$都是分段线性的，两个部分之间的界面点称为节点，在这种情况下，节点值为1。这两个函数在下面的讨论中被称为反射对。由于想要为每个输入$X_j$形成具有节点在每个观测值$X_{ij}$的反射对，基函数的集合是

$$C = \left\{ (X_j - t)_+, (t - X_j)_+ \right\}_{j=1,2,3,4,...,N}^{\in \{ x_{1j},x_{2j},x_{3j},...,x_{Nj} \}} \quad (2.2)$$

在模型构建过程中应用了前向逐步线性回归，但是不直接使用原始输入X，而是使用来自集合C的函数及其交互作用。因此，模型的形式为：

$$F(X) = \beta_0 + \sum_{m=1}^{M} \beta_m h_m(X) \quad (2.3)$$

其中每个$h_m(X)$是集合C中的一个函数，或者是两个或多个这样的函数的乘积。模型从一个常数函数 $h_0 = 1$开始。在前向阶段的每一步中，我们的目标是从当前模型K中的所有函数$h_m$与集合C中的一个反射对中获得一个新的基函数对，以产生训练误差的最大减小。每一步的获胜乘积都会被添加到模型中，直到达到预设的最大项数。考虑到当前具有M个基函数的模型，要添加到模型中的下一个对的形式为

$$\acute{\beta}_{M+1} h_l(X)*(X_j - t)_+ + \acute{\beta}_{M+2} h_l(X)*(t - X_j)_+, h_l \in K, \quad (2.4)$$

每个 $\beta$通过最小化残差平方和来估计，即通过标准线性回归来估计。

图2.3展示了一个示例，说明了MARS算法如何利用基函数来拟合提供的数据模式。MARS的数学方程为

$$y = -5.0875 - 2.7678h_1(X) + 0.5540h_2(X) + 1.1900h_3(X) \quad (2.5)$$

![](img/5ce1e4105823f61cc4a022440f0309be_25_0.png)

图2.3 简单MARS示例的节点和基函数

节点位于 $x = 5$ 和 $17$ 处，将 $x$ 范围分为三个区间，识别出不同的线性关系。

MARS中的前向建模策略是分层的，因为高阶交互只有在其较低阶的“足迹”已经存在时才可能存在。例如，只有在模型中存在其两个二元乘积之一时，三元乘积才能被添加到模型中。这种策略有助于避免在指数级增长的替代空间中进行搜索。

通常，MARS过程中交互作用的阶数上限被设置为两个，以保持其高解释性，但不会对其适应线性结构和非线性行为产生太大的负面影响。当集合C中的反射对乘以常数函数 $h_0 = 1$ 时，得到线性项，否则形成非线性结构。为了防止特征空间边界附近的急剧增加或减少，每个输入在一个项中最多只能出现一次。

不需要预设依赖和独立变量之间的基本函数关系，相反，MARS允许模型自身通过添加一个节点和其对应的基函数对来确定最适合的线性和非线性结构，以在每一步中最大程度地减少残差平方和误差。这个过程在预定的情况下结束，通常会导致一个有过拟合的模型。

因此，采用了向后删除的方法，在每一步中删除对训练误差减少贡献最小的项，得到一个估计的最佳模型 $f_{\lambda}$ 。每个大小 $\lambda$ 的广义交叉验证用于优化 $\lambda$ ，准则定义为

$$\mathrm{GCV}(\lambda) = \frac{\sum\limits_{i=1}^{N} (y_i - \hat{f}_{\lambda}(x_i))^2}{(1 - M(\lambda)/N)^2}, \qquad (2.6)$$
$$\mathrm{M}(\lambda) = r + cK, \qquad (2.7)$$

其中， $M(\lambda)$ 表示模型中的有效参数数量， $r$ 表示模型中的项数， $K$ 表示前向过程中选择的节点数， $c$ 表示用于确定一个最优节点值的参数数量。一些数学和模拟结果表明，在分段线性回归中应使用三个参数来选择节点，因此通常情况下 $c = 3$ 。但在模型被限制为可加性时，使用 $c = 2$ 。

由于其高度灵活性，MARS通常被视为逐步线性回归的推广。除此之外，它还可以被视为CART方法的修改。

尽管它们看起来可能非常不同，但通过应用以下变化，MARS实际上可以转化为CART。

-   将分段线性基函数（方程2.1）改写为阶梯函数

```
I(x - t > 0) 和 I(x - t ≤ 0). (2.8)
```

-   当模型项参与乘法运算时，将其替换为交互项，以便进一步的交互。

第一个变化导致每个步骤中的阶跃函数乘以一对反射阶跃函数，这相当于在步骤中分割一个节点。第二个变化对应于CART模型中同一节点只能分割一次的事实，这导致CART无法处理加性结构。通过允许某个模型项在与候选选项相乘时仍能进一步交互，MARS放弃了CART模型的二叉树表示，从而更有效地适应了加性结构。

与CART类似，MARS不仅在回归问题上表现出色，而且还能高效处理分类问题。

通常情况下，对于二元分类问题，可以将输出编码为0/1，并将其视为回归问题。对于多类别分类问题，可以使用指示器响应方法。K个响应类别通过0/1指示变量进行编码，然后执行多响应MARS回归，在所有响应变量中使用一组共同的基函数。最终，输出是具有最大预测值的类别。然而，这种方法存在潜在的掩盖问题，更好的方法是使用“最优评分”方法（Hastie等，2017年）。

具体而言，有一种名为PolyMARS的MARS混合模型，由Kooperberg等人（1997年）设计用于处理分类问题。它使用多重逻辑框架，在类似MARS的前向逐步方式下构建模型，但在每个阶段，使用多项式对数似然的二次近似来搜索下一个基函数对。一旦找到，就通过最大似然拟合扩展模型，并重复该过程，直到达到预设目标为止（Hastie等，2017年）。

# 第三章 机器学习与应用

## 3.1 监督学习

监督学习是通过使用带有标签的数据集来训练算法，以准确地对数据进行分类或预测结果的定义。当输入数据被输入模型时，它会调整权重，直到模型适当地拟合为止。这是交叉验证过程的一部分，以确保模型避免过拟合或欠拟合。一些用于监督学习的方法包括神经网络、朴素贝叶斯、线性回归、逻辑回归、随机森林、支持向量机（SVM）等。

### 3.1.1 人工神经网络

人工神经网络（ANN）是一种模拟生物神经网络结构和功能的数学模型（Agatonovic-Kustrin和Beresford 2000）。它使用大量的人工神经元连接来进行计算。网络由许多相互连接的“神经元”组成。每个神经元代表一个特定的输出函数，也称为激活函数。每层神经元之间的连接表示通过连接传递的信号的加权值，称为权重系数，它等同于ANN的记忆比例。网络的输出由网络的连接规则确定，ANN可以被理解为自然界中某种算法或函数的近似。

### 3.1.2 极限学习机

极限学习机是一种由黄等人（2006）提出的新型前馈神经网络训练方法。该方法随机生成输入层、隐藏层的连接权重和隐藏层神经元的阈值。在训练过程中，只需设置隐藏层神经元的数量，即可获得唯一的最优解。假设有N个不同的样本，其中$x_i = [x_{i1}, x_{i2}, ..., x_{in}]^T \in R^n$, $t_i = [t_{i1}, t_{i2}, ..., t_{im}]^T \in R^m$，具有隐藏层神经元M和激活函数$g(x)$的标准单隐藏层前馈神经网络可以表示为

![](img/5ce1e4105823f61cc4a022440f0309be_32_0.png)

$$\sum_{i=1}^M \beta_i g(w_i \cdot x_i + b_i) = O_j, j = 1, \cdots, N, \quad (3.1)$$

其中 $w_i = [w_{i1}, w_{i2}, \cdots, w_{in}]^T$ 表示输入层节点与隐藏层节点$i$之间的连接权重；$\beta_i = [\beta_{i1}, \beta_{i2}, \cdots, \beta_{im}]^T$ 表示隐藏层节点$i$与输出层节点之间的连接权重；$b_i$ 表示隐藏层的偏差系数；$w_i \cdot x_j$ 表示 $w_i$ 和 $x_j$ 的内积；$O_i = [O_{i1}, O_{i2}, \cdots, O_{in}]^T$ 表示输出值。具有隐藏层$M$和激活函数 $g(x)$ 的标准单隐藏层前馈神经网络可以以零误差近似训练样本，当：

$$\sum_{i=1}^{M} \| O_j - t_j \| = 0 \quad (3.2)$$

因此，$\beta_i, w_i, b_i$满足以下公式：

$$\sum_{i=1}^{M} \beta_i g(w_i \cdot x_i + b_i) = y_i, \quad j = 1, \ldots, N, \quad (3.3)$$

可以表示为 $H\beta = Y$，其中 $H$是神经网络的输出矩阵。由于随机选择的输入权重和隐藏层偏置是固定的，网络训练等效于求解最小二乘解 $\hat{\beta}$ 以下方程的最小值：

$$\min_{\beta} \| H\beta - Y \| \quad (3.4)$$

上述线性系统的最小二乘解为 $\hat{\beta} = H^{+} Y$，其中 $H^{+}$是隐藏层输出矩阵 $H$的Moore-Penrose广义逆。

极限学习机的学习算法主要包括以下三个步骤。首先，随机设置输入层与隐藏层之间的连接权重 $w$ 和隐藏层神经元的偏置 $b$。其次，计算隐藏层的输出矩阵 $H$。最后，计算输出层的权重，$\hat{\beta} = H^{+} Y$。

### 3.1.3 决策树

决策树是一种用层次、顺序结构递归地对数据进行分区的表示数据规则的方法 (Murthy 1998) 。决策树是一种常见的机器学习算法，它基于树结构进行决策。图3.4展示了建立决策树的流程图。$S$表示训练样本集，$A$表示分类样本集，$N$表示分类叶节点。

决策树算法包括迭代二分器3 (ID3)、C4.5、分类与回归树 (CART)、有监督学习中的Quest (SLIQ) 等 (Quinlan 1993; Breiman et al. 2017) 。在这些算法中，著名且常用的是Quinlan的ID3算法。C4.5和CART是从ID3决策树算法派生出来的。SLIQ算法是基于C4.5决策树分类算法的修改进一步得到的。在C4.5算法中，树是按照深度优先策略构建的，而SLIQ的结构是按照广度优先策略构建的 (Niuniu 2010) 。

### 3.1.5 遗传编程

遗传编程（GP）是一种从问题的高级描述开始自动创建计算机程序的方法（Oltean等，2009年）。

![](img/5ce1e4105823f61cc4a022440f0309be_37_0.png)

- 遗传编程（Wieczorek和Czech，2000年）：编码方法主要采用树结构，基于树结构的特殊交叉和变异操作符，不仅用于问题规划和离散优化，还用于回归分析和其他领域。（Koza，1992年；Angeline，1994年）对遗传编程进行了大量发展，并系统编辑了GP的书籍。GP开始广泛应用于自动编程和数值与组合优化问题。
- 编码和解码：遗传编程主要使用语法树进行编码，而不是传统的线性编码。图3.8显示了表达式（A1 + 10）÷（A2 - A3）+（20 - A3）× A5的树形编码。该表达式包含变量 A1、A2、A3、A5和常数10、20。这些变量和常数位于树的叶子上，被称为终止符。

### 3.1.6 朴素贝叶斯

朴素贝叶斯（NB）是一种基于贝叶斯定理和属性之间的绝对独立假设的分类器。由于朴素设计和简化的假设，NB模型可以高效工作，并且只需要相对较小的数据库来优化超参数（Bhargavi和Jyothi 2009）。
一般来说，NB模型的构建包括五个主要步骤：(i) 收集数据，(ii) 估计每个类别的先验概率，(iii) 估计各类别的均值，(iv) 生成协方差矩阵并找到每个类别的逆矩阵和行列式，(v) 构建每个类别的判别函数（Bhargavi and Jyothi 2009; Pham et al. 2017）。
假设$x = (x_1, x_2, ... x_n)$是输入向量，$y = (y_1, y_2, ... y_K)$是输出向量。NB模型的分类过程可以表示如下：
```
$$\hat{y} = \arg\max_{k=\{1,2,...,K\}} = \frac{P(y_k) \prod_{i=1}^{n} P(x_i|y_k)}{P(x)}$$ (3.5)
```

### 3.1.7 支持向量机

支持向量机（SVM）是一种监督学习算法，最早由Vapnik（2000）提出，并逐渐在分类和回归问题中获得了更多的流行。 在接下来的内容中，将介绍SVM模型的一些基本原理。
假设有一个包含 \(l\) 个数据点的特定训练集，如 \((x_1, y_1), (x_2, y_2), ..., (x_l, y_l)\)，其中 \(x_i \in R^n\) (i = 1,2,...,l)是输入变量， \(y_i \in R\) (i = 1,2,...,l)是相应的目标值。根据Xue等人（2014）的说法，支持向量回归（SVR）的目标是解决以下优化问题：最小化
```
\[ \frac{1}{2} \|w\|^2 + C \sum_{i=1}^{l} (\xi_i + \xi_i^*) \] (3.6)
```
受到
```
\begin{cases} 
y_i - \sum_{j=1}^{n} \sum_{i=1}^{l} w_j x_{ji} - b \leq \varepsilon + \xi_i \\ 
\sum_{j=1}^{n} \sum_{i=1}^{l} w_j x_{ji} + b - y_i \leq \varepsilon + \xi_i^* \\ 
\xi_i, \xi_i^* \geq 0 
\end{cases}
```
其中
```
f(x) = \sum_{j=1}^{n} w_j x_j + b \quad \text{与} \quad w \in R^n, b \in R \quad (3.8)
```
关于这三个方程的一些详细信息如下： n在方程(3.7)和方程(3.8)中，是用于训练和测试的所有变量的数量，而 b表示偏差。j是对应输出的 l个输入的权重。 系数 C保证了 f(x)的简洁性和预测准确性。 当预测结果与观测值之间的误差小于 ε时，这是允许的，不需要惩罚，其中 ε被称为图3.14中的无敏损失函数。除了 ε之外，图3.14中还有两个松弛变量， ξ和 ξ*来确定对样本的惩罚程度。在构建SVR模型时，有几种常见的核函数，包括线性核函数、多项式核函数、径向基函数(RBF)等。(Zhang et al. 2016)。 为了获得一个理想的SVR模型，可用数据通常被分为训练集和验证集。 首先利用训练集建立一个粗略的模型，然后采用验证集进行调整

## 3.2 无监督学习

无监督学习使用机器学习算法分析和聚类无标签数据集。这些算法可以发现隐藏的模式或数据分组，无需人工干预。它能够发现信息中的相似性和差异性，是探索性数据分析、交叉销售策略、客户分割、图像和模式识别的理想解决方案。它还用于通过降维过程减少模型中的特征数量；主成分分析（PCA）和奇异值分解（SVD）是两种常用的方法。
无监督学习中使用的其他算法包括神经网络、k均值聚类、概率聚类方法等。

### 3.2.1 K均值聚类

K-means是一种典型的无监督学习算法，它将样本分成几个不相交的簇(Jain 2010)。聚类结果具有高内部相似性和低间部相似性。K-means算法的思想相对简单。假设数据要分成K个类别，步骤如下：
```
K-means算法首先初始化 \( k \) 个不同的质心 \( \{\mu^{(1)}, \dots, \mu^{(k)}\} \)。计算每个样本与质心 \( k \) 之间的距离，并选择与质心 \(\mu^{(i)}\) 最接近的簇作为样本 \( i \) 的类别。然后重新计算每个簇的质心，并再次调整每个样本的类别，直达到最大迭代次数或调整范围小于阈值。由于每次都要计算所有样本与每个质心之间的相似度，所以K-means算法在大规模数据集上的收敛速度相对较慢。
```

### 3.2.2 主成分分析

在数据科学领域，我们经常面临过拟合的问题。减少原始特征空间的维度并简化隐藏在数据库中的结构是一种高效的解决方案。为了降低维度，有两种常用的技术，即特征消除和特征提取。特征消除用于剔除对目标贡献较小的不重要变量。这是一种简单的方法，并且尽可能保持可解释性。特征提取用于生成新的独立变量，以解释数据库中的相关性，每个新创建的变量都是原始变量的组合。主成分分析（PCA）是一种广泛使用的特征提取技术。它以特定的方式组合输入变量，然后可以将最重要的独立主成分用作进一步分析的预测变量。它具有几个优点，但由于原始变量完全被消除，模型的可解释性会降低（Bakshi 1998年）。

## 3.3 半监督学习

半监督学习在监督学习和无监督学习之间提供了一种折中的方法。 在训练过程中，它使用较小的标记数据集来指导从较大的未标记数据集中进行分类和特征提取。半监督学习可以解决标记数据不足（或无法负担足够的标记数据）以训练监督学习算法的问题。

## 第四章 深度学习及应用

深度学习试图模仿人脑，尽管远远不能与其能力相匹配，但能够使系统对数据进行聚类和预测，具有令人难以置信的准确性。 它由多个相互连接的节点层组成，每一层都在前一层的基础上进行改进和优化预测或分类。 这种通过网络进行的计算过程称为前向传播。 深度神经网络的输入和输出层称为可见层。 输入层是深度学习模型摄取数据进行处理的地方，而输出层是最终进行预测或分类的地方。
另一个称为反向传播的过程使用梯度下降等算法来计算预测中的误差，然后通过向后移动层来调整函数的权重和偏差，以训练模型。 前向传播和反向传播共同使神经网络能够进行预测并根据错误进行修正。 随着时间的推移，算法逐渐变得更加准确。 上述描述了最简单类型的深度神经网络，用最简单的术语来说。 然而，深度学习非常复杂，有不同类型的神经网络来解决特定的问题或数据集，例如自编码器、深度置信网络、卷积神经网络、循环神经网络和长短期记忆网络等。

## 4.1 AE

自编码器主要用于降维和压缩（Hinton和Zemel 1994）。 策略是尝试使输出等于输入，或者尝试重构数据。 自动编码器包括编码器和解码器。 编码器接收输入，然后将其编码为低维潜在空间中的向量，然后解码器负责解码向量以获得原始输入。
自编码器可以从网络中的输出中获得较低维度输入的特征表示，实现降维和压缩。

此外，它可以用于检索略有不同的输入数据，甚至更好的数据，这些数据可以用于训练数据增强、数据去噪等。

## 4.2 DBN

受限玻尔兹曼机（RBM）是一种具有生成能力的随机神经网络，可以通过输入学习概率分布（LeCun等，1989年；Hinton，1989年）。与其他网络相比，它最大的特点是只有输入和隐藏层，没有输出。在前向训练中，输入经过后生成相应的特征表示，然后在反向传播中，从这个特征表示中重构出原始输入。这个过程类似于自编码器，但是它是在一个单一的网络中实现的。

多个受限玻尔兹曼机可以堆叠在一起形成一个深度信念网络（DBN）。它类似于全连接层，但训练方式不同。DBN的训练是根据RBM的训练过程，逐层训练其网络层。最近，由于生成对抗网络和变异自编码器的出现，使用DBN和RBM的人越来越少。

## 4.3 CNN

CNN是一种深度学习算法，广泛应用于视频识别、图像分类、推荐系统和自动驾驶问题（LeCun等，1998年；Krizhevsky等，2017年）。与传统方法相比，CNN具有无与伦比的预测准确性和工作效率，因为预处理阶段需要较少的努力。CNN的核心特点在于构建块，它由许多相互连接的层组成，使用一组共享的权重和偏差。图4.1显示了CNN模型的典型架构，包含卷积、池化和全连接层。

## 4.5 LSTM

为了解决RNN面临的“梯度消失或梯度爆炸”问题，提出了LSTM（Hochreiter和Schmidhuber，1997年）。与RNN相比，LSTM更好地利用了序列问题的历史信息，并且对于复杂的序列学习问题更加适用。LSTM的主要改进在于“记忆单元”，如图4.3所示。LSTM中的记忆单元包含三个特别设计的门，即输入门、遗忘门和输出门。输入门控制应该传递多少新信息到细胞状态。遗忘门确定细胞状态中应该保留或删除的信息。输出门计算输出结果，这个过程通常由一个sigmoid函数完成。总之，基于当前输入和先前输出的组合，LSTM模型预测下一个时间步的目标值。在达到所需的误差或最大迭代次数之前，这个过程将持续整个生命周期。

## 第五章 优化算法及应用

当你在尝试做出关于涉及大量因素的复杂决策时，优化可以帮助你捕捉关键组成部分，建立工程情况的数学模型，让你更快地做出更好的决策。

优化问题可以分为函数优化问题和组合优化问题。 函数优化问题的对象是在一定区间内的连续变量，解决连续性问题。组合优化的对象解空间是离散状态，解决离散问题。 典型的组合优化包括旅行商问题（TSP）、调度问题、背包问题、装箱问题、图着色问题、聚类问题等。优化问题还可以分为精确算法和近似算法（启发式算法）。 前者通常非常复杂，只适用于解决小规模问题，在地质工程中不实用。 后者基于直觉或经验构建，为要解决的每个组合优化问题实例给出可行解，时间和空间成本可接受。 可行解与最优解之间的偏差无法预测。 元启发式算法是启发式算法的改进。 它是随机算法和局部搜索算法的结合。 元启发式算法还可以进一步分为基于个体的算法，如禁忌搜索算法和模拟退火算法，以及基于群体的算法，如进化算法和前述的新型群体智能优化算法。 如今，在地质工程和地球科学中，群体智能优化的应用占主导地位。

## 5.2 进化算法

进化计算是一种工程和优化方法，其解决方案不是基于第一原理构建的，而是通过模拟达尔文进化元素的过程进行演化。进化计算是所谓的“自然启发计算”中的主要方法之一。进化算法是进化计算中的主要对象。存在一个需要解决的问题，解决方案被认为位于可能候选解的空间中——搜索空间。根据文献调研，常见的进化算法包括:

### 5.2.1 GA

遗传算法是模拟进化计算的最具代表性策略之一，由Holland（1992）提出。它是一种模拟自然生物进化的随机全局搜索和优化方法。该方法在搜索过程中获取和积累关于搜索空间的知识，并自适应地控制搜索过程以获得最优解。

它主要采用交叉和变异操作实现全局搜索，有效地减少了传统算法易陷入局部最优解的缺点。标准遗传算法包括编码过程、适应度函数和遗传操作。图5.2展示了遗传算法的网络可视化。

### 5.2.2 BO

Snoek等人（2012）在2012年提出了用于超参数调优的BO算法。由于其出色的能力和节省时间的优势，它已广泛应用于深度学习、系统优化、环境监测等领域，为解决复杂优化问题提供了新的方向。

BO可以被看作是找到使未知目标函数达到最大或最小值的最优参数的过程。 这个过程涉及到贝叶斯优化算法的两个核心部分，即概率代理模型和采集函数。 BO算法使用概率代理模型来拟合真实的目标函数，并根据采集函数主动选择下一个评估点的潜力最大，从而可以在少数评估次数内获得目标函数的最优解，并避免不必要的采样。 图5.3展示了BO算法的网络可视化。

### 5.2.4 DE

DE是一种随机、简单且高效的进化算法，通过Storn和Price（1997）在许多无约束测试函数中寻找全局最优解。与其他进化算法相比，DE保留了基于种群的全局搜索策略，并使用了简单的差分变异和一对一竞争的操作，因此可以减少遗传操作的复杂性（Huang和Chen 2013）。

同时，差分进化算法的特殊内存容量使其能够动态跟踪当前搜索，并通过强大的全局收敛性和鲁棒性来调整搜索策略。因此，它适用于解决一些复杂的优化问题（Huang和Chen 2013）。差分进化算法的主要过程包括三个主要操作符：变异、交叉和选择。在迭代过程中，假设每一代G的种群包含N个个体。初始种群通常是根据变量域上的均匀分布随机生成的。

变异操作用于生成一个个体，通过将两个种群个体之间的加权差异添加到第三个个体中。交叉操作用于多样化新种群。根据交叉策略，旧个体和新个体交换部分代码以形成新个体。选择操作是贪婪策略，它比较候选个体和目标个体的成本函数值，以确定最佳个体。图5.5提供了差分进化算法的网络可视化。

### 5.2.5 ABC

ABC是一种基于群体智能的元启发式算法，用于解决优化问题。它受到蜂群蜜蜂的智能觅食行为的启发 (Karaboga 2005)。在ABC算法中，蜂群中有三种类型的蜜蜂：就业蜜蜂 (觅食蜜蜂)，旁观蜜蜂 (观察蜜蜂) 和侦察兵。每种类型的蜜蜂都处理特定的任务。就业蜜蜂开发和搜索食物来源，对于每个食物来源只有一个就业蜜蜂。就业蜜蜂与旁观蜜蜂在蜂巢中共享信息，以便旁观蜜蜂可以选择一个食物来源进行觅食。侦察兵与通过探索蜂巢周围的环境寻找新的食物来源相关联。蜜蜂通过摇摆舞来展示被探索的食物来源的花蜜含量，舞蹈的持续时间与食物来源的质量成正比。旁观蜜蜂在选择一个好的食物来源之前会观看许多舞蹈，根据该食物来源的潜在质量进行选择。当蜜蜂 (无论是侦察兵还是旁观蜜蜂) 找到一个食物来源时，它就成为就业蜜蜂。当一个食物来源被丢弃时，与之相关的就业蜜蜂将再次随机成为侦察兵或旁观蜜蜂。研究发现，侦察兵负责探索工作，而就业蜜蜂和旁观蜜蜂负责开发工作 (Garg 2014)。ABC算法中有三个重要的控制参数，即群体大小、限制和最大周期 (Yusup et al. 2014)。更详细的计算过程可以参考 (Karaboga and Akay 2009)。

### 5.2.6 ACO

ACO是一种最近开发的基于群体的技术，由Dorigo和Gambardella (1997) 应用于解决优化问题。该算法受到真实蚂蚁群体觅食行为的启发。在寻找食物时，蚂蚁以随机方式探索其巢穴周围的环境。如果一只蚂蚁找到食物源，它会评估并携带一些食物返回巢穴。在返回的过程中，蚂蚁通过在路径的边缘上释放一定量的信息素来标记路径。信息素包含食物源的数量和质量信息，其他蚂蚁会被信息素吸引，引领它们找到食物。

信息素路径代表了蚂蚁之间的间接通信方式 (也称为信息素传递)，使得寻找巢穴和食物源之间的最短路径成为可能 (Nait Amar等, 2018)。图5.7提供了ACO算法的网络可视化。

### 5.2.7 CS

CS是由Yang和Deb（2009）开发的一种自然启发的基于种群的随机元启发式算法。该算法的主要思想受到了一些杜鹃鸟物种的寄生行为以及Lévy飞行的启发，后者是鸟类寻找食物的一种运动方式。在自然界中，一些杜鹃鸟会将它们的蛋放在其他鸟类的巢中，选择适合的巢，其中寄主鸟最近下了蛋，并移除已有的蛋，以增加自己蛋的孵化概率。杜鹃搜索算法可以基于三点理想化条件进行描述（Kanagaraj等，2014；Zhang等，2019）。关于该算法更详细的构建过程可以参考Yang和Deb（2009）。图5.8展示了CS算法的网络可视化。

### 5.2.10 WOA

### 5.2.11 SFLA

SFLA由Eusuff和Lansey（2003）提出，是一种用于解决离散优化问题的元启发式算法（Elbeltagi等，2005年）。它起源于青蛙的进食行为。Memetic算法（MA）是一种由Moscato（1989年）使用的群体智能算法，用于通过启发式搜索解决优化问题。“Memetic”意味着它是存储在人类或动物大脑中以指导其行为和传播的信息，类似于基因中的染色体。SFLA本质上结合了基于遗传的多智能体和基于社会行为的粒子群优化算法的优点。图5.12描述了SFLA的工作机制。

# 第6章 在边坡位移预测中应用LSTM和Prophet算法

## 6.1 简介

中国是世界上地质灾害最严重的国家之一（Wang等，2004年; Keqiang等，2008年; Li等，2010a年; Du等，2013年）。在某些情况下，山体滑坡会导致人员生命和财产的巨大损失，同时对环境造成重大影响（Yin等，2010年; Zhang等，2020a年）。特别是由于高山地区的复杂地质条件和密集的人类工程活动，山体滑坡在三峡库区经常发生，存在陡峭的坡面和洪水（Xu和Niu，2018年; Xie等，2019年）。2003年7月13日的黔江坪滑坡摧毁了4家工厂和129座房屋，造成24人死亡，1000多人无家可归（Jian等，2014年）。这是三峡库区最具破坏性的滑坡之一，因为它发生在水库首次蓄水后不久（Wang等，2004年; Jian等，2014年）。

众所周知，季节性降雨和水库水位的周期性波动的耦合效应对边坡安全构成了巨大威胁(Li等，2010a; Du等，2013; Huang等，2017; Yang等，2019; Chen等，2020; Wang等，2020a; Zhang等，2020a)。然而，滑坡是复杂的、动态的和非线性的系统，其失效机制尚未被研究人员充分探索(Xu和Niu，2018)。近几十年来，为了更好地控制灾害风险并发出预警，在滑坡易发区广泛使用了监测网络和控制系统(Li等，2010a)。基于大量记录的数据，可以分析滑坡位移与其触发因素之间的复杂关系。滑坡位移预测模型的准确性取决于如何表达和反映这些复杂关系(Xu和Niu，2018; Yang等，2019)。

传统的滑坡位移方法，如物理实验、数值分析和经验公式，具有一定的优势，但也存在一些缺点（Huang等，2017年; Xu和Niu，2018年; Yang等，2019年）。例如，实验室实验在揭示滑坡演化和增强方面表现良好对于失效机制的理解，例如基于蠕变理论和土壤岩石力学。另一方面，这些实验成本高，与数值模拟相比需要更多的时间和精力。最近，随着人工智能的发展，许多数据驱动模型已被用于滑坡位移预测（Li等，2010a；Du等，2013年；Zhou等，2016年，2018年；Huang等，2017年；Xie等，2019年；Yang等，2019年）。与上述方法相比，数据驱动模型因在计算成本和模型准确性之间取得平衡而越来越受欢迎（Lian等，2015年；Wang等，2020b年；Zhang等，2020b年）。因此，可以推断出在未来的滑坡问题中，传统方法和人工智能方法的结合将更加适用。

在预测滑坡位移方面，人工智能方法主要分为两大类：考虑触发因素综合影响的方法和考虑滑坡位移本身的方法（Xu和Niu，2018）。在这些方法中，人工神经网络（ANNs）、极限学习机（ELMs）和支持向量机（SVMs）是常用的预测滑坡位移的方法（Du等，2013；Lian等，2015；Zhou等，2016，2018；Cao等，2016）。例如，Du等（2013）使用反向传播神经网络（BPNN）方法结合时间序列模型预测滑坡位移。Cao等（2016）提出了一种极限学习机模型，分析了位移特征，并使用时间序列表达。Zhou等（2016）比较了改进的支持向量机模型和BPNN方法预测的位移时间序列。这些例子说明了时间序列分析和机器学习方法相结合在预测滑坡位移方面越来越受欢迎。

利用时间序列模型，将滑坡位移分为两个主要部分：长期趋势主要受内部因素（即地质条件和材料性质）控制，周期性位移主要受外部因素（即降雨和水库水位）的显著影响。长期趋势更加时间相关且增长缓慢，可以通过多项式函数进行高精度拟合。关于周期项，其周期性与季节性降雨和定期水库调度密切相关，因此预测更加复杂（Du等，2013年；Xu和Niu，2018年；Yang等，2019年）。

最近，长短期记忆（LSTM）模型因其能够有效反映降雨和水库水位耦合效应下滑坡位移的动态响应而受到广泛关注（Xing等，2019年；Zhang等，2019年；Jiang等，2020年）。LSTM模型最早由Hochreiter和Schmidhuber（1997年）提出，可以被视为改进的循环神经网络（RNN）（Sak等，2014年）的方法，它使用特殊的“记忆”结构更好地控制和判断历史信息。例如，Fan等（2014年）研究了LSTM模型在TTS语音合成中的应用。Weytjens等（2019年）发现，与ARIMA等经典预测方法相比，LSTM模型在时间序列预测方面特别有用。Kong等（2017年）高效地使用LSTM模型预测了个体住宅的短期负荷预测问题。

在许多先前的研究中，包括滑坡位移预测问题，LSTM模型已被证明是一种可靠的建模方法。例如，张等人（2019）发现LSTM模型在预测滑坡位移方面比支持向量机（SVM）和Elman神经网络（ENN）模型表现更好，尽管它需要更多的训练时间。杨等人（2019）分别比较了支持向量机和LSTM模型的预测位移，并验证了后者的更高准确性。邢等人（2019）通过堆叠神经网络层提出了一种改进的LSTM模型，并成功应用于中国的大水田滑坡。然而，LSTM模型在滑坡位移预测方面的性能仍需要通过更多案例研究来探索和改进。

本节采用LSTM模型基于时间序列分析和触发因素选择来预测TGR地区巴子门滑坡的位移。预测模型的实施包括四个主要任务：

- 1. 将总位移分解为趋势项和周期项；
- 2. 使用非线性回归函数拟合趋势项；
- 3. 通过分析周期位移与降雨和水库水位之间的关系来构建模型以预测周期位移；
- 4. 将预测的趋势位移和周期位移相加作为预测的总位移。

还使用了支持向量机（SVM）、MARS和Prophet模型来验证结果并比较模型性能。

## 6.2 方法论：Prophet模型

前几节已经介绍了机器学习的基本方法，只有先知模型会被基本介绍。

先知是由Facebook开发的开源算法（Taylor和Letham，2018），它已经被用于基于加法模型的时间序列分析（Zunic等，2020）。它将时间序列数据分解为三个主要组成部分：趋势、季节性和假期（Yan等，2019）。该模型的控制方程可以表示如下（Taylor和Letham，2018）：

```
y_t = g(t) + s(t) + h(t) + ε_t \quad (6.1)
```

其中，g(t)是反映原始数据中长期组成部分的趋势函数，s(t)表示与时间效应（例如年度、月度、周度和日度季节性）密切相关的周期性组成部分，h(t)表示不规则发生的假期的剧烈变化，ε_t是假设为正态分布的误差项。

长期趋势是滑坡位移最重要的组成部分，主要受内部因素控制，例如地质条件和材料性质。Prophet中实现了两种趋势模型：饱和增长模型和分段模型。考虑到本研究中滑坡位移的类似阶梯特征采用分段模型来建模长期趋势，并可以表示为以下形式：

$$g(t) = \frac{C(t)}{1 + \exp(-(k + \alpha(t)^T\delta)(t - (m + a(t)^T\gamma))))} \tag{6.2}$$

其中 \(C(t)\) 是系统在任意时间点的承载能力， \(k + \alpha(t)^T\delta\) 是时间变化的增长率， \(\delta\) 表示速率调整， \(m\) 是偏移参数，而 \(\gamma\) 被设置为正确的调整以使函数连续。

由于外部因素的显著影响，如季节性降雨和水库水位的周期性波动，滑坡位移通常呈现出类似的发展模式，即周期性（张等，2020a）。为了适应和预测这些影响，季节性函数应包含时间的周期函数。Prophet使用傅里叶级数提供了一个灵活的模型。季节性效应 \(s(t)\) 可以表示如下：

$$s(t) = \sum_{n=1}^{N} \left( a_n \cos\left( \frac{2\pi n t}{P} \right) + b_n \sin\left( \frac{2\pi n t}{P} \right) \right) \tag{6.3}$$

其中 \(P\) 是预期作为时间序列数据特征的常规周期（例如，年度数据的 \(P = 365\) 或每周数据的 \(P = 7\)）， \(N\) 是周期数。

为了商业目的，在Prophet中考虑了假日效应，以模拟和预测假日或一些重大事件上的异常变化。对于滑坡位移没有这样的影响，因此在这项工作中不予考虑。

### 6.2.1 模型开发

滑坡位移受内部和外部因素的影响很大，并且可以根据特定影响将其发展分解为两个组成部分：长期趋势和周期项。地质条件和材料性质等内部因素主要控制滑坡位移，并使其在长时间尺度上随时间单调增长。外部因素主要包括降雨和水库水位，它们对滑坡位移有滞后效应，使得地表变形周期性变化。累积位移可以被看作是以下两个组成部分的总和：

$$s(t) = f(t) + g(t) \tag{6.4}$$

其中，\(s(t)\) 表示监测系统记录的总位移，\(f(t)\) 表示长期趋势位移，\(g(t)\) 表示周期性位移，\(t\) 表示时间。

### 6.2.2 预测过程

本研究使用了四种机器学习方法来预测滑坡位移。通过数学函数和机器学习方法（LSTM、MARS和SVM），分别预测了两个位移分量（趋势项和周期项），如图6.1所示。由于Prophet模型和本研究中提出的模型具有类似的机制，我们直接使用前者来预测滑坡位移的总位移，而无需进行任何分解，这样效率更高。

因此，预测过程可以如下实施：

- (1) 监测到的总位移应分解为两个分量。

![](img/5ce1e4105823f61cc4a022440f0309be_83_0.png)

- (2) 趋势位移是时间相关的，随着时间的推移稳定增加，可以用多项式函数进行高精度拟合。

- (3) 周期性位移受降雨和水库水位的影响较大，其随时间的波动更加复杂。通过分析降雨、水库水位和位移之间的相关性，确定了几个控制变量（见第6.4.3.1节），使用上述方法预测三峡库区巴子门滑坡的周期性位移，如图6.1所示。在这部分中，使用一步预测方法来预测下一个时间步的周期性位移。总体而言，当需要预测准确性时，该方法在预测时间序列数据方面表现良好。然后，通过将预测的趋势位移和周期性位移相加，可以得到总位移。

- (4) 在同一阶段，可以使用Prophet模型直接预测总位移，完全使用记录的累积位移。这个预测模型不需要复杂的数据处理和关系考虑；它更容易实施。在这种情况下，Prophet被用来预测多步位移时间序列。与单步方法最大的区别是，这个方法可以预测更多时间步长的数据。

当决策者需要更多预测时间点以长期视角行动时，这可能非常有帮助。

- (5) 最后，将不同机器学习方法得到的预测总位移与监测数据集进行比较，以确定最佳模型性能。

整个数据集包含175个监测点，从2004年6月到2018年12月每月记录一次。除了用于测试的2018年数据集外，其余数据集被分为两个部分：80%用于训练，20%用于验证。

### 6.2.3 模型性能评估

这里使用了三个常见的统计准则来评估这些模型的性能，包括平均绝对误差（MAE）、均方根误差（RMSE）和决定系数（R²）。MAE的定义如下：

$$ MAE = \frac{1}{m} \sum_{i=1}^{m} |y_i - \hat{y}_i| \qquad (6.5) $$

其中，$m$是样本数量，$y_i$是第$i$个样本的观测值，$\hat{y}_i$表示在同一样本$i$上的预测值。

RMSE是一个与尺度相关的度量，它可以有效地比较应用于同一数据集的不同方法。这个准则的定义可以表示如下：

$$ RMSE = \sqrt{\frac{1}{m} \sum_{i=1}^{m} (y_i - \hat{y}_i)^2} \tag{6.6} $$

方程6.6中使用的符号与方程6.5中的符号相同。RMSE或MAE的值越低，模型的预测能力越强。

R²，由方程6.7定义，是皮尔逊相关系数的平方（ρ）通常用于评估模型对数据的拟合质量。它的取值范围从0到1，数值越高表示相关性越强。

$$ R^2 = \left[ \frac{\sum_{i=1}^{m} (y_i - \overline{y})(\hat{y}_i - \hat{\overline{y}})}{\sqrt{\sum_{i=1}^{m} (y_i - \overline{y})^2} \sqrt{\sum_{i=1}^{m} (\hat{y}_i - \hat{\overline{y}})^2}} \right]^2 \tag{6.7} $$

其中 $\hat{\overline{y}}$ 是预测结果的平均值， $\overline{y}$ 表示真实值的平均值，其他符号与方程6.5和6.6中相同。

### 6.3.1 背景

巴子门滑坡位于秭归县贵州村，位于湘西河右岸（$30^\circ58'16''N$，$110^\circ45'30''E$），是长江的一条主要支流。它距离湘西河与长江交汇处约0.8公里，距离三峡大坝38公里（图6.2）。坡度角度从10°到30°，初始滑动方向为N120°。滑坡的上边界位于岩石和土壤的交界处。左右边界均由同源沟定义。滑坡目前位于139-280米的海拔高度，前部分的原始高度在2003年三峡水库首次蓄水前为55米。滑坡呈扇形，下部滑动体较缓，上部较陡。它的面积约为$1.35 \times 10^5$平方米，纵向尺寸为380米，宽度为100-350米，估计体积为$2.0 \times 10^6$立方米。

巴子门滑坡的滑动体主要由砂质页岩、砂岩、砾石和黏土组成。根据钻孔数据，发现了两个主要的滑动面（Du等，2013年；Yang等，2019年）。较深的一个是位于10-35米深度的初始滑动面，较浅的一个是位于6-18米深度的次生滑动面。这两个滑动面的材料都是由粉砂和黏土以及碎石组成的。滑动带的厚度在中部较厚，在上部和下部较薄，如图6.3所示。

与控制因素相关的位移特征可以分析和总结如下：

- (1) 水库的调度与滑坡的累积位移之间存在密切关系：当水库水位增加或稳定运行时，位移几乎保持不变；每次水库水位下降，累积位移都会增加。水库水位在2007年至2018年间年度变化，累积位移相应增加。然而，位移并不会立即对水位下降做出反应。图6.4表明，水位对累积位移有滞后效应，后者在前者下降一个或两个月后开始增加。

- (2) 降雨也会影响滑坡的稳定性，尽管累积位移可能与这个因素弱相关（Zhang等，2020a）。如图6.4所示，在雨季期间，累积位移迅速增长，此时会有大量降雨。尽管每年8月底水库水位开始上升，但高强度降雨可能会在一定程度上减弱，而滑坡位移仍然在这个时期继续增加，表明降雨在促进滑坡位移方面起到一定作用。

- (3) 滑坡在降雨和水库放水的耦合作用下急剧变形，监测到的位移具有明显的阶梯特征，表明周期性因素的影响。此外，滑坡的变形速率在5月至8月的洪水季节突然增加，累积位移曲线呈陡峭跳跃，此时降雨量最大，水库水位下降速度最快。可以推断，在水库水位下降期间，叠加降雨条件将进一步加剧滑坡位移，这对滑坡的稳定性更加不利。

## 6.4 数据处理与结果分析

### 6.4.1 位移分解

如第6.2.1节所述，累积位移主要包含两个分量：长期趋势和周期项。从总位移中去除趋势项后，剩余部分是周期位移。许多研究表明，移动平均法适用于位移分解（Du等，2013年；Yang等，2019年），可以用于计算趋势位移，如下所示：

$$ f(t) = (y_t + y_{t-1} + ... + y_{t-k+1}) / k \quad (t = k, k+1, ..., N) \quad (6.8) $$

其中，$f(t)$ 是时间 $t$ 的趋势项；$y_t$ 是时间 $t$ 的累积位移；$k$ 是移动平均周期，在这里设置为6个月，因为雨季和旱季各持续大约半年；$N$ 是样本总数。

### 6.4.2 趋势位移预测

### 6.4.3 周期性位移预测

#### 6.4.3.1 控制因素

模型在数据上的性能主要通过准确性来评估，因此探索输入变量和目标值之间的功能关系非常重要。经分析，季节性降雨和水库水位的年度波动的耦合效应在触发滑坡变形中起着重要作用。因此，像普通回归方法一样，控制因素（即降雨和水库水位）被用作输入序列，周期性位移被作为输出值在这项工作中。

除了外部控制因素外，不同演化阶段的变形状态也会影响预测准确性。正如杜等人（2013年）所报道的，影响因素的性能取决于边坡是否稳定。这意味着即使重要的触发因素在边坡稳定条件下可能不会对边坡位移造成显著威胁。相反，在极不稳定的条件下，即使轻微的干扰也会导致山体滑坡严重变形。因此，在考虑对滑坡位移的影响时，稳定状态应该优先考虑。这可以通过以前的数据来揭示，并且一些研究也表明在考虑这个因素时关注模型的必要性和效率。

#### 6.4.3.2 LSTM模型的开发

整个数据集被分为三部分：2018年的数据用于测试，剩余数据集的80%用于训练，最后的20%用于验证（图6.5）。训练数据集用于模型构建，验证数据集用于验证预测能力。考虑到数据集的规模，其中包含175个数据点，测试数据集也被用于检查模型的泛化能力。在某些情况下，当数据集只有少量数据点时，插值方法可能会有帮助，可以省略测试数据集。模型构建使用了Python环境中的TensorFlow 2.0。

正如前面所提到的，LSTM模型对数据结构非常敏感，归一化可以在一定程度上避免收敛问题。因此，输入变量和目标值都被归一化到[-1, 1]。使用网格搜索方法对模型进行优化，它可以在模型准确性和计算时间之间取得平衡。该方法的过程可以表达如下：在训练集上通过交叉验证方法组合每个参数的合理值，然后可以自动确定最佳参数组合。本质上，它是一种遍历方法，根据模型的复杂性，确定最优参数组合可能需要花费一定的时间。考虑到数据集的规模，开发了一个四层的LSTM模型，前三层为LSTM层，最后一层为密集层。

LSTM模型使用先前的数据（选择的变量作为输入序列）来建立预测目标值的潜在关系，时间步长是一个重要参数需要考虑，它显示了用于预测的前几个时间步长的数据。网格搜索方法表明，在这项工作中，最佳时间步长为3，这意味着最新的数据对构建LSTM模型更有用。网格搜索方法还用于优化本文中提到的其他三种机器学习方法的参数：MARS、SVM和Prophet。

#### 6.4.3.4 预测累积位移

如第6.2.1节所述，对于MARS、SVM和LSTM模型，预测的累积位移应该是预测趋势和周期性的总和。位移趋势位移可以通过公式（6.9）获得，该公式基于历史位移进行拟合。对于Prophet模型，由于其机制与本研究提出的模型相似，直接使用该模型根据记录值预测总位移。该模型不需要进行位移分解过程，也不需要先前时间步长的先前数据来预测多个时间步长的位移。

与其他三种试图找到滑坡位移与外部触发因素之间关系的方法相反，Prophet模型试图捕捉位移的时间特征。因此，对用户来说非常方便和简单，因为该模型不需要找到候选变量，而是位移本身。图6.8显示了Prophet模型对近3年位移（2016年2月至2018年12月）的预测，基于从先前数据（2004年6月至2016年1月）中学习到的探索模型。尽管预测的总位移与监测数据不完全一致，但这个模型很好地捕捉到了逐步发展的模式。从某种意义上说，这是一个真正的预测模型，因为它基于学习到的发展模式进行预测，而不仅仅是输入和输出值的简单函数，更重要的是，该模型除了历史位移之外不需要额外的输入变量。

在特定长度的序列中，包含有历史周期性信息，以预测当前时间步的目标值。与本文提到的其他三种方法不同，该模型可以充分利用历史信息，基于特殊的“门”设计。这也使得它在构建、开发和测试方面是最耗时的。LSTM模型在不同时间步的先前输入变量和目标值之间建立动态连接，这意味着学习经验会持续被利用和修改用于当前任务。

在这个过程中，模型可以通过记忆有用的信息和删除无效的规则来更新潜在规则。换句话说，这个模型的记忆单元使得更好地反映了滑坡的动态演化过程成为可能。对于相同的数据集，两种经典的机器学习方法，SVM和MARS模型的模型性能较差。缺乏类似的记忆这可能是两个模型中主要的缺陷。这两个模型不会在不同时间步骤的先前数据和目标值之间建立连接，这意味着模型只能使用特定时间点的历史信息来预测相应的位移。然而，最新的数据结构与历史数据不同，这可能在一定程度上导致错误。因此，一些建议是，为了建立更准确可靠的预测模型，应及时更新数据集，删除旧数据并添加新数据（Du等，2013年；Yang等，2019年）。

至于Prophet模型，其性能不如上述三个模型。然而，它的优势并不在于短期预测的准确性。该模型自动实现了时间序列分析，并基于学习到的趋势和季节特征预测目标值。此外，该模型不需要复杂的数据处理和变量选择，这意味着它更容易和方便进行。此外，由于该模型基于时间和位移之间的特征捕捉，因此可以预测比下一个时间步长更长时间段内的目标值。可以合理推断，该模型可能非常适用于工程目的，其中成本控制和时间管理是从更长期的角度规划而不是特定的时间步长。

## 6.6 本章小结

在本节中，应用了四种机器学习方法来预测边坡位移，包括MARS、SVM、LSTM和Prophet。这些模型的预测结果通常与监测到的位移具有良好的一致性，这表明所提出的模型可以处理滑坡位移与外部触发因素（如季节性降雨和水库水位的周期性波动）之间的动态和非线性关系。

与其他三种方法相比，LSTM模型在动态滑坡位移预测方面显示出显著的潜力，因为它建立了历史信息和当前任务之间的联系。记忆单元的特殊设计可能是其更好性能的主要原因。在给定新的输入时，该模型可以结合记忆的规则和先前的信息来实现更准确的预测。通过不断重复这个过程，该模型可以删除无效的规则并保留有用的规则以更新学习到的关系。此外，这个过程在整个模型的生命周期中持续进行。

SVM和MARS模型在预测能力上表现出类似的能力，但当外部触发因素变得不规则和罕见时，它们的偏差很大。静态机制可能是其中的原因。与LSTM相比，SVM和MARS都试图在特定时间点找到滑坡位移与输入触发因素之间的关系。这两种方法无法反映隐藏在函数背后的动态响应。

Prophet模型显示其在预测位移方面的优先性和便利性，因为它可以学习总位移与时间之间的关系。然而，它在预测多个时间步长的能力上取得了成果，但以模型准确性为代价。因此，它对于需要更长时间段信息以进行更好估计的成本控制和时间管理问题非常有用。

# 第7章 基于BO的XGBoost和RF预测无排水抗剪强度

## 7.1 引言

软敏感黏土通常具有低剪切强度和高压缩性，在近海环境中广泛分布。因此，涉及软敏感黏土的岩土工程设计相当具有挑战性（D’Ignazio等，2016年）。如今，在软敏感黏土上建筑的需求正在增加，考虑到最近基础设施发展和港口连通性项目的趋势。因此，通过可靠的方法准确评估无排水抗剪强度对岩土工程师来说是一个关键问题。

软敏感黏土的无排水抗剪强度可以通过各种室内试验和原位试验来评估。最常见的是通过原位试验来测试，例如典型的现场十字板剪切试验（FVT）和锥形圆锥穿透试验（CPTU）。一般来说，原位试验成本高且耗时。此外，越来越多的人一致认为无排水抗剪强度与主要的CPTU参数之间存在不可避免的相关系数的不确定性。至于实验室试验，需要相对复杂和昂贵的取样和实验技术（D’Ignazio等，2016年；Ching和Phoon，2018年）。必要的岩土工程设计参数并不总是直接从实验室和原位试验中测量得到，而是经常从经验或数值相关性中估计，这些相关性是通过对预定数据集进行回归拟合而得到的。

自Hansbo（1957）的开创性工作以来，基于经验或半经验回归分析提出了变换模型，用于评估土壤的相关设计参数，根据其性质（Mesri等，1975年；Jamiolkowski等，1985年；Kulhawy和Mayne，1990年）。然而，变换模型通常包含大量来自土壤性质、土壤行为和场地地质等方面的不确定性。这些可能与校准变换模型的数据源不同（Kulhawy和Mayne，1990年；D’Ignazio等，2016年）。因此，需要认识到这些模型的局限性，否则可能导致偏见作为直接后果。为了增强工程师的判断力，许多基于贝叶斯方法的概率变换模型已被提出。

包括(Ching和Phoon 2012, 2018; Wang和Cao 2013; Ching等人2014; Cao和Wang 2014; Wang和Aladejare 2016; Wang等人2018a,2019)。人工智能（AI）技术在过去几年中迅速发展，许多新颖的机器学习（ML）算法已被提出并广泛应用于各个领域，可能在这方面提供新的视角。这与国际土壤力学与岩土工程学会（ISSMGE）最新的倡议一致，该倡议探索岩土工程中的机器学习方法(Ching和Phoon 2018)。

机器学习算法，如人工神经网络（ANN）(Goh 1995; Teh等人1997; Youssef pour等人2011; Zhang等人2019)，贝叶斯网络（BN）(Li等人2018; Koduru 2019)，支持向量机（SVM）(Rodriguez-Galiano等人2015; Zhang等人2016; Zhou等人2016)，以及多元自适应回归样条（MARS）(Zhang和Goh 2013; Zhang等人2015; Liu等人2019)等，逐渐成为岩土工程问题的替代解决方案。然而，单一的复杂算法可能不是构建设计土壤参数代理模型的考虑因素，因为来自不同场地的数据库具有不同的土壤性质、土壤行为和场地地质。因此，在岩土工程中应用集成方法是必要的。集成学习结合了多个处理不同假设的算法，形成更好的假设，因此其预测性能良好（Nascimento等人2014; Xia等人2017a）。如(Nannini和Lumini 2009; Lessmann等人2015)所报道，集成方法优于单一的机器学习和其他统计方法。因此，鼓励在岩土工程中进一步应用集成方法是有前景的。

集成和应用SC、ML、DL和OA方法在地质工程和地球科学中可以广泛分为两种方法：装袋（并行）和提升（顺序）。装袋集成方法是不同学习算法的组合，每个算法在并行中生成一个独立的模型（Xia等，2017b）。作为一种著名的装袋集成学习方法，随机森林（RF）是可以理解的，因为它可以被看作是一种共识决策树的过程，每个决策树独立地对特定问题做出决策，然后将结果组合起来做出最终决策（Breiman，2001）。一些研究人员还应用RF来解决岩土工程问题，例如Zhou等人（2016）研究了使用RF预测隧道施工引起的地表位移的可行性。Zhang等人（2019）构建了RF和MARS模型来预测桩打击性能，并使用了大量数据。在提升中，决策树是通过提升的方式构建的，以便每个后续的决策树都旨在减少前一个决策树的错误。每个决策树都从前面的决策树中学习并更新残差错误。因此，随后生长的树更容易从更新的残差中学习（Friedman，2001）。极端梯度提升（XGBoost）是一种先进的监督算法，由Chen和Guestrin（2016）在提升框架下提出，由于其高鲁棒性和足够的灵活性，在Kaggle机器学习竞赛中得到了广泛认可。到目前为止，基于XGBoost的模型在岩土工程中很少使用。

受到上述好处的激励，本文开发了基于XGBoost和RF的集成学习方法，用于捕捉无排水抗剪强度与各种基本土壤参数之间的关系。为了减少对经验法则和蛮力搜索的依赖，本文采用贝叶斯优化方法来寻找合适的模型超参数。据作者所知，这是集成学习方法首次用于构建设计土壤参数的代理模型。

本文的剩余部分首先介绍了两种高效的集成学习算法和贝叶斯超参数优化方法，然后介绍了研究的数据集及其初步分析。然后，使用四个性能指标来分析和评估预测结果。最后一部分将开发的模型与三种比较机器学习方法和两种转换模型进行对比分析，以获得更有说服力的结果。

## 7.2 方法论

对于RF和XGBoost，前几章已经提供了详细信息，本节主要关注优化方法的方法论介绍。

### 7.2.1 贝叶斯超参数优化

在机器学习中，超参数是需要在学习过程之前预先设置的参数，算法很少是无超参数的。对于XGBoost、RF和比较的机器学习算法，都有几个对模型的预测准确性有很大影响的超参数。因此，合理调整这些超参数，即超参数优化，非常重要。然而，超参数优化是一个组合优化问题，不能像一般参数那样通过梯度下降方法进行优化。此外，由于调整超参数需要重新训练来评估效果，计算一组超参数配置的计算量非常大。

在机器学习实践中，常用的调整超参数的方法有网格搜索（GS）和随机搜索（RS）。GS是一种通过尝试所有超参数的组合来搜索合适的最优超参数配置的方法。然而，GS容易受到维度约束的影响，即在考虑更多参数时，进行超参数优化时需要评估模型的次数呈指数增长，这在XGBoost中被证明是不可行的。此外，甚至不能保证找到最优解，经常会错过最优配置。RS是一种选择合适的一组通过尝试超参数的随机组合来找到最佳超参数配置。RS的缺点在于不必要的高方差。该方法是完全随机的，在选择试验点时不使用任何智能。GS和RS都没有利用超参数不同组合之间的相关性。

越来越多的超参数调整过程现在通过自动化方法来完成，这些方法旨在使用基于策略的启发式搜索在更短的时间内找到最佳超参数。此外，除了初始设置外，不需要额外的手动操作。贝叶斯优化是优化目标函数的首选方法（Snoek等人, 2012年; Ghahramani, 2015年; Xia等人, 2017b年）。贝叶斯优化通过构建基于过去评估结果的概率模型来找到最小化目标函数的值。最近广泛应用于机器学习超参数调整。

结果表明，该方法在测试集上更具有普适性，并且比GS和RS (Bergstra等人，2013)需要更少的迭代次数。

## 树形帕尔森估计器（TPE）

超参数优化基本上是在图结构化配置空间上优化特定映射函数的问题。为了在标准 SMBO 算法中运行，在贝叶斯优化环境中可以使用各种概率回归模型，如高斯过程、随机森林和树形帕尔森估计器（TPE）等。

正如 Bergstra 等人（2011）所建议的，TPE 中的配置空间被限制为树状结构，这提供了确定模型 S（即表 7.1 中的第 6 行）和找到局部最优超参数设置（即表 7.1 中的第 3 行）的简单解决方案。该模型不定义目标函数的预测分布；相反，它创建了两个层次过程，l(λ) 和 g(λ) 作为所有域变量的生成模型。当目标函数低于和高于特定分位数 c* 时，这些过程对域变量进行建模。

$$
p_{S}(c|\lambda) = \begin{cases} l(\lambda), & c < c^{*} \\ g(\lambda), & c \geq c^{*} \end{cases} \qquad (7.1)
$$

其中 l(λ) 是通过关于损失值低于 H 中 c* 的观测构建的密度估计，而 g(λ) 则由剩余的观测生成。

同时，一个更重要的问题出现了，即确定一个本地最优的超参数设置 λ* 根据当前模型 S_t 进行 t 次循环。SMBO 采用获取函数来处理这一步骤，它收集 S_t 在任何可能的超参数设置下的预测信息，直到达到最大值，然后获取最有希望的设置。两种形式的获取函数被广泛采用，即期望改进（EI）和改进的概率。在本文中，由于其令人满意的性能和直观性，选择 EI 作为标准（Bergstra 等人，2011 年）。改进可以表示为：

$$
I(\lambda) = \max(c_{\min} - c(\lambda), 0) \qquad (7.2)
$$

实际上，改进很难直接估计，因为 c(λ) 在设置通过损失函数（即表 6.4 中的第 4 行）进行评估之前是未知的。然而，值得注意的是，EI 可以计算为：

$$
\text{EI}(\lambda) = \int_{-\infty}^{c_{\min}} \max(c_{\min} - c(\lambda), 0) \cdot p_{S_t}(c|\lambda) \mathrm{d}c \qquad (7.3)
$$

其中，c_min 表示 H 中的当前最小损失，c(λ) 表示在超参数设置 λ 下的损失。通过以上所有过程，前述两种集成学习算法和三种比较算法的超参数都可以通过贝叶斯优化进行选择，参数调整过程的一些细节可以参考 Bergstra 等人的研究(2013)。

## 7.4 实施步骤

### 7.4.1 K折交叉验证

理想情况下，如果我们有足够的数据，最佳实践是将数据集随机分成三部分：训练集、验证集和测试集。训练集用于拟合模型；验证集用于估计模型选择的预测误差；而测试集用于评估最终模型的泛化误差（Franklin 2005）。由于数据通常很稀缺（本文仅有336个数据点），无法真正反映模型的泛化性能是很常见的。为了避免数据选择的偏见，在超参数调优和模型评估过程中，本文采用了最流行的验证方法之一，即k折交叉验证（Kohavi and Kohavi 1995; Rodríguez et al. 2010; Wong 2015; Zhou et al. 2019）。在k折交叉验证中，原始样本D被随机分成k个互斥的子集，大小相似，即D = D₁ ∪ D₂ ∪…∪ D_k，D_i ∩ D_j = ∅（i≠j）。每个子集D_i尽可能地保持数据分布的一致性，即通过D的分层抽样。然后，每次将k-1个子集的并集作为训练集，剩余的子集作为测试集；这样就可以得到k组训练/测试集，从而进行k次训练和测试。虽然确定k的值没有明确的严格规则，但在应用机器学习领域，k的值通常为5（或10）是非常常见的。在这方面，根据Kohavi和Kohavi（1995）；Wong（2015）的建议，将折数k设置为五，并与计算时间和偏差之间的权衡相结合。因此，本文采用了五折交叉验证的方法来验证模型（见图7.4）。

### 7.5.3 特征重要性分析

特征重要性是特征选择和模型解释的重要参考。经过训练的基于XGBoost的模型可以自动计算特征重要性，可以通过接口特征重要性准则（即增益准则）获得。增益是通过计算模型中每棵树对每个特征的贡献来计算的，表示每个特征对模型的相对贡献。与另一个指标相比，该指标的值较高意味着该特征对生成预测的重要性更大。图7.9显示了五折交叉验证下五个特征变量的平均特征相对重要性（%）。在当前的XGBoost-based模型中，PS（54.3%）是最重要的特征变量，其次是通过VES（20.3%）、PL（12.5%）、W（6.8%）和LL（6.1%）进行。这个结果意味着对探索软黏土的无排水抗剪强度特性具有重要的指导作用，并且敏感性分析的类似结果也在(D'Ignazio等人，2016)中提出。

## 7.6 总结与结论

集成学习方法是非常流行和强大的多元回归和分类问题的机器学习工具。与基于经验证据的转换模型的广泛研究不同，数据驱动的集成学习方法在岩土工程中受到了有限的研究关注。本文首次使用基于XGBoost和RF的集成学习方法，用于预测软黏土的无排水抗剪强度。还将集成学习方法的性能与三种基准机器学习算法（即SVM、MLP和MARS）和前期研究中的两种转换模型进行比较，使用了来自F-CLAY/7/216和S-CLAY/7/168的组合数据库。根据这项研究，可以得出以下结论。

所有提出的机器学习模型都在数据库上使用四个性能指标进行验证，采用五折交叉验证。与基准模型相比，XGBoost和RF模型在预测性能方面表现出更高的优势，整体预测准确性的排序为XGBoost RF，SVM，MLP，MARS。

与先前的转换模型相比，机器学习方法在预测准确性方面具有明显优势。两个集成学习模型在测试集上的平均MAPE值分别为19.23%和19.63%，对于包含多源和一定噪声数据的数据集来说，这些MAPE值可以被认为是较小的。

采用贝叶斯超参数优化方法来调整机器学习算法的超参数。这个指标确保了所提出的模型能够充分发挥预测潜力，并且在捕捉USS与各种基本土壤参数之间的关系方面表现更好，优于先前的转换模型。

所提出的基于XGBoost的模型可以自动计算土壤参数在USS中的相对重要性，PS被确定为最重要的特征变量，其次是VES，PL，W和LL。这个指标增强了模型的解释性和综合性，并为实际应用中的岩土工程实践者提供了特征选择的指导。

# 第8章 使用四种超参数优化方法和RF模型进行TBM掘进率预测

## 8.1 引言

随着技术的不断发展和进步，隧道掘进机（TBM）的出现为隧道挖掘提供了更安全、更有效和更便宜的施工方法。由于这些优势，TBM技术已经在全球范围内广泛应用。为了降低项目成本和施工风险，对TBM掘进效率进行准确评估非常必要（Gong和Zhao 2009；Mahdevari等2014；Armaghani等2017；Adoko等2017；Liu等2018；Zhang和Gao 2018；Tan等2020），其中，穿透率（ROP）是一个关键评估因素（Alvarez Grima等2000）。然而，对于大多数工程项目来说，复杂的地质条件使得ROP预测充满了不确定性和随机性。一般认为，在隧道施工过程中，地质环境、TBM参数和隧道几何条件对ROP有重要影响（Blindheim等1979；Alvarez Grima等2000；Barton 2000；Sapigni等2002；Yagiz和Karahan 2011；Xu等2019；Zhang等2020）。如何确定每个潜在参数对ROP的影响，并分析每个参数的相对重要性，具有重要的工程实际意义。早期的预测模型主要基于经验、理论或两者的结合（Bruland 1999；Sapigni等2002；Kahraman等2003；Hassanpour等2009；Yagiz和Karahan 2011；Li等2016, 2020；Xiong等2017；Koopialipoor等2019；Zhang等2020；Yan和Jiang 2020；Hou等2020；Zhu等2020；Yan等2021）。基于三个隧道的监测数据，Sapigni等人（Yagiz 2008）发现ROP与岩体质量分类（RMR）密切相关，两者可以用二次多元回归方程近似。Kahraman等人（Breiman 2001）基于现场测量数据建立了ROP回归模型，并通过统计分析发现ROP与岩石性质密切相关。Hassanpour等人（Breiman 2001）根据Nowsood隧道数据建立了ROP和不同地质参数的经验公式。结果表明，ROP与岩石的可塑性参数，特别是现场穿透指数（FPI）密切相关，此外，本研究表明使用UCS和RQD等常见参数建立预测公式的便利性。然而，经验公式通常是基于局部数据建立的，只适用于少数项目，因此适用性有限（Breiman 2001）。数值分析在这类问题中也被广泛使用。（Gong和Zhao 2009）使用UDEC软件对新加坡两个隧道中的TBM刀具断裂过程进行了数值模拟，并发现TBM的掘进速度会随着岩石脆性指数的增加而增加。对实际数据的统计分析也显示了类似的观点，岩石脆性作为重要的机械参数，在TBM开挖过程中起着重要作用。（Li等人2019）使用离散元法（DEM）研究了ROP与楔角、约束应力等参数之间的关系。结果表明，较大的楔角会增加岩石的横向破坏，从而降低驱动速度。此外，原位应力的增加也会增加岩石的延性，不利于开挖。

数值模拟非常依赖于本构模型、材料参数和边界条件，但由于隧道工程的复杂性和不确定性，这些因素通常很难准确获取。因此，数值分析只能反映一般趋势，无法实现准确预测。

近年来，由于人工智能（AI）技术的发展，许多数据驱动模型已成功用于ROP预测。与传统方法相比，AI模型可以在很大程度上捕捉变量之间的复杂非线性关系，并进行更准确的预测。（Xu和Niu 2018; Chen等2019; Xu等2019）提出了五种不同的机器学习方法，基于马来西亚一条隧道的现场和实验室实验数据预测ROP。主要使用的输入参数包括单轴抗压强度（UCS）、巴西拉伸强度（BTS）、岩石质量指标（RQD）、风化带（WZ）、推力（TF）和转速（RPM）。通过将预测结果与实际值进行比较，发现K最近邻（KNN）算法可以实现最佳预测准确性。此外，参数敏感性分析发现UCS和RPM对ROP影响最大。以马来西亚一条隧道为例，（Jahed Armaghani等2017）建立了基于人工神经网络（ANN）的ROP预测模型，并使用两种优化算法（PSO和ICA）进行超参数调优。结果表明，优化模型的预测准确性远高于普通模型，证明了优化算法的必要性。目前，将AI模型应用于ROP预测的研究很多，但应用优化算法对相关模型进行超参数调优以解决计算精度和计算效率之间的平衡的研究还很少。（Eberhart和Kennedy 1995; Fabian Pedregosa等2011; Mahdevari等2014; Zafar等2017; Mahmoodzadeh等2020; Zhang等2020）本文根据已发布的数据集（Reshef等2011），提出了一个随机森林（RF）模型来预测ROP，并使用四种常见的优化算法（PSO、DE、GA和BO）来优化模型的超参数。最后，使用最优模型输出参数的相对重要性。

## 8.2 工程背景和数据介绍

本文使用的数据集来自文献（Reshef等，2011年）。数据集由来自隧道的151个不同位置的153个数据样本组成，包括岩石地质参数、隧道几何条件和掘进机的掘进速度等7个特征组成部分。由于RF模型本身适用于数据维度低、精度要求高的场景，本文使用的数据集对于类似的工程项目具有一定的代表性和指导意义。

### 8.2.1 项目描述和地质调查

纽约市建造了皇后水隧道的第二阶段，以改善城市的淡水分输和运输。隧道全长7.5公里，直径7.06米，深度约200米。施工过程由一台强大的隧道掘进机（TBM）完成。隧道沿线施工区域的地质结构非常复杂，由不同的变质岩和具有剪切带、节理和断层等局部弱结构面组成。

岩石组成主要包括流纹岩、花岗片麻岩、角闪岩、火成片麻岩和片岩，各自的比例如（Barton 2000）所示。数据集是从隧道的151个不同位置采样得到的，包括153个样本。还分析了每个采样点的TBM的ROP（推进速度）基于测量数据。因此，整个数据集包括岩石的强度、脆性和韧性特征。强度特征通过BTS和UCS反映，而脆性和韧性通过冲击穿透实验进行校准。对于冲击穿透实验，通常使用岩石样本能承受的最大载荷与相应最大位移之间的比值（峰值斜率指数，PSI）来反映岩石样本的脆性和韧性。

此外，为了反映岩石不连续性的类型和密度对TBM掘进速度的影响，还计算并记录了隧道轴线与软表面之间的夹角，并在数据集中进行了记录。根据NTNU开发的分类系统，岩石断裂级别（RC）可以根据其完整性分为7个类别，编号从1到7。数字越大，岩体的弱结构面越多，岩体的断裂程度越高。同时，数据集还记录了与分类结果相对应的弱面之间的平均距离（DPW）。

事实上，盾构机的行驶速度是一个复杂的动态非线性系统，与许多因素有关。线性相关性不能完全反映变量对ROP的影响程度，因此有必要引入非线性评估指标来进一步说明相关性。最大信息系数（MIC）的概念首次由（Reshef等人，2011年）建立，它可以用于综合评估两个变量之间的线性或非线性强度，属于基于最大信息的非参数探索统计类别。

其基本原理是将两个变量之间的关系离散化为二维空间，并用散点图表示。然后，当前的二维空间分别沿X和Y方向划分为一定的间隔数，然后检查散点落入每个方格的情况，即联合概率的计算。在获得互信息的最大值后，进行归一化，然后将不同尺度下的互信息最大值进行比较，以反映变量之间的关系。

## 8.4 讨论

使用四种常见的优化算法（PSO、GA、DE和BO）和随机森林模型（RF）构建TBM推进速度（ROP）的预测模型。岩石样本强度指数（UCS、BTS）、冲击穿透试验指数（PSI）、隧道岩石完整性（DPW、RC）和隧道与盾构方向几何条件（AN）作为输入，推进速度（ROP）作为输出。通过比较不同优化模型在训练集和测试集中的性能，发现BO-RF模型在预测准确性和计算时间方面表现最佳。主要原因是贝叶斯优化采用高斯过程（GP），可以考虑先前的参数信息并不断更新先验信息。其次，贝叶斯优化非常适用于小维参数调整要求，因为迭代次数少且收敛速度快。然而，其他优化算法采用梯度下降法，在操作上较慢并且在使用多个参数维度时容易导致维度爆炸，从而严重影响参数调整过程。因此，对于本文中的RF模型，BO优化显示出一定的优势，因为只有两个参数需要优化。对于相同类型的实际工程问题，当使用RF模型进行预测时，可以将BO算法作为首选项进行超参数优化，而PSO算法可以作为备用项来比较计算结果和计算成本。

## 8.5 总结

针对隧道掘进速度的复杂非线性动态系统，提出了基于四种不同优化算法的随机森林预测模型。将用于预测皇后区纽约地区的水运输隧道推进速度，尽管地质条件复杂，隧道施工区域模型输入参数较少，但模型预测与实际值具有高度一致性，并且模型输出参数的相对重要性和前期相关性分析的结果基本一致，表明模型的可靠性和适用性。通过分析和比较结果，可以得出结论，BO-RF模型在捕捉潜在影响因素与TBM掘进速度之间的复杂非线性关系方面表现最佳。该模型具有高准确性和较少的时间消耗，因此建议将其应用于此类问题的预测。PSO-RF模型的性能与GA-RF模型相似，但前者所需时间较少且更高效。DE-RF模型所需时间最长，但模型的性能并没有显著提高。该方法对TBM掘进速度的预测仍需进一步研究。敏感性分析结果显示，每个输入参数对TBM掘进速度的影响不同，岩石本身的强度参数是绝对影响因素。

作为一种成熟的机器学习方法，随机森林模型因其较少的需要优化的超参数、高计算效率和避免过拟合问题的能力，在实际工程问题中被广泛应用。然而，数据的大小和质量也直接影响模型的输出结果。因此，未来可以添加更多的数据样本来提高优化模型的性能，并对输入参数的相关性进行全面深入的研究。此外，预测模型的发展严重依赖于超参数的优化，这可能需要专业知识和历史经验，而一些智能模型的“黑盒子”特性使其难以解释，需要根据问题的需求进行应用。

# 第9章 我们从应用中学到了什么

众所周知，软计算和更先进的人工智能技术，包括机器学习、深度学习和优化算法，极大地提高了地质工程结构响应预测的准确性和效率。从第5章可以得知，与隧道和支护开挖相比，人工智能算法在桩基和管道领域很少使用。因此，建议未来在这两个领域中普及更多的算法使用。此外，根据收集的文献，单一算法模型在岩土工程中被广泛使用，如DT、SVM、RF、ANN、BPNN。集成或混合模型和优化算法的普及程度还有待进一步提高。图9.1简要介绍了机器学习的最新研发情况以及主要挑战和限制。

此外，传统的基于物理的模型是基于已知的地质结构响应和影响因素之间的关系构建的。然而，基于人工智能的模型只是建立了输入和输出之间的直接映射，并不能提供任何明确的表达式。因此，在未来的研究中，将物理模型和人工智能数据驱动模型相结合值得更多关注。此外，表9.1列出了上述大部分方法的优点和缺点。

自然灾害对人们的生命、财产和环境构成威胁。许多研究人员试图通过经验方程、实验室实验、数值模拟来模拟和更好地理解这些灾害。它们具有一定的优点，另一方面，它们在进一步应用方面受到了约束或限制。最近，人工智能方法越来越受欢迎，因为它们可以模拟自然灾害的复杂数学表达式，提供更好的性能和具有成本效益的解决方案。然而，人工智能方法也并非完美，因为它们严重依赖所收集数据的质量和数量。可以猜测，如果一个机器学习模型在给定的数据集上表现得非常好，它仍然有可能在其他数据集上表现不佳。

此外，针对特定的地质科学问题，有许多种类的机器学习方法可供选择。因此，在紧急情况下如何选择合适的方法是一个问题。

要正确优化超参数，需要一定的专业技能。这可能在一定程度上限制了机器学习方法的适用性。此外，机器学习方法的黑盒特性使得无法解释输出的物理过程，这对决策者来说可能是不可接受的，因为机器学习方法不是透明且可解释的。

针对地质灾害，尤其是滑坡，已经进行了大量的SAR、InSAR和LiDAR监测工作，目前已经引起了越来越多的关注，以进行灾害预防和减灾、抗灾和救灾工作，但是还需要进一步研究如何以一致、可靠和智能的方式从这些大数据中检测潜在的滑坡。机器学习和深度学习已经广泛应用于计算机科学和遥感领域，在这些领域中，统计技术被用来从给定的数据中学习特定和复杂的任务。最近的研究报告称，机器学习可以从大数据集中识别与地质灾害相关的信号。然而，预测滑坡何时发生仍然是一个巨大的挑战，考虑到只有少数成功案例的报道。图9.2总结了深度学习在地质工程和地球科学中的主要受限应用。

## 10 正在进行的工作和未来建议

据作者所知，以下工作正在进行中，同时可能提出一些未来的建议：

+   (1) 机器学习模型建立在数据上，根据设计它们不包含任何物理定律（如质量和能量平衡），并且在训练数据范围之外的范围内不会很好地外推。从历史上看，物理建模和机器学习通常被视为两个不同的领域或途径，具有非常不同的科学范式（理论驱动与数据驱动，如图10.1所示）。然而，事实上，这两种方法是互补的，物理方法原则上是直接可解释的，并且具有超出观察条件的外推潜力，而数据驱动方法在适应数据方面非常灵活，并且易于发现意外的模式。地质工程师和地球科学家面临的首要挑战是如何从机器学习和模拟中提取尽可能多的有用信息，并从两者之间的相互作用中获得新的见解。不同的技术，如正则化和将物理约束嵌入模型架构中，已成功应用于解决这个问题。然而，更多的研究和投资在构建领域感知模型和使用强化学习等技术方面是一个重要的优先事项（图10.1）。

+   (2) 基于机器/深度学习的原则“垃圾进，垃圾出”，训练观测的选择直接影响预测的成功或失败。在使用机器学习和深度学习进行地质工程或地球科学预测时存在一些不足之处。首先，一个常见的问题是缺乏训练样本。训练数据是监督式机器学习技术的主要组成部分，并且在地质工程和地球科学的应用中越来越成为主要瓶颈。少量的训练样本无法为机器学习或深度学习模型提供足够的特征，这导致预测准确性较低。另一个问题是观测不平衡，即标签数量之间存在显著差异。

此外，数据科学挑战是必不可少的，然而，实现数据科学和机器学习的好处的一个主要障碍在于我们的本科和研究生课程。地质工程标准课程中很少准备他们迎接更多数据驱动科学的未来。在这方面需要进行更多的工作。

(6) 气候变化和不断增加的人类活动使得可能的地质灾害的预测和地质工程响应的估计变得更加具有挑战性。因此，考虑到气候变化和人类活动，具有模块化的机器/深度学习方法使得预测更具可扩展性、可维护性和可扩展性。

此外，我们正在步入5G时代，6G移动通信即将到来，适用于地质工程（如隧道和管道）建设和维护过程中的5G或6G大数据的机器/深度学习方法是必不可少的，同时提出了它们的经济潜力。另一方面，大数据有其自己的生命周期，包括从数据采集到分发和归档等阶段。因此，在数据生命周期的每个阶段利用机器学习的机会，并可以总结为五个类别： (1) 改善数据利用率， (2) 自动化操作， (3) 提高服务水平， (4) 提供创新服务，以及 (5) 利用现有信息管理技能来策划标记数据。为了利用这些机会，需要构建精心策划的大规模训练数据集，正如趋势项目一中提出的。

(7) 基于“具体问题具体分析”的原则，对于各种复杂性的地质工程，应采用不同的机器/深度学习和优化方法。对于具有多变量大数据且存在噪声的地质工程，需要采用具有错误校正和较强鲁棒性的算法。而对于具有有限特征和不完整输入的问题，选择应该完全不同。根据 (Phoon 2019) 的研究，MUSIC-X特征

（MUSIC-X：多变量、不确定和独特、稀疏、不完整、可能被破坏的地质技术数据，其中X表示时间和/或空间维度）决定了方法的采用不应该盲目。换句话说，盲目应用机器学习模型永远无法达到最优结果。

（8）地震、火山喷发、海啸或滑坡威胁着全球数十亿人口。只有当我们对潜在的物理过程了解更多时，才能将影响最小化。一方面，需要开发下一代地球观测系统，另一方面，创新的系统用于对获得的大数据进行建模和可靠解释是必不可少的。从这个角度来看，将新的观测量和监测技术与ML/DL方法相结合，当然是发展的趋势。此外，未来发展的方法应该能够满足从秒到分钟（如滑坡、地震的警报），从天到周（如火山喷发的现在预报）到年（如地面沉降或隆起的预测）的不同时间尺度的考虑。

（9）目前，对地质工程性能/响应的评估通常基于单一回归或分类数值，无论是数值还是图像。通过结合使用数值、图像甚至音频信息，可以获得更多的好处，包括提高准确性或鲁棒性。例如，在现场地质勘探过程中，地质工程师通常通过岩石表面纹理、颜色和敲击音信息对岩性进行初步判断。基于专家经验和智能模式，可以利用岩石图像和敲击音提出一种用于岩性分类的深度学习和智能识别分析方法。

（10）地质工程和地球科学应用领域的另一个趋势是视觉计算，它处理图像和3D模型，即计算机图形学、图像处理、可视化、计算机视觉、虚拟和增强现实、视频处理，还包括模式识别、人机交互、机器学习和数字图书馆的方面。核心挑战是获取、处理、分析和渲染视觉信息（主要是图像和视频）。Mohammady等人（2019年）、Wang等人（2020年）开发了CNN EXPLAINER，帮助用户更容易理解CNN的内部工作原理，它可以在用户的Web浏览器中本地运行，无需安装或专门的硬件，扩大了公众对现代深度学习技术的教育获取。

最近，Stefan Sietzen成功通过Unity 3D实现了CNN可视化，并且有很多代码（https://vimeo.com/stefsietz）。

虽然许多机器学习方法如ANN在早期被批评为黑盒引擎，决策过程不可解释，但最近的神经符号AI为这个障碍提供了解决方案。

此外，通过众包、综合工作空间和厚数据分析对收集到的数据进行专家标记和注释的广泛参与，扩大了解决各种地质工程和地球科学问题的良好训练数据集的可用性。总之，只有当模型变得更加透明和可解释时，机器学习在这两个领域的广泛应用才能进一步促进：它们的参数和特征排序应具有最小的物理解释，并且模型应该可以通过一组规则、描述符和关系来简化或解释。

最后但并非最不重要的是，在建立原始数据采集的系统标准时，对数据收集、测试和分析进行规范是必不可少的。特别是在工程实践中，原始数据需要符合统一的管理标准，这需要由官方管理部门和机构（数据隐私）强制执行。否则，尽管在算法开发方面付出了巨大努力，由于缺乏足够的数据样本进行机器学习，研究进展将停滞不前。因此，ML、DL和优化的工程应用不仅是一个科学问题，也是一个管理和系统问题，需要多方合作。
