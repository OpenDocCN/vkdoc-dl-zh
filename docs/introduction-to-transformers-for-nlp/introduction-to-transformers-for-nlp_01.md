# 引言

本书将带领读者踏上自然语言处理的旅程，从 n-gram 模型开始，历经 RNN 等神经网络架构，最终抵达当今最前沿的技术——Transformer。本书详细阐述了 Transformer 架构，并重点解释了自注意力机制，这是 Transformer 概念的基础。

本书深入探讨了 Transformer 这一主题，并辅以来自不同 NLP 领域（如文本生成、情感分析、零样本学习、文本摘要等）的示例。本书深入研究了 Hugging Face API 及其在创建基于 Gradio 的简单应用程序中的用法。我们不仅会详细讲解如何使用预训练模型，还会探讨如何利用我们自己的数据集对现有模型进行微调。

我们将涵盖 BERT、GPT2、T5 等模型，并展示如何直接使用这些模型在自然语言处理和理解领域创建一系列不同的应用程序。

本书并未将 Transformer 的知识和探索局限于 NLP 领域，还从较高层面介绍了 Transformer 如何在视觉等领域得到应用。
