# 页面浏览量的危害

数据驱动的在线新闻经济学

> *互联网经济学催生了一套扭曲的激励机制，使得流量比真相更重要——也更有利可图。*
> 
> ——瑞安·霍利迪，《相信我，我在撒谎：一个媒体操纵者的自白》

我们了解（或自以为了解）的世界上正在发生的事情，大部分是通过阅读新闻获得的。但如今，“新闻”的含义与过去几代人相比已大不相同。我们今天主要阅读的是互联网上的文章——从随意的博客帖子，到国内外新闻网站上经过精心调查的报道，无所不包。新闻业从印刷品向屏幕的转型，本质上并不意味着我们读到的内容比过去更不真实。然而，这种技术变革催生了一种不那么明显却极具影响力的经济转型：新闻业的*数据化*。正如我将要讨论的，我们在互联网上留下的页面浏览量和点击量，正是滋养着日益增长的错误信息和假新闻花园的数字肥料。通过追溯当代新闻周期中涉及的财务激励，我希望在本章中揭示一个令人震惊的程度：那些我们大多数人看不见、却由我们的行为和活动创造的数据，正在从根本上塑造我们每天阅读的内容，并威胁着传统新闻标准的堡垒。

## 故事的传播

让我先做一个分类。在互联网媒体食物链的底层，如果你愿意这么理解的话，是那些专注于特定议题、兴趣或领域的小型博客和网站；它们可以是单人作者或多作者。往上一层是报纸、杂志和电视台的博客。这是一个令人困惑的中间地带，因为许多这类博客共享一个知名新闻来源的名称、网址和标志，但其编辑标准通常低于母机构，而且许多撰稿人缺乏人们可能期望从母机构获得的新闻培训。最顶层是官方新闻网站，它们可能是区域性的，但往往吸引大量的全国性或国际性读者。这种层级划分并非关乎质量——事实上，一些非常专注的小型博客能产出极高品质的内容，而一些大牌全国性新闻网站却持续发布准确性严重存疑的文章。这里的层级更多关乎受众和机构的规模，以及内容的范围。

信息在这个互联网新闻层级中既垂直流动，也水平流动。当《华盛顿邮报》爆出一则重大新闻时，只需几个小时，《纽约时报》也会进行报道，反之亦然，而且往往只是通过报道另一家报纸文章中的内容。这就是*水平传播*，之所以会发生，是因为即使第二家报纸不能声称是爆出该新闻的功劳，它也不希望自己的读者直接从竞争对手的报纸那里获取这些信息。*垂直传播*则有两个方向。顶层爆出的重大新闻会被较小的新闻机构和博客报道和复制，因为与水平传播类似，这是一种无需做太多工作就能留住读者的简便方式；这是信息的*向下*流动。

虽然水平传播和向下垂直传播都存在明显的冗余，从而导致整体系统效率低下，但对于追求真相的读者来说，唯一真正的危害是，当故事从一个机构传递到另一个机构时，重要的细节可能会被遗漏，事实可能会被扭曲——尽管有时，一个更专业的博客会通过深入挖掘故事的某个特定方面，提供有价值的服务，而这对于顶层机构来说可能并不合适。找到一家报纸爆出的新闻，然后比较它在其他一系列报纸和博客上的报道，会非常有启发性；这是揭示不同机构意识形态倾向的绝佳方式，因为同样的事实会因涉及的不同观点而被着色。

剩下的新闻传播形式是*向上*的垂直流动，即故事始于小型博客，有时最终会一路到达全国性新闻网站。这是本章的关键主题之一，因为它导致了我们所见到的数量惊人的错误报道和彻头彻尾的假新闻，而且这几乎完全是由数据和现代媒体的经济学驱动的。在探讨这个具体主题之前，退一步审视驱动博客和报纸的财务力量会有所帮助；在此过程中，我对“博客”采取宽泛的定义，将其基本涵盖所有在线发布书面内容的形式。
