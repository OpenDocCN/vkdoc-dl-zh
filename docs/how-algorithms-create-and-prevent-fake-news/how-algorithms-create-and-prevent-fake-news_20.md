# 结语

2018 年，Facebook 宣布计划成立一个新的监督委员会，以帮助审议和裁决有关该平台对公共话语影响的事务。该委员会常被称为“Facebook 的最高法院”，由一系列国际专家组成——从媒体研究、法律和公共政策领域的顶尖学者，到人权组织和智库的领导人，甚至还有一位诺贝尔和平奖得主和一位丹麦前首相。作为该委员会活动的一个例子，其任务是判定无限期暂停特朗普的 Facebook 账户是否合理，以及是否应继续维持。

然而，哥伦比亚大学言论自由研究所的两位学者最近指出^(³³⁴)，该委员会的权限范围非常有限——它几乎只关注个别内容删除的案例——以至于在某种意义上只是一个幌子。他们指出，内容审核的具体问题固然重要，但公司对其平台设计以及驱动平台的算法所做的决策影响更为深远：“[Facebook 的]排名算法决定了哪些内容出现在用户信息流的顶部。它关于哪些类型的内容可以分享以及如何分享的决定，帮助决定了哪些想法能获得关注。[……]该委员会实际上被引导着将 Facebook 的平台架构视为既定事实。”

换句话说，该委员会给公众留下了外部监管和致力于减轻社会负面影响的印象，但有害内容在社交媒体上传播的问题以及如何审核的问题，远比针对个别内容的决策要深刻得多。真正的讨论必须涉及对当前旨在最大化用户参与度的算法方法进行调查，并可能进行彻底的反思。你在本章中已经看到，关于算法放大有害内容的研究已经有很多富有洞察力的成果，甚至还有一些关于重新设计社交网络结构以应对这一问题的有前景的想法。这些算法设计问题对于未来的发展方向至关重要，但 Facebook 却方便地将所有相关决策都掌握在自己手中，并排除在其最高法院的管辖范围之外。

而 Facebook 及其他科技巨头通过算法手段改善问题的努力，也并非总是充满足够的热情。凯西·奥尼尔，作为最早一批呼吁关注社会过度依赖算法危险性的杰出数据科学家，在 2021 年 2 月写道^(³³⁵)：“我自己在内容审核方面的经历让我对公司的动机深表怀疑。”她说，她曾受邀参与谷歌一个关于 YouTube 有毒评论的人工智能项目，但在看到该项目分配的微薄预算后拒绝了，并得出结论：“这要么是不认真的，要么就是注定要失败。”她在 Twitter 的一个反骚扰项目上也有类似经历。即使这些公司在审核的一些技术问题上取得了进展，新的挑战也会迅速且频繁地出现。例如，用户现在利用 Facebook、YouTube 和其他平台的直播功能来传播他们的信息，有时受众极其庞大，而直播视频流在审核方面显然带来了许多棘手的技难题。



科技记者凯伦·郝（Karen Hao）多年来一直撰写探讨机器学习算法中的偏见及其意外后果的文章。2021 年 3 月，她发表了一篇文章^(³³⁶)，并将其描述为^(³³⁷)“我做过的最艰难、最复杂的故事”。这篇文章的标题是什么？《Facebook 如何沉迷于传播虚假信息》。她说：“通过这次彻底的报道，我确信自我监管行不通，也根本不可能奏效。”她还表示，这篇文章“并非关于腐败之人做腐败之事……而是关于善良之人真心想做正确的事。但他们被困在一个腐朽的系统中，竭尽全力推动着僵化不变的现状。”

郝这篇雄心勃勃的文章要点如下：Facebook 通过设计算法不惜一切代价最大化用户参与度，从而构建了其庞大的平台。无数内部研究得出结论：旨在最大化参与度的算法也会加剧两极分化，并放大可疑内容，但扎克伯格及公司其他高层领导却执着于将参与度作为增长手段。公司内部为在 AI 应用中引入更多伦理考量所做的努力，大多集中在算法偏见上。减轻算法偏见（尤其是为了防止联邦法律禁止的歧视行为）是一个重要且具有挑战性的课题，但据称 Facebook 并未优先处理其他关键任务，例如根除虚假信息；一些内部人士甚至怀疑，即使他们尝试了，也未必能成功。此外，一些反偏见努力导致虚假信息审核算法被搁置，原因是这些算法标记的保守派内容多于自由派内容——尽管人们早已认识到虚假信息对极右翼群体的影响尤为严重。这里涉及的算法、经济和心理主题及动态，你在本书中已多次以不同形式见到，但我鼓励你亲自阅读郝的文章，看看这一切是如何在一家公司内交织在一起的，并了解这个传播虚假信息的庞然大物是如何被构建出来，随后又失控的。

另一个额外的挑战——在 2020 年大选期间尤为明显——是内容如今经常跨不同社交媒体平台传播，这使得内容审核比过去更像一个涉及多家公司的复杂问题。奇怪的是，或许也颇具深意的是，近年来社交媒体审核最有效的案例之一，是科技巨头们联手，基本上切断了新兴平台 Parler 的电源。在 Facebook 和 Twitter 开始打击支持特朗普的选举虚假信息后，许多不满的用户离开并加入了 Parler，该平台自称是一个以言论自由为导向的替代选择，对政治保守派的言论更为友好。但是，尽管 Parler 本身没有针对在其平台上迅速传播的有害内容制定政策，但为其应用提供技术支持和分发渠道的科技巨头（亚马逊、苹果和谷歌）却有这样的政策，并迅速而有力地采取了行动，导致 Parler 难以生存。冒着对这个情况过度冷嘲热讽的风险，我必须说，这些公司在相互监管方面似乎比自我监管更有能力，也更愿意这样做。

## 总结

本章开篇引用了一项调查，该调查显示，与主要通过其他大多数媒体形式获取新闻的人相比，主要依赖社交媒体获取新闻的人往往更年轻，更容易接触到假新闻，并且对政治和全球健康的了解更少。接着，我讨论了推荐和排序算法在假新闻传播中所扮演的角色，并特别以 Facebook 为例，以说明该平台上的问题，并展示围绕该问题的一些言论和行动。我还讨论了像“匿名者 Q”（QAnon）这样范围广泛且不断演变的阴谋论运动如何对内容审核构成特殊挑战。随后，我总结并阐释了一系列学术论文，这些论文使用复杂方法定量研究了假新闻如何在社交媒体上传播（主要是 Twitter，因为其数据比 Facebook 更容易获取），以及机器人如何参与其中。接下来，我审视了社交媒体公司（主要是 Facebook，因为 Twitter 一直不太公开）在对抗机器人和虚假信息时使用的一些机器学习方法，然后我考察了学术研究人员为此目的提出的一些方法。最后，我传达了近期关于第 230 条（Section 230）的一些辩论和声明——该法律保护社交媒体公司免于为其托管的内容承担责任——并以对 Facebook 等公司不愿在打击假新闻的斗争中加强行动的思考作为本章结尾。在下一章，也就是最后一章中，我将介绍一些可用的算法工具，以帮助你在个人层面打击假新闻。
