# 从底层崛起

斯坦福互联网观察站的研究员蕾妮·迪雷斯塔最近在《大西洋月刊》上写道：“媒体和社交媒体已不再泾渭分明；有影响力的叙事既自上而下，也自下而上地涌现，并在不同渠道间来回反弹。”回想一下，我尚未直接探讨的传播方向——尽管我声称它对我们当前媒体谎言的泥潭负有最大责任——正是这种自下而上的流动：故事从小型、通常是特定兴趣和/或地理上局部的博客开始，设法沿着食物链向上攀升，有时最终登上国家级新闻网站的顶端。我们在此必须提出的问题是：这种情况如何发生、为何发生，以及为何会导致新闻真实性下降？正如我接下来要讨论的，答案本质上都源于博客的流量经济学。

所有博客和新闻来源，即使是顶端备受推崇的那些，都在不断寻找新故事。其中存在一个根本性的不平衡：实际故事（即世界上应当被报道的真实事件）的供应量，远小于博客和在线报纸所生产的故事供应量——因为，正如我前面所讨论的，积累流量的压力迫使作者们以不健康的速度填满互联网无限的带宽。这造成了一个危险的真空：各级博主都承受着巨大压力，必须想方设法不断寻找故事，常常还要无中生有，以维持现代媒体机器的运转。

层级最低的博客通常资金不足、人手短缺，并且往往依赖其所属社区的小型、紧密联系的性质——这意味着他们经常根据社区成员的建议发布内容，并追踪社交媒体上的线索，而不真正质疑其真实性。从很多方面来看，这相当合理：一家受人尊敬的国家级新闻台在听到关于拜登政府的某些丑闻八卦时，必须在向公众报道前确保其绝对准确；而一个关于大白熊犬及其疯狂行为的博客，则不太担心其帖子可能被解读为假新闻。一般来说，规模更小、更专业的博客调查线索的资源更少，无论怎样，他们这样做的动力也更小。

然而，当我们审视层级的中层时，问题就开始出现了。在这里，博主们仍然极度渴望故事，他们根本没有时间用传统的新闻方式去寻找，因此显而易见的捷径就是搜罗低层级的博客。那些展现出能吸引更广泛受众、产生流量潜力的激动人心的帖子，会迅速被中层博主捡起并重新包装。但这些博主缺乏时间和资源去追溯故事的源头并仔细核实事实，因此一个稳妥的规避风险的做法就是简单报道：某某博客（低层级的那个）报道了某某事件。这样你就不可能出错：无论原始故事是否真实，该故事确实在那个博客上被报道过，这是毋庸置疑的。

接下来，随着横向传播的增多，故事本身与元故事之间的界限变得模糊，博主们互相引用，竞相分享这个独家新闻带来的流量。久而久之，这个故事本身的流行度就可能成为故事本身——因为病毒式传播本身就有新闻价值，不是吗？——到了这个地步，国家级报纸就可以安全地将事情提升到最高层级，用“这个故事在互联网上引起轰动”这样的标题来报道。我们在史蒂夫·乔布斯生命的最后几年里经常看到这种情况：来源不明的谣言在互联网的阴影中盘旋，以不可预测的方式获得关注，达到临界质量后最终影响了苹果公司的股价，并因此，可以这么说，变成了*真正的*新闻。

博客帖子在层级中向上爬升也以更直接的方式发生。一项全国性调查发现，近九成记者使用博客来研究他们的报道，因此即使是顶层的记者也会向下寻找信息。此外，博主获得大量流量的最佳方式，就是他们的故事被更高层级的组织——尤其是国家级新闻网站——选中并链接。因此，中层博主经常将他们的帖子提交给大众媒体记者监控的新闻聚合网站，他们甚至直接联系记者，希望能引起他们的兴趣——因为，毕竟，即使是这些记者也在不断寻找能产生流量的热门故事。

瑞安·霍利迪根据自己为了商业利益而故意鼓励和利用这种博客圈向上流动形式的经历，写了一本关于此现象的精彩著作——《相信我，我在撒谎：一个媒体操纵者的自白》。在书中，他描述了自己如何能够“通过将一个故事投给一个标准极低的小博客，使其成为大博客的故事来源，进而成为更大媒体机构的故事来源，从而将无中生有。”他说，他经常看到“付费编辑插入维基百科的措辞独特或经过选择性编辑的事实，后来以完全相同的措辞出现在主要报纸和博客中”，这是新闻捷径的明显标志，也显示了这些捷径如何被利用。他深刻且令人恐惧地总结了他多年来玩的这个游戏的社会后果：“新闻，无论是在网上还是印刷品中找到的，都只是成功通过了媒体过滤器的内容。[……]由于新闻塑造了我们对周围发生的事情的理解，这些过滤器创造了一个被构建的现实。”请记住，霍利迪所指的这个被构建的现实，源于数据驱动的流量经济学。21 世纪的数据本应为我们提供一个强大、未经修饰的通往世界真相的新窗口，但在这场关于互联网新闻的讨论中，我们令人担忧地看到，它同时也支撑着对我们基本世界认知的一种危险扭曲。
