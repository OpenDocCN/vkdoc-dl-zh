# 摘要

19 世纪末的美国报纸按日零售，为了争夺销量，它们竞相使用最耸人听闻的标题，即便实际内容被夸大或捏造。订阅模式在 20 世纪兴起并占据主导地位；它通过为记者提供经济激励，促使他们撰写准确、经过深入调查的报道，从而遏制了假新闻，因为误导性内容会导致客户取消订阅，转而投向竞争对手的报纸。

随着 21 世纪新闻业转向在线形式，广告收入成为主流财务结构，页面浏览量作为基本货币的地位日益凸显。与此同时，新闻行业变得多元化，涌现出大量规模各异的出版商，社交媒体和新闻聚合器成为人们获取新闻的常见途径。读者对特定报纸的忠诚度下降，争夺用户注意力的战斗卷土重来，19 世纪的许多问题也随之重现——甚至更糟：量化方法使作者能够炮制点击诱饵标题和文章，以实现最大程度的病毒式传播，即使这样做需要捏造假新闻。

对广告收入的激烈竞争也促使记者走捷径，他们将时间花在搜寻博客和论文上寻找素材，而非进行直接调查。这导致了一种垂直传播模式：假新闻可能从编辑标准极低的博客或低级报纸的底层混入系统，然后一路向上渗透至顶端。

订阅模式已以在线付费墙的形式回归部分报纸，但仍有大量由广告收入支持的免费报纸存在。此外，新闻业技术和经济格局变化的一个长期后果是地方报纸的急剧萎缩，且这一趋势毫无减缓迹象。机会主义的政治宣传者和职业假新闻贩子正迅速用欺骗性报纸填补这一空白，这些报纸利用了人们对本地新闻的传统信任。

虽然本章并未深入探讨假新闻的算法层面——这是本书的主题——但它通过展示新闻业当前的结构和资金模式奠定了基础，并揭示了系统中的一些脆弱性，这些脆弱性将在后续章节中发挥重要作用。本章还展示了数据——以页面浏览量的形式——如何扮演核心角色。你在本书后面遇到的所有算法都由各种形式的数据驱动。本章确实包含了一个简短的自动化新闻生产的原始示例——一个虚假地方新闻网站网络，它们粘贴其他来源的新闻稿，并根据当地天气预报拼凑出简单的通用内容。在下一章中，我将展示算法生成的新闻内容如何达到了前所未有的复杂程度，并探讨它目前在假新闻泛滥中所扮演的角色。
