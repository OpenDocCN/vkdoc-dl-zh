# 总结

TensorFlow 2.x 为开发深度学习机器学习应用提供了一个非常强大的平台。该平台从数据准备和模型构建，到最终在生产服务器上部署，都能为你提供便利。它就像使用一个工具进行端到端开发一样。流行的机器学习开发库 Keras 现已完全集成到 TF 中。利用 TF 的新特性，例如即时执行、跨多个 CPU、GPU、TPU 的训练和推理分布，以及高效的数据管道，你将能够在 Keras 中开发出非常高效的机器学习应用。在开发过程中，TensorBoard 提供了有用的分析，帮助你优化模型。一个完全训练好的模型会保存为一种格式，甚至可以部署在移动设备和嵌入式设备上。TF 还提供了视觉和语音套件，用于在嵌入式设备上部署图像/视频识别和语音控制的机器学习模型。社区贡献了多个预训练模型可供你使用。使用 Edge TPU 可以在设备本身上进行推理。总之，TF 2.x 可以被视为一个从开发到部署的单一机器学习平台。

在本章末尾，我介绍了在几个平台上安装 TF 的方法。我还讨论了使用 Google Colab（它提供了一个基于云的 Python 项目开发环境）来开发 TF 应用。如果你拥有良好的互联网连接（这在当今世界大部分地区已不罕见），你可以依赖 Google Colab 来完成所有机器学习应用，而本书正是这样做的。
