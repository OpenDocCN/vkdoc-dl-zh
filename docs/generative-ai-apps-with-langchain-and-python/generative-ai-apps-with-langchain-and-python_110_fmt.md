# 第 7 章 使用检索增强生成（RAG）构建高级问答与搜索应用

当你拥有一批需要快速高效搜索的数据时，向量存储便派上了用场。它们以称为向量的特殊格式存储数据，这使得查找相似项变得轻而易举。

现在，让我们分解使用向量存储的过程，并通过一个示例使其变得具体：

1. **加载源数据**：第一步是收集所有数据并将其加载到向量存储中，以便后续进行搜索。你可以从各种来源加载数据，例如文本文件、数据库，甚至网页。

2. **查询向量存储**：数据加载完成后，你可以使用查询来让向量存储查找与查询最相似的项。例如，你可以让向量存储找出与你的查询（如搜索词或问题）最相似的项。随后，向量存储会在后台将你的查询转换为向量表示。接着，它会将此查询向量与存储中的所有向量进行比较，并检索出最相似的向量。
