# 第 2 章 使用 LangChain 集成 LLM API

## 其他注意事项

除了我们之前讨论的内容之外，这里还有一些其他考虑因素，它们会显著影响你的生成式 AI 项目的成功。

### 选择合适的用例

如图 1-1 所示，你需要选择合适的用例。这个初始步骤至关重要，因为它为整个项目设定了方向。思考你想要解决的具体问题，或者你想要创建的用户体验。LLM 可以应用于广泛的场景，例如聊天机器人、内容生成、情感分析等。

**问题：** 思考一个没有使用生成式 AI，但本可以发挥作用的项目。引入 LangChain 和 LLM 会如何改变结果？例如，在客户服务中，使用聊天机器人处理常见咨询可以释放人力资源来处理更复杂的问题，从而提高整体效率。

### 选择合适的模型

你必须为你的用例选择合适的模型，并对其进行微调以提升性能。模型应与你的特定需求保持一致，无论是回答问题、生成文本还是执行其他任务。

就 LLM 本身而言，有多种选择，例如 `GPT-4`、`BERT`、`Gemini` 等。每个模型都有其自身的优势和劣势，因此你需要了解它们的能力和局限性。
