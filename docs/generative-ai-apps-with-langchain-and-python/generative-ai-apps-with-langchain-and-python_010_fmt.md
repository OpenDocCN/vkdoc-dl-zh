# 关于作者

**拉比·杰伊**拥有超过 15 年推动数字化转型的经验，并具备独特的“敏锐洞察力”。他作为 `Java` 和 `SAP ABAP` 开发人员的背景，使其对大型语言模型通常需要集成的企业系统有着深刻见解。作为德勤数字/云原生业务的领导者，他积累了跨行业应用人工智能解决方案的经验，这使他能够准确识别大型语言模型在哪些领域最具商业影响力。

他热衷于让复杂技术变得易于理解，这促使他撰写了《SAP NetWeaver Portal 技术》和《企业级云人工智能》两本书，并定期为行业出版物撰稿。他曾担任《基于大型语言模型的解决方案》一书的**技术审校**，并在 `HCL America` 担任负责数字化转型的副总裁，这充分体现了他在大型语言模型领域的积极参与。此外，他还运营着一份领英新闻通讯（“企业人工智能转型”）和一门免费的领英课程（“面向商业创新的人工智能”）。
