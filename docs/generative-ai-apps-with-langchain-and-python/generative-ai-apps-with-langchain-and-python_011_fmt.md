# 关于技术审校

**穆拉利·克里希南**是一位亲力亲为的软件领导者和初创公司创始人，热衷于构建高效且能创造营收的业务。他在微软（B2B、B2C 领域）拥有超过 20 年的经验，随后在星巴克（B2C 领域）担任过高管。作为联合创始人，穆拉利在 RapL（一个移动优先、人工智能驱动的自适应微学习平台）领导了融资、产品、客户成功、市场营销和销售工作。他拥有 30 多项美国专利，在尖端技术（云、人工智能/机器学习、移动、大数据）方面具有经过验证的创新能力。穆拉利还指导、协助初创公司创始人并提供建议。他与家人住在西雅图，喜欢跑步、骑行、阅读和烹饪。

**希瓦库马尔·戈帕拉克里希南**在软件开发、DevOps、SRE 和平台工程领域拥有超过 25 年的经验。他曾在多个行业工作，从医疗保健企业到面向消费者的网络规模公司。他创立过一家初创公司，曾是一家财富 1000 强公司的关键架构师，目前是 BD 公司的首席架构师。他是《基于 Azure 的 Kubernetes 实战》的合著者，以及《求职者 Kubernetes 指南》和《使用 ChatGPT 的现代 Python 编程》的作者。

**基尔蒂·巴拉特**是人工智能领域一位成就卓著的思想领袖。他拥有运营多家初创公司以及在跨国公司领导人工智能项目的经验。他还是一位投资者和公司导师。他毕业于纽约雪城大学和金奈金迪工程学院。
