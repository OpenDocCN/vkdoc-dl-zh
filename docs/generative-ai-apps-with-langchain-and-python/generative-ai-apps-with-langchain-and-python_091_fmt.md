# 第 6 章 使用链构建智能聊天机器人与自动化分析系统

随着本章内容的深入，你将了解 LangChain 中可用的不同类型的链、它们的用例，并学习如何使用它们构建强大的生成式 AI 应用。

## 理解链的组成部分

现在你已经对链是什么以及它们为何重要有了基本概念，让我们来探索这些链是由什么构成的。

在讨论链的组成部分时，有两种看待方式。首先，我们从较高层次来看如何分解这些组件，然后我们将更深入地研究链的内部工作原理，即它如何将各种 LangChain 模块（如 LLM、提示和工具）整合在一起，以执行工作流中的某个步骤。

### 链应用的高级组件

我们已经讨论过，LangChain 中的链可以被视为一系列相互连接的步骤，每个步骤旨在执行特定功能，最终实现预期结果。但接下来，我们需要讨论如何构建这些组件：

1. **触发器**：首先是触发器。这是启动链的要素。它可以是用户请求、定时任务或应用中的事件。

2. **步骤（或节点）**：这些是链执行的各个任务。每个步骤可能涉及处理数据、做出决策或与外部系统交互。你将使用这些步骤按预定义顺序执行任务，每个步骤的输出可以作为下一步的输入。在 LangChain 中，你将利用 LLM、提示以及可能的工具来实现这些步骤。你将使用 LLM 处理输入，使用提示引导 LLM 的响应，并使用工具与外部资源交互。
