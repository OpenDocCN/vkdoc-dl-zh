# 第 5 章 掌握创意内容的提示词

## 提示模板

在使用语言模型时，你可以利用两种类型的提示模板：

1. **常规提示模板**：用于直接的文本生成用例，例如回答问题、完成句子或任何其他文本生成任务。

2. **聊天提示模板**：专门为更具交互性和对话性的体验而设计，包含系统消息和用户消息等组件来指导交互。

你应该注意，大多数最新模型（包括默认启用聊天补全的模型）都支持这两种模板类型。这意味着你可以使用同一个模型来完成一般任务（如回答问题）以及创建对话交互。因此，你可以根据具体需求选择合适的模板，而无需切换模型。

让我们讨论一下提示模板中包含的内容：

1. **指令**：你将使用指令向 LLM 提供清晰的指示，并向 LLM 传达你的期望。

2. **少样本示例**：包含几个期望输出的示例，为 LLM 提供具体的参考点。通过这些示例，你可以传达你期望的风格、语气和格式，使模型能够掌握你需求的细微差别。
