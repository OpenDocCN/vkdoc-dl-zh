# “谁雕刻了大卫雕像？”问题的模型

“谁雕刻了大卫雕像？”这个问题的模型基于带有示例选择器的提示模板。该模型利用选定的示例及其已有知识来生成正确答案。

这两个提示模板的主要区别在于，不带示例选择器的模板包含所有示例，而带示例选择器（`LengthBasedExampleSelector`）的模板则根据示例长度进行筛选。

在这两种情况下，语言模型都会使用提供的示例及其已有知识，为输入的问题生成合适的答案。因此，示例充当了模型的指南或上下文，帮助其理解所问信息的类型以及预期的答案格式。
