# 第 5 章 掌握创意内容的提示词

## 优化方法：提示工程与模型微调

区分提示词微调（即提示工程）和模型微调非常重要。你应该首先从优化提示词开始，这包括完善指令、调整示例以及尝试不同的措辞。推荐这种方法，因为它是一种快速的方法，不需要更改模型。例如，不要使用这样的提示词——“总结这篇文章”——你可以使用更精细的提示词，例如“提供这篇文章关键点的 3 个要点总结，适合高中生阅读。”

### 少样本学习

如果简单的提示词不够，你可以尝试少样本学习，在提示词中包含两到三个示例。这更有帮助，因为它更明确地指导模型。

### 模型微调

然而，如果提示工程和少样本学习都没有帮助，你应该考虑模型微调。这涉及在特定数据集上重新训练模型，以使其能力适应你的任务。当你拥有大型高质量数据集时，这非常有用。如果你希望在提示工程无法帮助的特定类型任务上获得一致性能，那么模型微调可以是你的解决方案。
