# 第 2 章：使用 LangChain 集成大语言模型 API

在本章中，我们将以第 1 章的基础知识为基石，更深入、更实际地理解如何运用 LangChain 和大语言模型 API 来构建稳健且可扩展的生成式 AI 应用程序。第 1 章对 LangChain 和大语言模型进行了广泛概述，而本章将深入探讨将大语言模型 API 与 LangChain 集成的实际方面，包括动手练习和详细的技术设置说明。

你将发现这些大语言模型 API 带来的诸多好处，例如更高的效率、可扩展性和成本效益。这些 API 使不同技能水平的开发者都能使用先进的 AI 功能，而无需深厚的机器学习专业知识。你将学习如何利用这些 API 来扩大应用程序的全球覆盖范围，并无缝集成最新的大语言模型进展，以构建实用的生成式 AI 解决方案。

此外，你还将学习应对集成大语言模型 API 时可能遇到的一些常见挑战，例如解决安全问题、管理已弃用的功能以及处理速率限制。针对这些挑战，我将提供实用的策略，帮助你有效克服它们。
