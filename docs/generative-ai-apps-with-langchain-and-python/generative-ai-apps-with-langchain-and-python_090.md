## 为什么链在生成式 AI 中如此重要？

现在你已经对链有了基本了解，我们来讨论一下为什么它们在开发生成式 AI 应用时至关重要。

使用链，你可以在构建 AI 应用时采用结构化和模块化的方法。你可以将复杂任务分解为更小、更易于管理的步骤。这种模块化使你的代码更有条理、更易读且易于维护。你可以轻松替换组件、修改现有链，或以最小的努力和对整体应用功能的最小影响创建新链。

其次，你可以充分利用语言模型的全部能力，同时无需担心 LLM 的内部工作机制以及与它们交互所涉及的复杂性。通过利用链的高级接口，你可以专注于手头的任务。

第三，你可以轻松地将各种工具和服务集成到你的 AI 应用中。你可以轻松地从 API 检索数据、查询数据库，或使用外部库处理文本，并将它们作为应用工作流的一部分。

当你的应用规模增长时，其复杂性也可能随之增加。通过使用链，你可以通过简单地修改现有链或添加新链来轻松扩展操作，而无需彻底改造整个应用。你还可以轻松维护应用并独立排查各个组件的问题。

你可以在应用的多个部分甚至不同项目中重复使用同一个链，这不仅节省了时间，还确保了这些任务在你的应用中执行方式的一致性。
