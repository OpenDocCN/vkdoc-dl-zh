## 文本嵌入模型

继续我们关于向量的讨论，让我们更深入地探讨一下文本嵌入。使用向量存储的一个关键部分是借助嵌入来创建向量，以便后续进行搜索。无论你是在构建语义搜索引擎、推荐系统，还是任何需要理解词语背后含义的应用，文本嵌入都是你的答案。文本嵌入将帮助你从大量文档中找出与特定查询最相似的文档。你可以尝试直接比较词语，但这就像大海捞针。相反，文本嵌入可以拯救你！

### 关键词搜索 vs. 文本嵌入

值得注意的是，有时对于简单的查询，纯关键词搜索可能更有效。近期的建议是将关键词搜索与嵌入相结合，以发挥两种方法的优势。这种混合方法可以提供关键词搜索的精确性，以及嵌入带来的语义理解深度。
