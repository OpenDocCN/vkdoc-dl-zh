# 第 2 章 使用 LangChain 集成 LLM API

你需要尝试不同的模型并调整其参数，以便为你的应用获得最佳结果。这个过程可能涉及调整模型架构、训练数据和超参数等。

## 准备数据

你必须妥善准备数据，才能从 LLM 获得更准确、更可靠的输出。在将数据输入 LLM 之前，你需要确保数据是干净的、格式正确且准备就绪。这可能涉及分词（将文本拆分为单个单词或子词）、规范化（将文本转换为一致的格式）以及处理任何缺失或不一致的数据等任务。

## 评估模型性能

一旦你训练好 LLM，就需要评估其性能，看看它的表现如何。这可能涉及困惑度（模型预测下一个词的能力）、`BLEU` 分数（衡量生成文本与参考文本的相似度）或特定于你的用例的自定义指标等指标。

不要忘记迭代和实验的力量。构建 LLM 应用很少是一蹴而就的过程。你可能需要尝试不同的方法、调整提示、多次微调模型，才能获得你想要的结果。拥抱开发过程的迭代特性，不要害怕尝试新想法。
