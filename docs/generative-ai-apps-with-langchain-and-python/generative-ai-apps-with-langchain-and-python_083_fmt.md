# 第五章：掌握创意内容提示词

你使用 `input()` 提示用户输入一些文本，并将其存储在 `user_input` 变量中：

```python
user_input = input("Enter some text: ")
```

#### 使用用户输入格式化提示词

你使用 `prompt` 实例的 `format_prompt()` 方法，用用户的输入来格式化提示词。

- `text` 参数作为关键字参数传递，用于替换提示词模板中的 `{text}` 占位符。

- 格式化后的提示词存储在 `formatted_prompt` 变量中。

```python
formatted_prompt = prompt.format_prompt(text=user_input)
```

#### 打印格式化后的提示词

你打印一条分隔线和标题“Formatted Prompt:”。你使用 `to_messages()` 方法将格式化后的提示词转换为消息列表并打印：

```python
print("\nFormatted Prompt:")
print(formatted_prompt.to_messages())
```

#### 生成聊天补全

你将格式化后的提示词消息传递给 `chat` 实例以生成聊天补全。生成的响应存储在 `response` 变量中：

```python
response = chat(formatted_prompt.to_messages())
```

#### 打印助手的响应

你打印一条分隔线和标题“Assistant's Response”，并使用 `response.content` 访问生成的响应内容并打印：

```python
print("\nAssistant's Response:")
print(response.content)
```

恭喜！你刚刚学会了如何使用 LangChain 库创建聊天提示词模板，用用户输入进行格式化，并使用 OpenAI 聊天模型生成聊天补全。聊天提示词模板包含一条定义助手角色和行为的系统消息，以及一条代表用户输入的人类消息。

该代码创建了一个聊天模型实例，提示用户输入，用用户输入格式化提示词，并基于格式化后的提示词生成聊天补全。最后，它打印出助手的响应。

### 提示词的初始设计与定制

#### 目标

贵公司希望开发一组初始提示词，以便高效处理最常见的客户咨询，并通过定制化提升对客户需求的响应速度。

#### 初始设计

以下是您可以遵循的流程步骤：

### 开发流程

您的团队分析了历史客服数据，以识别与订单状态、产品问题和退货相关的常见咨询。

### 提示词构建
