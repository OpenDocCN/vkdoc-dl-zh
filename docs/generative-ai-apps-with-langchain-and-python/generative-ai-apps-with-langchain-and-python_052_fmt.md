## OpenAI 的模型

让我们首先讨论 OpenAI 模型。

### GPT：下一代语言模型

首先，你决定从 OpenAI 开始你的模型探索工作，因为它提供了广泛的模型，每个模型都有其独特的能力和优势。你对 GPT-4 和 GPT-4o 印象尤其深刻，它们是 OpenAI 内外最受欢迎的大型语言模型。你的研究表明，你可以构建令人难以置信的应用程序，如聊天机器人、内容创作工具和 AI 写作助手。它们能够以惊人的准确性理解和生成类似人类的文本，这也给你留下了深刻印象，因为它们是在海量数据上训练的。

现在你想通过调用一个 GPT 模型来进行更多研究。

#### GPT 入门（代码片段）

要开始使用 GPT，你需要安装 OpenAI Python 包。然后，你向 GPT-4 模型提供一个提示，并指定你希望它生成的最大令牌数（单词或子词）。你还使用温度参数来控制生成文本的创造性。温度越高，输出就越有创造性和不可预测性。
