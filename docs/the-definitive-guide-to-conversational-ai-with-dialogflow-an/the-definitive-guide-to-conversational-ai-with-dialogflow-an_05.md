# 聊天机器人与人工智能

现代世界的聊天机器人或智能助手，核心就是 AI。

我们来看看 Google Nest Mini，谷歌的智能音箱。它实际上不过是一个连接互联网的带麦克风的音箱，用于访问谷歌的 AI——Google Assistant。

你对它说话。不知何故，Google Assistant 能听到你的语音并将其转换为文字。这是一个称为**语音转文字**（STT）的机器学习模型。Google Assistant 能理解所说的内容，即理解转换后的文字。这是一个称为**自然语言理解**的机器学习模型。Google Assistant 将你的文字匹配到特定的脚本流程，我们称之为**意图匹配**或**意图分类**。基于训练示例，我们可以匹配用户的真实意图。最后，当它找到答案时，会通过文本合成器为你朗读出来。这是一个**文字转语音**的机器学习模型（TTS），一种使用 WaveNet 模型、能发出逼真人声的合成器。
