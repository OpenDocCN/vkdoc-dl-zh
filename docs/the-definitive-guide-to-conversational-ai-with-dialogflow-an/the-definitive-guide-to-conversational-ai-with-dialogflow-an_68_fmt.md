
# 准确率

**准确率**是正确预测的观测值与总观测值的比率——即所有正确处理意图的比率。你可以通过以下方式计算：

- `总正确数 = 总 TP + 总 TN`

- `总错误数 = 总 FP + 总 FN`

- `准确率 = 正确数 / (正确数 + 错误数)`

你可以将类似上述的测试用例存储在你的数据仓库中，这样每次对智能助手进行更改时，都可以重新运行该场景。你需要从 BigQuery 中请求总的`TP`、`TN`、`FP`和`FN`。并且你需要存储`准确率`，以便日后在报告中检索这些数据。

## 精确率

**精确率**是正预测值的比率，用于判断是否存在假正例/**被误解的请求**的问题。精确率越高，假阳性率越低。

- `精确率 = 总 TP / (总 TP + 总 FP)`

你可以将类似上述的测试用例存储在 BigQuery 中，这样每次对智能助手进行更改时，都可以重新运行该场景。你需要从数据仓库中请求总的`TP`和`FP`。并且你需要存储`精确率`，以便日后在报告中检索这些数据。
