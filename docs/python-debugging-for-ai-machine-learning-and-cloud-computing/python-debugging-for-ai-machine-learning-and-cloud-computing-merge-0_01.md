# 引言

Python 是人工智能和机器学习领域中使用的主流语言，其数据和管道均部署在云环境中。除了在流行的集成开发环境、笔记本和命令行调试器中调试 Python 代码外，本书还涵盖了理解、诊断和调试复杂软件问题所必需的本地操作系统接口（Windows 和 Linux）。

本书首先介绍面向模式的软件诊断和调试流程，这些流程在进行 Python 调试之前，会先诊断内存转储、跟踪和日志等各种软件工件中的问题。接着，本书通过模拟异常软件行为的 Python 案例研究，教授各种调试模式。此外，本书还涵盖了云原生和机器学习环境中 Python 调试的特定内容。最后，本书总结了人工智能/机器学习的最新进展如何助力 Python 调试。本书还深入探讨了环境问题、崩溃、挂起、资源激增、泄漏和性能下降等案例研究。除了内存转储及其使用原生 `WinDbg` 和 `GDB` 调试器进行的分析外，本书还涵盖了跟踪和日志记录。

本书适合希望了解 Python 调试如何用于开发健壮可靠的 AI、机器学习和云计算软件的人士阅读。它采用一种新颖的面向模式的方法来诊断和调试异常的软件结构和行为。软件开发人员、AI/ML 工程师、研究人员、数据工程师、MLOps、DevOps 以及任何使用 Python 的人士都将从本书中受益。

源代码：本书中使用的所有源代码均可从 `github.com/Apress/Python-Debugging-for-AI-Machine-Learning-and-Cloud-Computing` 下载。
