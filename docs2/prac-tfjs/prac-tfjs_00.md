引言

许多著名的科幻作品描绘了一个未来地球，那里有飞行汽车、超凡脱俗的结构和人工智能。后者在某种程度上，现在已经成为现实。但这种人工智能（AI）与我们在电影中看到的不同。没有具有感知能力的或有机的机器人、无所不知的虚拟助手，或者具有人类良知的计算机。相反，我们目前正在体验的人工智能专注于智能的一个微妙组成部分：预测。在他们的著作《预测机器》中，经济学家 Ajay Agrawal、Avi Goldfarb 和 Joshua Gans 将预测定义为“使用先前信息作为基础来填补缺失信息”的过程，这种信息更广为人知的是**数据**。

数据是预测系统的骨架。你拥有的越多，系统学习得越好。如今，得益于计算能力的进步、更便宜的存储、可访问的互联网、我们之间的互联互通以及现代移动设备的出现，我们正在以前所未有的规模积累数据。但数据本身并不能做很多事情。它需要一个模型来消费它并从中学习。

涉及预测系统的人工智能学科被称为**机器学习**。这个学科研究使用数据学习任务而不需要明确编程的算法。例如，一个看到许多猫的图片的机器学习模型最终会学会识别猫。

虽然 AI 并不是一个新概念，但在 1950 年代和 1960 年代他们首次取得突破性发现之后，公众的兴趣、资金和 AI 项目的开发显著减少。这一时期在社区中被称为“AI 寒冬”。到了 2000 年代末，人工智能领域从沉睡中苏醒。这种复苏通常归因于杰弗里·辛顿和亚历克斯·克里泽夫斯基发表的关于计算机视觉的两篇论文。辛顿开发了一个能够识别手写数字的神经网络，而克里泽夫斯基创建了一个能够以最先进的精度对日常物体进行分类的神经网络架构。这些论文创造了“深度学习”这个术语，它用来指代深度人工神经网络。

这两个深度网络的成就使机器学习重新回到了聚光灯下，并引发了一波创新和兴趣的浪潮，其结束似乎遥不可及。在过去的十年中，机器学习和深度学习模型被应用于各种用例，例如图像分类、物体检测、预测、文本和图像生成、语音识别和语言翻译。更令人印象深刻的是，其中一些模型在相同任务上的表现超过了人类。因此，我们现在有了理解这些网络的理论，有了为它们提供数据的数据，有了处理它们的计算能力，以及工业和学术社区对尝试深度学习的兴趣，我们所需要的是更多框架来在您选择的平台上训练它们。最后一个原因将我们带到了 TensorFlow.js 和这本书。

## 关于本书

TensorFlow.js 是一个用于在 **JavaScript**（网络的语言）中直接在您的网络浏览器中开发和部署基于人工神经网络的模型的机器学习框架。在这本名为 *Practical TensorFlow.js* 的书中，您将通过动手实例学习和探索如何使用 TensorFlow.js 开发和部署模型。

正如标题所述，在这本书中，我们将专注于实际用例，这些用例不仅限于训练模型。例如，在某一章中，你将构建一个游戏，在另一章中，一个 Google Chrome 扩展程序。你将在这里编写的应用程序不会止步于浏览器。在后面的章节中，你将学习如何使用 Node.js 在服务器端运行 TensorFlow.js 来训练和部署模型。由于我们处于网络环境，构建网络应用程序，示例包括基本的网络开发概念。但如果你没有网络开发背景，请不要担心，因为本书也解释了这些部分。

本书将涵盖的一些机器学习应用包括图像分类、目标检测、自然语言处理、迁移学习和时间序列分析。你将使用卷积神经网络、循环神经网络和生成对抗网络以及“传统”的机器学习算法（如线性回归）等网络来探索它们。然而，你不会实现所有这些。TensorFlow.js 的一个特点是提供了针对各种场景的多个预训练模型。在我们的冒险中，我们将学习如何使用其中的一些。

虽然本书侧重于深度学习的实际应用，但在每个练习之前，都会简要回顾章节中涉及到的模型。最后，为了扩展示例的范围，你将使用其他工具和框架，如 TensorFlow.js 可视化、Google Cloud、TensorBoard 和 ml5.js。

## 这本书是为谁写的？

本书针对那些对使用 TensorFlow.js 开发 Web 深度学习解决方案感兴趣的人。你不需要之前的机器学习经验。然而，至少了解它是什么会有所帮助。对于初学者来说，首先，欢迎来到激动人心的机器学习领域！这本书将为你提供正确的工具和知识，让你开始你的机器学习（ML）之路。完成它之后，你将能够理解各种神经网络，它们的用途。最重要的是，在本书之后，你将能够使用 TensorFlow.js 开发神经网络，围绕网络构建应用程序，并在不同的平台上部署它们。

对于专家来说，你很可能来自 Python 背景。如果是这样，通过这本书，你将学习如何使用 TensorFlow.js Keras-like API 开发你的模型，或者在 Google Cloud 中训练它们，如何将现有模型带到 TensorFlow.js，以及如何在 Node.js 中提供服务。

结构

## 本书假设你具有编程和软件开发的经验，因为我们将会动手编写代码并使用像“服务器”、“Git”和“后端”这样的技术术语。本书的主要语言是 JavaScript，因此了解其基础知识会有所帮助。然而，本书假设你不知道它，因此它解释了那些可能令人困惑或特定于该语言的细节。这里的目的是让你完全理解练习背后的每一个细节。

书中的大多数章节由五个主要部分组成。这些部分是一个引言、模型的概述、数据集的解释、应用和总结。

### 引言

在引言中，我将介绍我们将要解决的问题、用于解决问题的模型或算法，以及我们将在本章中看到的框架或库。

### 模型

在这部分，我们将了解模型、其特征、特性以及它是如何工作的。

### 数据集

本节简要描述我们将用于训练模型的数据库。

### 应用

在这里，我们将构建应用。由于大多数练习都是 Web 应用，我们将首先在 HTML 文件中设计其界面。然后，我们将继续构建功能。

### 总结和进一步练习

为了结束每一章，我们将看到主要术语和发现的课程的小总结，然后是你可以做的进一步练习，以继续巩固你所学的内容。
