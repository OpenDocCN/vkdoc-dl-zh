引言

我们在 Google Colaboratory（Colab）生态系统中应用 TensorFlow 端到端开源平台，通过实际操作的 Python 代码练习，为中级到高级 Python 用户展示最新的深度神经网络模型。Colab 生态系统是谷歌研究的产品，它允许任何人通过浏览器编写和执行任意 Python 代码。该生态系统特别适合深度学习、数据分析、研究和机器学习教育应用。Colab 生态系统是一个托管 Jupyter 笔记本服务，无需设置即可使用，同时提供免费访问强大的计算资源，如图形处理单元（GPU）和张量处理单元（TPU）。

本书分为 14 章。第一章向您介绍了使用 tf.data 应用程序编程接口（API）的 TensorFlow 输入管道，该接口使您能够从简单的、可重用的组件构建复杂的输入管道。输入管道是任何深度学习实验的生命线，因为学习模型期望以 TensorFlow 可消费的形式接收数据。第二章利用输入管道创建增强数据实验，通过应用随机（但真实）的变换来增加训练集的多样性。第三章向您介绍了 TensorFlow Datasets。TensorFlow 提供了超过 255 个预处理的实践数据集，这些数据集是为深度学习模型消费而准备的。这些数据集由 Google Brain TensorFlow 团队创建，以提供多样化的数据集，用于练习机器和深度学习实验。第四章展示了使用大型和复杂的 TensorFlow 数据集（TFDS）对象进行的两个端到端深度学习实验。第五章通过代码示例介绍了 TPUs。Tensor Processing Unit 是一种专为加速机器学习工作负载而设计的专用集成电路（ASIC）。第六章介绍了迁移学习。迁移学习是通过微调先前训练的神经网络来创建新的学习模型的过程。第七章利用第六章中迁移学习的知识来开展高级迁移学习实验。第八章将重点从监督学习转移到无监督学习算法，从简单的堆叠自编码器开始。第九章介绍了包括卷积神经网络自编码器和变分自编码器在内的高级自编码器。第十章通过生成对抗网络（GAN）实验介绍了基本的生成建模。第十一章通过渐进式增长 GAN 实验深入探讨了更高级的生成建模。第十二章转向计算机视觉，通过神经风格迁移实验进行。第十三章通过目标检测实验深入探讨了更高级的计算机视觉。第十四章以一个非常简单的强化学习实验结束本书。
