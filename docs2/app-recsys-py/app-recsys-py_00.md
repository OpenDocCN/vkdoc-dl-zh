前言

本书献给那些从零开始新推荐引擎项目但在此领域没有先前经验的数据科学家。他们可以通过本书轻松学习概念并获得实际知识。推荐引擎最近在不同领域获得了大量的关注和流行，并有增加销售和收入的良好记录。

本书分为十一章。第一部分，第一章和 2 章，涵盖了基本方法。接下来的部分，包括第三章、4 章、5 章和 6 章，涵盖了基于协同过滤、基于内容和混合推荐系统等流行方法。下一部分，第七章和 8 章，讨论了使用最先进的机器学习算法实现系统。第九章、10 章和 11 章讨论了推荐系统中的趋势和新兴技术。

每章实现代码和所需数据集可在 GitHub 上找到，网址为[`github.com/apress/applied-recommender-systems-python`](http://github.com/apress/applied-recommender-systems-python%25ef%25bb%25bf)。

要成功完成本书中的所有项目，您需要在任何基于 Windows 或 Unix 的操作系统上运行 Python 3.x 或更高版本，处理器至少为 2.0 GHz，并且至少有 4 GB 的 RAM。您可以从 Anaconda 下载 Python，并利用 Jupyter 笔记本进行所有编码操作。本书假设您已了解 Keras 基础知识以及如何安装机器学习和深度学习的基本库。

请升级或安装所有库的最新版本。
