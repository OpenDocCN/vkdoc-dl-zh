引言

在当今世界，机器学习是我们生活的重要组成部分。我们甚至无法想象一个没有机器学习方法的未来，它已经开始占据我们日常活动的很大一部分。网站、移动应用、自动驾驶汽车、家用设备以及我们周围许多事物都使用了机器学习算法。计算能力的黎明，尤其是图形处理单元的出现，伴随着深度学习实施的实用开始。深度学习研究深度神经网络的设计。这种方法显示出令人印象深刻的效率，并在近年来经历了爆炸式增长。

并非巧合，需要解决的任务数量和对机器学习专家的需求持续增长。与此同时，开发人员和数据科学家执行以解决机器学习问题的常规操作也在增加。同时，研究人员开发了特殊技术以节省时间和自动化最常见的机器学习任务。这些技术被分离到被称为自动化机器学习（AutoML）的特定领域。本书专注于自动化深度学习（AutoDL）领域，该领域研究深度学习问题的自动化。AutoDL 考虑创建和设计最佳深度学习模型的问题。这种方法在近年来迅速发展，在某些情况下，可以完全自动化典型任务的解决方案。

这本书是关于使用微软神经网络智能（NNI）实现 AutoDL 方法。NNI 是一个包含最常见和先进 AutoDL 方法：超参数优化（HPO）、神经架构搜索（NAS）和模型压缩的 Python 工具包。NNI 支持最流行的深度学习框架。本书涵盖了使用 PyTorch 和 TensorFlow 框架实现各种 AutoDL 技术的 NNI 实现。

第一章专注于自动深度学习的基础知识以及为什么我们应该将这种方法付诸实践。我们还将安装 NNI 并检查其主要基本使用场景。我们将学习如何运行简单的 Hello World 实验，并通过命令行和 WebUI 与 NNI 进行交互。

在第二章，我们将继续研究最常见的一种 AutoDL 任务——超参数优化（HPO）。我们将学习超参数优化是什么，超参数是什么，以及如何使用 PyTorch 和 TensorFlow 组织 NNI HPO 实验。我们还将构建三种类型的研究，以追溯深度学习的起源。第一种研究将帮助我们确定 MNIST 问题中最佳 LeNet 模型超参数。第二种研究将一个新的 dropout 层和修正线性单元（ReLU）激活集成到原始 LeNet 模型中。第三种研究将展示我们如何使用简单的 HPO 技术在 AlexNet 中进化 LeNet 模型。

在第三章，我们将研究 NNI 的主要搜索算法（调优器），这些算法旨在解决超参数优化（HPO）任务。在这里，我们将考虑以下算法的实际应用和描述：进化调优器、退火调优器和 SMBO 调优器。第三章提供了自定义调优器的创建，并将其应用于经典的浅层 AutoML 问题——使用 Scikit 方法构建最优的管道。

在第四章，我们将开始研究神经架构搜索（NAS）。NAS 是一种研究创建和设计最适合解决特定问题的神经网络的方法。本章涵盖了多试验 NAS 及其主要原则。我们将讨论 NNI Retiari 框架，定义模型空间和模型突变体，并设置构建最优神经网络的实验。此外，本章介绍了探索多试验 NAS 模型空间的多种探索算法：正则化进化、TPE 策略和 RL 策略。接下来，我们将构建基于 LeNet 和 ResNet 的多试验 NAS 实验，以解决 CIFAR-10 问题。

在第五章，我们将继续研究单次 NAS，这是 AutoDL 中的最新进展之一。本章解释了如何构建超网络，如何设计基于细胞的神经网络架构，以及执行高效神经网络架构搜索（ENAS）和可微架构搜索（DARTS）单次 NAS 算法。

在第六章中，我们将探讨模型剪枝这一重要主题。模型剪枝通过移除冗余权重或甚至层来压缩神经网络。这项技术在需要节省计算资源时对轻量级设备至关重要。本章将考察基本的单次和迭代剪枝算法。

第七章将专注于使用 NNI 组织稳健、广泛和大数据实验的实际配方。

本书探讨了 AutoDL 方法的实际应用，并描述了其理论。因此，这本书对于想要了解各种 AutoDL 技术和算法背后的理念的数据科学家来说非常有帮助。

本书需要具备中级深度学习理解能力和 TensorFlow 或 PyTorch 知识。
