引言

你是否希望学习如何构建实用的深度学习和计算机视觉系统，但又不愿意切换到 Linux 进行开发？你是否觉得你对 Windows 更熟悉，并希望能够在 Windows 上构建一切？好吧，你再也不用担心了。最新的深度学习和计算机视觉库已经成熟到几乎可以在 Windows 上无缝运行的程度。本书将向你展示如何做到这一点。

Windows 操作系统占桌面 PC 使用量的 70%以上。Windows 提供了许多便利，拥有丰富的生产力工具，因此吸引了大量用户。此外，由于更好的硬件兼容性和驱动程序支持，大多数性能良好的个人电脑倾向于运行 Windows。这意味着有大量的人工智能爱好者和发展者希望学习深度学习/人工智能的非凡能力。但他们由于担心工具的复杂性，以及普遍认为 AI 系统只能在开发者友好的操作系统（如 Linux）上构建的信念，而不愿或害怕迈出第一步。本书旨在帮助他们克服这些心理障碍，开始构建实用的深度学习系统。

在 Windows 上进行深度学习将帮助您学习使用 Python、TensorFlow、Keras、OpenCV 等工具构建深度学习和计算机视觉系统，这一切都发生在熟悉的 Microsoft Windows 元素中。本书的目标是让尽可能多的您对深度学习领域产生兴趣，并确保您构建的操作系统不会成为学习的障碍。

在这个过程中，我们将学习深度学习是什么以及它是如何产生的。我们将澄清围绕深度学习的一些误解和困惑，并回顾它多年来取得的重大里程碑。我们将深入编码，同时学习如何在构建过程中应用这些概念。您将学习如何在 Windows 系统上设置所有您需要的工具和技术，以开始编写深度学习系统代码。

在本书中，您将：

+   学习深度学习背后的概念、历史和里程碑，以及它与机器学习和人工智能的关系，同时解决围绕这些人工智能概念的某些误解。

+   学习您需要（TensorFlow、Keras、OpenCV、CUDA 等）的工具来成功构建深度学习系统，并学习如何逐步设置、配置和解决它们的问题。学习如何在 Microsoft Windows 上使这些工具工作，并了解为什么操作系统或您正在开发的硬件不会阻碍您构建最先进的 AI 系统。这应该允许您打破任何心理障碍，并将您所学应用到任何操作系统或其他系统中。

+   学习构建您的第一个深度学习模型，并通过代码示例了解其每个步骤的概念是如何工作的。学习如何可视化模型内部的工作原理和结构，以获得对它们如何工作的更深入理解，并将这些经验应用到未来开发更复杂模型中。

+   学习使用迁移学习和微调的概念，在有限的数据量下构建现实世界、实用的深度学习计算机视觉系统。学习如何配置大型数据集上更大模型的训练，以及训练完成后如何部署您的应用程序。

+   一旦你掌握了基础知识，就可以学习更多令人兴奋和高级的概念，例如生成对抗网络和强化学习（游戏编程的基础）。

如果你是机器学习和 AI 的爱好者，从入门到中级水平，并想尝试一下深度学习能做什么，这本书就是为你准备的。如果你喜欢通过动手实践、通过尝试编码来学习，并且不害怕与代码打交道，这本书也是为你准备的。最后，如果你渴望构建实用、真实的 AI 系统，这本书也是为你准备的。
