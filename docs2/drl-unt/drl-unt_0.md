引言

机器学习自诞生以来就在塑造技术范围方面发挥了重要作用。ML 在自动驾驶汽车和机器人等领域的发展中发挥了重要作用。深度强化学习是这样一个学习领域，其中代理通过奖励来学习——这是一个从自然界中汲取的思想。通过本书，作者试图展示强化学习算法在游戏开发以及科学研究中的多样性。Unity，一个跨平台引擎，被用于各种任务，从视觉效果和电影摄影到机器学习和高性能图形，是本书中使用的首要工具。利用 Unity ML Agents Toolkit 的力量，Unity 构建的深度强化学习框架，作者试图展示这种学习范式的巨大可能性。

本书从基于状态的动作强化学习介绍开始，从马尔可夫过程到贝尔曼方程和 Q 学习，为后续章节奠定了基础。本书提供了从迪杰斯特拉算法到 A*星算法的复杂变体的众多不同路径查找算法，并附带了 Unity 中的模拟。本书还涵盖了 Unity 中自动路径查找的导航网格的工作原理。本书还提供了一个 ML Agents Toolkit 的介绍，从标准的安装流程到使用深度强化学习算法（近端策略优化[PPO]）训练 AI 代理。在本书的整个过程中，广泛使用了 Tensorflow 框架以及 OpenAI Gym 环境，以正确可视化复杂深度强化学习算法的模拟、机器人和自主代理。本书的后续章节深入研究了各种在线和离线策略算法，从离散的 SARSA/Q 学习到演员-评论家变体、深度 Q 网络变体、PPO 以及它们在 Gym 上使用 Keras Tensorflow 框架的实现。这些章节对于理解如何创建和构建不同的模拟，如著名的 Puppo（Unity 柏林）、Tiny 代理和其他 Unity ML Agents 样本至关重要。本书还提供了关于如何使用 C#软件开发套件为 ML Agents 构建模拟以及使用软演员评论家（SAC）、PPO 或行为克隆算法（如 GAIL）训练它们的详细描述。

本书的后半部分深入探讨了课程学习和对抗网络，分析了 AI 代理在 FIFA 等游戏中的训练方式。在这些章节中，提供了关于神经网络变体——MLP、卷积网络、循环网络以及长短期记忆和 GRU 及其实现和性能的详细描述。这特别有助于它们在构建深度学习算法时的广泛应用。本书还提供了关于卷积网络在基于 Atari 的 2D 游戏（如 Pong）中的图像采样的重要性。将计算机视觉和深度强化学习知识相结合，产生了自动驾驶汽车和无人驾驶汽车，这也为读者提供了一个构建的示例模板（游戏）。

最后，本书还包含了对 Unity Technologies 组织的障碍塔挑战的深入回顾，该挑战旨在挑战最先进的深度强化学习算法。本书提供了关于某些进化算法以及 Google Dopamine 框架的章节，以帮助理解强化学习这一广阔领域。通过本书，作者希望激发读者在深度强化学习领域的热情并促进研究。
