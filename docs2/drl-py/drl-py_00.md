引言

本书关于强化学习，将读者从基础知识引导到高级主题。尽管本书假定读者对强化学习领域没有先验知识，但它期望读者熟悉机器学习的基础。你用 Python 编写过代码吗？你是否熟悉 NumPy 和 scikit-learn 等库？你听说过深度学习，并探索过在 PyTorch 中训练简单模型的基本构建块吗？你应该对这些问题的回答是肯定的，以便充分利用本书。如果不是，我建议你先学习一些这些概念。不需要太深入——任何关于这些主题的 Apress 在线教程或书籍都足够了。

在这一版中，我在保留第一版大部分内容的同时，进行了一些重大修改。主要新增内容与大型语言模型（LLM）和多模态生成人工智能领域的新发展有关，这些发展自 2022 年底以来彻底改变了世界。强化学习（RL）通过人类反馈强化学习（RLHF）在实现这一点中发挥了关键作用。这一版新增了一章专门介绍这一主题。它为读者提供了关于 transformers、LLMs 以及相关主题（如提示工程、检索增强生成（RAG）、参数高效微调（PEFT）和 LLMs 及其基于 LLM 的自动代理的链式操作）的高级概述，随后详细解释了 RLHF 的概念。在同一章中，你还将探索近端策略优化（PPO），这是一种流行的最先进的基于强化学习的算法，OpenAI 曾用它来对 ChatGPT 进行 RLHF 微调。

另一个新增内容是关于多智能体强化学习（MARL）和深度多智能体强化学习（DMARL）的章节，它涉及多个智能体在同一环境中合作或竞争的情况。在这一章中，我从介绍开始，一直讲解到可工作的示例。我限制讨论范围，仅介绍关键概念，使感兴趣的读者能够进一步探索关于 MARL 的专业文本。

这一版还涵盖了其他主题，如超参数调整。它包括其他主题的概述，如好奇心学习、在强化学习中以各种方式使用 transformers、样本高效离线强化学习、决策 transformers、自动课程学习、零样本强化学习以及自第一版以来该领域的其他各种进步。关于深度 Q 网络的章节已被拆分为两部分，以更好地组织这一主题。

在代码方面，也发生了显著的变化。我涵盖了更多关于强化学习环境的内容。我介绍了 Gymnasium，这是 OpenAI Gym 的后继库。我还介绍了其他环境，例如 FinRL，它专注于在市场交易、其他机器人环境和多智能体强化学习（MARL）环境中应用强化学习。我还涵盖了自第一版以来获得流行度的各种其他强化学习库。2024 年代码的运行方式与以往大不相同。因此，在第一章 1 中，本书立即为读者提供了逐步安装说明，以便能够在本地机器和/或云服务提供商上运行配套代码，包括基于云的训练监控和跟踪。

本书引导读者了解强化学习的基础，在初始章节花费了大量时间解释概念。对强化学习有先验知识的读者可以快速浏览前四章。第五章开始加快节奏，并开始探讨高级主题，将深度学习与强化学习相结合。GitHub 上提供的配套代码是本书的重要组成部分。虽然本书列出了相关代码，但代码仓库中的 Jupyter 笔记本提供了关于编写这些算法的额外见解和实用技巧。读者最好先阅读章节，然后通过 Jupyter 笔记本中的解释来工作，最后通过代码来实践。本书还鼓励读者尝试自己编写代码，为不同的附加环境训练智能体，正如书中所讨论的那样。

对于这类主题，数学是不可避免的。然而，我已经尽量将其保持到最小。本书引用了许多研究论文，对所采取的方法进行了简要说明。希望对理论有更深入理解的读者应该阅读这些研究论文。本书的目的是向实践者介绍该领域许多最新技术的动机和高级方法。然而，本书绝无提供这些技术完整理论理解之意，这种理解最好通过阅读原始论文来获得。

本书共分为 13 章。

+   第一章，强化学习简介：读者将了解强化学习的基础及其在现实世界中的应用。本章还提供了在不同平台（如本地计算机或云）上设置本书附带代码的步骤。

+   第二章，基础：马尔可夫决策过程：本章描述了强化学习旨在解决的问题。我介绍了强化学习系统的组成部分——智能体、环境、奖励、价值函数、模型和政策。我解释了马尔可夫过程及其不同变体，以及理查德·贝尔曼的回溯方程。

+   第三章，基于模型的途径：本章探讨了智能体拥有模型并为其行动制定最佳结果计划的情况。我还介绍了 Gymnasium，一个流行的强化学习环境库。最后，我考察了用于规划的值迭代和政策迭代方法，包括广义政策迭代和异步回溯。

+   第四章，无模型方法：本章探讨了当智能体无法访问环境/模型的动态时可以使用的无模型学习方法。它专注于蒙特卡洛（MC）和时间差分（TD）学习方法。它首先分别检查它们，然后根据 n 步回报和资格痕迹的概念将它们合并。

+   第五章，函数逼近与深度学习：探讨了连续值状态，而不是像前几章那样使用离散状态。它从传统的手动设计函数逼近方法开始，特别是线性方法。最后，本章提出了使用基于深度学习的模型作为非线性函数逼近器的想法。

+   第六章，深度 Q-Learning（DQN）：深入探讨了 DQN，这是一种成功展示了深度学习与强化学习相结合的方法。它介绍了用于超参数优化和实验跟踪的流行库。最后，它对来自机器人、金融和交易等领域的强化学习环境进行了广泛的调查。

+   第七章，DQN 的改进：这是一个可选章节，讨论了 DQN 的改进。对于这些改进中的每一个，都有理论讨论，随后是使用 Python 和 PyTorch 的完整实现。它涵盖了优先回放、双 DQN、对抗 DQN、噪声网络 DQN、C51 DQN、Qunatile DQN 和事后经验回放等主题。

+   第八章，策略梯度算法：本章从基于价值的方法转向直接策略学习。在建立基础之后，它讨论了各种方法，包括最近的和非常成功的方法，如信任域策略优化和近端策略优化，并包括 PyTorch 中的实现。它还探讨了好奇心驱动的学习。

+   第九章，结合策略梯度与 Q-Learning：本章探讨了使用策略梯度方法与 Q-Learning 相结合的 DQN，以利用两种方法的优势，这也使得连续动作空间成为可能，这在其他情况下是不易实现的。它讨论了三种非常流行的算法——深度确定性策略梯度（DDPG）、双延迟 DDPG（TD3）和软演员-评论家（SAC）。

+   第十章，集成规划和学习：本章主要关于将第三章中的基于模型的方法和第四章至第九章中的无模型方法相结合。它讨论了一个名为 Dyna 的一般框架及其一些变体。最后，它探讨了蒙特卡洛树搜索（MCTS）及其在训练能够击败冠军围棋选手的 AlphaGo 中的应用。

+   第十一章，近端策略优化（PPO）和强化学习与人类反馈（RLHF）：本章探讨了激励 PPO 的自然策略梯度，并逐行介绍了 PPO 的实现。本章介绍了大型语言模型（LLM）和相关概念。随后，对 RLHF 进行了深入讨论，并附有工作示例。

+   第十二章，多智能体强化学习（MARL）：本章介绍了同一环境中的多个智能体概念，其理论基础，MARL 的新分类法，它与博弈论的联系，新挑战，以及 MARL 的强化学习解决方案方法。

+   第十三章，额外主题和最新进展：本章概述了强化学习的各种其他扩展。它涉及世界模型、决策转换器、自动课程学习、模仿和逆学习、无导数方法、迁移和多任务学习、元学习、无监督零样本强化学习等内容。
