简介

恭喜您决定探索使用深度学习来发现异常的激动人心的世界！

异常检测涉及寻找不符合被视为正常或预期行为模式的模式。企业可能会因异常事件而损失数百万美元。消费者也可能损失数百万美元。实际上，每天都有许多情况，人们的生命和财产处于危险之中。如果你的银行账户被清空，那是个问题。如果你的水管破裂，导致地下室被淹，那也是个问题。如果由于交通控制系统中的技术故障，机场的所有航班都延误，那也是个问题。如果你有一个被误诊或未诊断的健康问题，那是一个非常严重的问题，它直接影响到你的健康。

在这本书中，您将学习如何使用异常检测来解决商业问题。您将探索如何使用异常检测技术来解决实际用例，并在商业环境中解决现实生活中的问题。每个业务和用例都是不同的，因此虽然我们无法复制粘贴代码并在任何数据集中构建一个成功的模型来检测异常，但本书将涵盖许多用例，并通过动手编码练习来给您一个关于思维过程背后可能性和概念的启示。本书中的所有代码示例都是用 Python 3.8 编写的。我们选择 Python，因为它确实是数据科学中最好的语言，拥有大量的包以及与 scikit-learn、深度学习库等的集成。

我们将首先介绍异常检测，然后我们将探讨几十年来一直被用于检测异常的传统方法。接着，我们将了解深度学习，以尝尝它的味道。然后，我们将探索自编码器和变分自编码器，它们为下一代生成模型铺平了道路。在此之后，我们将探讨生成对抗网络（GANs）作为检测异常的一种方式，深入探讨生成人工智能。

然后，我们将探讨长短期记忆（LSTM）模型，看看如何处理时间序列数据。我们将介绍时间卷积网络（TCNs），它们非常适合时间序列数据的异常检测。我们还将简要介绍变换器架构，它作为另一种时间异常检测手段，彻底改变了自然语言处理领域。最后，我们将探讨各种商业用例中的异常检测的几个示例。

此外，所有代码示例都将提供在 TensorFlow 2/Keras 中，以及 GitHub 仓库中此书的 PyTorch 等价物。您将通过使用基于 Jupyter notebook 的练习，结合所有这些广泛的知识，亲身体验知识，并了解您可以在哪里使用这些算法和框架。祝您好运，欢迎来到深度学习的世界！
